2.0 Build Instructions
----------------------

2.1 Compilers
The following compilers and development environments were used to
create this software:
Microsoft Visual C++ 6

2.2 Libraries
None specified

2.3 Directories
Before building this project, you will need to add the following directories to
the IDE options:

Include Files
x:\ARP\WBFL\Include [Note: This must come after x:\ARP\AutoWBFL\Include]

Library Files
x:\ARP\WBFL\Lib

From the Visual C++ IDE, select Tools | Options. Then select the Directories tab.
Enter the specified directories

2.4 Build Instructions
Use the supplied DSW file for building. It has the correct dependencies
