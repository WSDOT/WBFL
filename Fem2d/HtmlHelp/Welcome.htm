Introduction

The WBFL Fem2d engine is a Windows-based COM component that can be used to analze two-dimensional linear-elastic frame structures. The engine designed to be high-performance while allowing a wide variety of clients to work with it. This is done by providing two-dimensional beam analysis Interfaces to any client that can call a COM server via the IUnknown or IDispatch interfaces. This means that the engine will work for a wide array of programing environments such as:

  • Visual C++

  • Excel

  • Visual Basic

  • Visual Fortran

  • Visio, AutoCAD, MS-WORD,

  • JavaScript, VBScript

  • and many more!

Analysis Features

The Fem2d engine supports a wide array of features for modeling 2D linear-elastic frames. Both axial and bending behaviors are modelled. Shear deformation is not modeled. The main features are as follows.

  • 2d linear-elastic prismatic beam elements (Members)

  • Model size is only limited to available memory

  • Member end releases for moment

  • Unlimited number of load cases (Loadings)

  • Supports many different types of loads including:

    • Concentrated loads applied to joints

    • Concentrated loads applied along members

    • Trapezoidally distributed loads applied to members

    • Axial and curvature strains applied to elements

    • Joint Displacements (support settlement) loads

  • Unlimited analysis points (POI's) along members (eliminates need to add joints where results are desired)
