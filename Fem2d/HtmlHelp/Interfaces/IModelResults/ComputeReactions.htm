ComputeReactions Method

(IModelResults Interface)

Summary:

Computes the reaction force at a given joint for a given loading

Syntax:

object.object.ComputeReactions([in]intLC, [in]intJntId, [out]fltRx, [out]fltRy, [out]fltRz)

Element

Description

IintLC

ID of Loading

intJntId

ID of joint where displacements are desired

fltRx

Reaction in x direction

fltRy

Reaction in x direction

fltRz

Reaction for moment in z direction

object

This Model

Remarks:

Reaction forces are in global coordinates. The reactions for unconstrained degrees of freedom are taken to be zero.
