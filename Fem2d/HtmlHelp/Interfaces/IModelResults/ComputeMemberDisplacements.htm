ComputeMemberDisplacements Method

(IModelResults Interface)

Summary:

Computes the displacements at the ends of a given member for a given loading

Syntax:

object.ComputeMemberDisplacements([in]intLC, [in]intMbrId, [out]fltStartDx, [out]fltStartDy, [out]fltStartRz, [out]fltEndDx, [out]fltEndDy, [out]fltEndRz)

Element

Description

IintLC

ID of Loading

intMbrId

ID of member where displacements are desired

fltStartDx

Displacement in x direction at start of beam

fltStartDy

Displacement in x direction at start of beam

fltStartRz

Rotation in z direction at start of beam

fltEndDx

Displacement in x direction at end of beam

fltEndDy

Displacement in x direction at end of beam

fltEndRz

Rotation in z direction at end of beam

object

This Model

Remarks:

Displacements returned are in member coordinates.
