ComputePOIDisplacements Method

(IModelResults Interface)

Summary:

Computes the displacements at a given POI for a given loading

Syntax:

object.ComputePOIDisplacements( [in]intLC, [in]intPOId, [in]intOrientation,, [out]fltDx, [out]fltDy, [out]fltRz)

Element

Description

intMbrId

ID of member where displacements are desired

IintLC

ID of Loading

intOrientation

Flag of type f2dLoadOrientation requesting results in Member or Global coordinates

fltDx

Displacement in x direction at POI

fltDy

Displacement in x direction at POI

fltRz

Rotation in z direction at POI

object

This Model
