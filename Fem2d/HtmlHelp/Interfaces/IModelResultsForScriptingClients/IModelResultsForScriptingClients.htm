IModelResultsForScriptingClients

An Interface for retrieving analysis results from the Model. This Interface is supported in the Fem2d engine by the Model object

Methods

ComputeJointDisplacements

Get the displacement at a given joint for a given loading

ComputeMemberDisplacements

Get the displacement at endpoints of a given member for a given loading

ComputePOIDisplacements

Get the displacement at a given POI for a given loading

ComputeReactions

Get the reactions at a given joint for a given loading

ComputeMemberForces

Get member end forces for a given member for a given loading

ComputePOIForces

Get forces at a given POI for a given loading

 

Remarks

Note that this interface uses VARIANTS for all [out] parameters. This is necessary for scripting clients, but it puts undue burden on C++, Visual Basic, and other high-performance vtable-based clients. It is recommended that you use the IModelResults interface for these clients. IModelResults is identical to this interface, except that it uses doubles for all [out] parameters.
