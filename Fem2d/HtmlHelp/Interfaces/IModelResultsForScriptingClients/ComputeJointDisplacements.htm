ComputeJointDisplacements Method

(IModelResultsForScriptingClients Interface)

Summary:

Computes the displacements at a given joint for a given loading

Syntax:

dblVal = object.ComputeJointDisplacements(intLC, intJntId, dof)

Element

Description

dblVal

The joint displacement

intLC

ID of Loading

intJntId

ID of joint where displacements are desired

dof

Constant of type f2dJointDOF requesting the degree of freedom to compute displacements for

object

This Model

Remarks:

Displacements are in global coordinates

Also See:

f2dJointDOF
