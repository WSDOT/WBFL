ComputeMemberForces Method

(IModelResults Interface)

Summary:

Computes the forces at the ends of a given member for a given loading

Syntax:

dblVal = object.ComputeMemberForces(intLC, intMbrId, dof)

Element

Description

dblVal

The member force

intLC

ID of Loading

intMbrId

ID of member where displacements are desired

dof

Constant of type f2dMbrDOF requesting the degree of freedom to compute forces for

object

This Model

Remarks:

Forces returned are in member coordinates

Also See:

Force Calculation Sign Conventions

f2dMbrDOF
