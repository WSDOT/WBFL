JointDisplacement

An enforced displacement load applied to a Joint. This type of load can be used to model support settlement and manufacturing errors among other things.

Properties

ID

ID of JointDisplacement

JointID

ID of Joint that load is applied to

Loading

ID of Loading this load belongs to

Dx

Applied displacement in X direction

Dy

Applied displacement in Y direction

Rz

Applied rotational displacement

Methods

SetDisplacement

Set all three displacement components in one call

GetDisplacement

Get all three displacement components in one call
