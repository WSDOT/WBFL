Joint

An individual joint (node) in a Model

Properties

ID

ID of Joint

X

X coordinate

Y

Y coordinate

Methods

Free

Releases all DOF's at joint

IsDofReleased

Returns TRUE if a given DOF is released at the joint

 

IsDofSupported

Returns TRUE if a given DOF is supported at the joint

IsSupport

Returns true if the joint is supported for any degree(s) of freedom

ReleaseDOF

Releases given DOF's

Support

Fixes the joint for all DOF's
