POI

Defines a Point Of Interest where forces and displacements can be calculated along the length of a member. Think of it as being able to place a sensor on a member.

Properties

ID

ID of POI

MemberID

ID of Member that POI is attached to

Location

Location of POI measured from start joint of member. Positive values indicate placement in base units. Negative values indicate fractional spacing and may range from -1.0 to 0.0.

Remarks

In general it is much more efficient to place POI's at locations where you want results to be calculated rather than using Joints.
