StructuredStorage Property

(Model Object)

Summary:

Get reference to the IStructuredStorage2 interface

Syntax:

objColl = object.StructuredStorage

Element

Description

objColl

Reference to the IStructuredStorage2 interface

object

This Model

Remarks:

This method provides a convenient way to get to the IStructuredStorage2 interface and is the only possible way to get to a results interface for scripting clients.

The IStructuredStorage2 interface provides a way to save and load models.

JavaScript clients should not call StructuredStorage directly as it will result in undefined behavior. JavaScript clients should call the methods on IStructuredStorage2 directly from an instance of Model.
