Loading

Describes a grouping of loads (e.g. PointLoads, JointLoads, etc...) applied to the model. Another common term for Loading is "load case".

Properties

ID

ID of Member

JointLoads

Collection of JointLoads applied to the model for this Loading

PointLoads

Collection of PointLoads applied to the model for this Loading

DistributedLoads

Collection of DistributedLoads applied to the model for this Loading

MemberStrains

Collection of MemberStrains applied to the model for this Loading

JointDisplacements

Collection of JointDisplacements applied to the model for this Loading
