PointLoad

A concentrated load applied along a Member

Properties

ID

Unique ID of PointLoad

MemberID

ID of Member that load is applied to

Location

Location of load measured from start joint along member.

Orientation

Orientation of load

Loading

ID of Loading this load belongs to

Fx

Applied Force in X direction

Fy

Applied Force in Y direction

Mz

Applied Moment

Methods

SetForce

Set all three force components in one call

GetForce

Get all three force components in one call
