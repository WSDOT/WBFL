DistributedLoad

A linearly distributed load applied along a Member

Properties

ID

Unique ID of DistributedLoad

MemberID

ID of Member that load is applied to

Direction

Direction of applied load

Orientation

Orientation of load

Loading

ID of Loading this load belongs to

StartLocation

Location of start of load measured from member start joint

EndLocation

Location of start of load measured from member start joint

WStart

Magnitude of distributed load at start of load

WEnd

Magnitude of distributed load at end of load
