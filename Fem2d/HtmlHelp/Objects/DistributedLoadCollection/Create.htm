Create Method

(DistributedLoadCollection Object)

Summary:

Creates a new DistributedLoad and adds it to the collection

Syntax:

objDistributedLoad = object.Create([in]intID, [in]intMbrID, [in]Direction, [in]fltStartLocation,[in]fltEndLocation, [in]fltWStart, [in]fltWEnd, [in]orientation)

Element

Description

intID

ID of the new DistributedLoad(must be unique)

intMbrID

ID of the Member this load is applied to

Direction

Constant of type f2dLoadDirection defining load direction with respect to Member or Global coordinate

fltStartLocation

Location of start of load measured from start joint of member. Positive values indicate placement in base units. Negative values indicate fractional spacing and may range from -1.0 to 0.0.

fltEndLocation

Location of end of load measured from start joint of member. Positive values indicate placement in base units. Negative values indicate fractional spacing and may range from -1.0 to 0.0.

fltWStart

Magnitude of distributed load at fltStartLocation

fltWEnd

Magnitude of distributed load at fltEndLocation

orientation

Optional constant of type f2dLoadOrientation that defines the orientation of the load. Default value is lotMember

object

this DistributedLoadCollection

Remarks:

Returns an error if a DistributedLoad with this id already exists

Member must exist or analysis will return an error.

Analysis returns an error if load is located off of member
