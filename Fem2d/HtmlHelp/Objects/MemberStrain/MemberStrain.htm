MemberStrain

A uniform strain and curvature applied along a portion of a Member

Properties

ID

Unique ID of MemberStrain

MemberID

ID of Member that load is applied to

Loading

ID of Loading this load belongs to

AxialStrain

Uniform strain applied along a portion of a member. Positive strains increase the length of the member.

CurvatureStrain

Uniform curvature applied along a portion of a member. Positive curvatures cause positive rotations (beam smiles)

StartLocation

Location of start of load measured from member start joint

EndLocation

Location of end of load measured from member start joint
