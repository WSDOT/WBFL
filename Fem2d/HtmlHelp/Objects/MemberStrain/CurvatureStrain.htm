CurvatureStrain Property

(MemberStrain Object)

Summary:

Uniform curvature strain applied along entire length of member.

Syntax:

fltAxialStrain = object.AxialStrain

ojbect.AxialStrain = fltAxialStrain

Element

Description

fltAxialStrain

Axial strain

object

This Point

Remarks:

Positive values cause positive curvature (beam smiles).

Curvature is defined as the reciprical of the radius of curvature of a deformed beam.

For a fixed-end beam, a uniform curvature of c will cause a constant moment
M = c * EI.

For linear temperature gradient problems, the curvature c can be determined as: c = a (T1-T2)/ d

Where:

 a

is the coefficient of thermal expansion

 

T1, T2

 are the temperatures at the top and bottom of the beam, respectively

 

d

is the depth of the beam (distance from top to bottom)

 
A beam deformed with uniform curvature of length L with a downward displacement of d at mid-span, has a curvature c = 8d/L2
