Create Method (MemberStrainCollection Object)

Summary:

Creates a new MemberStrain and adds it to the collection

Syntax:

objMemberStrain = object.Create([in]intID, [in]intMbrID, [in]fltStart, [in]fltEnd, [in]fltAxialStrain, [in]fltCurvatureStrain)

Element

Description

intID

ID of the new MemberStrain(must be unique)

intMbrID

ID of the Member this load is applied to

fltStart

Start location of the strain measured from the member start joint

fltEnd

End location of the strain measured from the member start joint

fltAxialStrain

Magnitude of uniform axial strain applied along member

fltCurvatureStrain

Magnitude of uniform curvature applied along member

Remarks:

Returns an error if a MemberStrain with this id already exists

Member must exist or analysis will return an error.

Analysis returns an error if load is located off of member
