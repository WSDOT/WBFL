Top Level View of Fem2d Object Model

The following diagram shows the high-level relationships between the COM objects in the Fem2d engine. This should help to serve as a road map when using the engine from the point of view of a client. If you plan to do some programming using the engine, it is probably worth printing this picture and keeping it close at hand. It does not show any of the details of the C++ implementation.
