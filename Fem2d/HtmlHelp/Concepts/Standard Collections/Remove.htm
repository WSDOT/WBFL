Remove Method

(Standard Collections Objects)

Summary:

Removes the specified Thing from the collection

Syntax:

intID = object.Remove([in]intIDorIndex, [in]AccessType)

Element

Description

intID

ID of the item that was removed

intIDorIndex

ID or index (depending on AccessType) of the Thing to be removed.

AccessType

Constant of type f2dAccessType that selects whether to find the Thing to be removed by ID or sequential index.

object

This ThingCollection

Remarks:

Returns an Invalid Argument error if the item cannot be found in the collection

Also See:

f2dAccessType
