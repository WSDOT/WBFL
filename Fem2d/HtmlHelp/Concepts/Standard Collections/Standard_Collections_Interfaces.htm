Standard Collections Interface

There are many different types of collections in the Fem2d engine. However, all of the collections have interfaces that are identical, except for a couple of functions.

For expediency, the documentation for the common functions for all collections is presented here. Objects in the collections are called "Thing"'s for lack of a better word. Refer to the documentation for the specific collection you are interested in for documentation that is specific to that object.

Properties

Count

Number of Things in collection

Item

Reference to the ith Thing in the Collection (zero-based)

Methods

Create

Creates a new Thing and adds it to the collection. This method is type-specific.

Find

Find a Thing with the given ID

Remove

Remove a Thing by ID or by sequential index.

Clear

Remove all Things from the collection

Restricted Properties

_EnumElements

Returns a fast, type-specific enumerator for C++ clients
