On-Demand Computation

If you have browsed the Properties and Methods for the Fem2d Model, you may have noticed that there is no Method called "AnalyzeModel" or "ComputeNow". This is because the engine only performs analyses on an as-needed basis. A basic flow chart for client programs is shown below:

The basic idea here is that you can request results at any time, and the engine will automatically update itself if it needs to. Of course, you must first build a valid model and apply loads to it.
