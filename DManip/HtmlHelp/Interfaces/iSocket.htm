iSocket

Interface implemented by Socket objects. Sockets can be added to connecable objects. The Plug objects of a connector object are connected to socket objects.

void SetPosition(IPoint2d* pos)
void IPoint2d* GetPosition()
Gets/Sets the position of the socket

pos

Position of the socket in world space

void Move(ISize2d* offset)
Moves the socket to a new position

offset

The amount to move the socket in world space

void Connect(iPlug* plug,DWORD* pdwCookie)
Connects a plug to the socket

plug

The plug object

pdwCookie

A number used to idenify the connection

void Disconnect(DWORD dwCookie)
Disconnects a plug from the socket

dwCookie

A number used to identify the connection

void Draw(CDC* pDC,iCoordinateMap* pMap)
Called by the framework whenever the socket needs to be drawn

pDC

The device context on which to draw

pMap

The coordinate map

void SetConnectable(iConnectable* connectable)
void GetConnectable(iConnectable** connectable)
Gets/Sets the connectable object this socket blongs to.

connectable

The connectable object
