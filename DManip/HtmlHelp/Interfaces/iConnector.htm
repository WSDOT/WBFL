iConnector

Interface implemented by connector objects. Connectors are object such as lines that connect two connectable object together.

void GetStartPlug(iPlug** plug)
Returns the plug object at the starting end of the connector

plug

The plug at the starting end of the connector

void GetEndPlug(iPlug** plug)
Returns the plug object at the ending end of the connector

plug

The plug at the ending end of the connector
