iDrawLineStrategy

Interface implemented by line drawing strategy objects. By implementing a line drawing strategy, line display objects can easily be drawn in different ways.

void Draw(iLine* pDO,CDC* pDC)
Called by the framework when a point display object needs to be drawn

pDO

The line display object that needs to be drawn

pDC

The device context on which to draw

void DrawDragImage(iLine* pDO,CDC* pDC,IPoint2d* startPoint,IPoint2d* dragPoint)
Called by the framework when a line is being dragged and it needs to be drawn

pDO

The line display object

pDC

The device context on which to draw

startPoint

The starting point of the drag/drop operation, in world space

dragPoint

The current point of the drag/drop operation in world space

void DrawHighlite(iLine* pDO,CDC* pDC,BOOL bHighlite)
Called by the framework whenever an appropretate object is dragged of this display object. The line display object needs to be drawn highlighted to indicate to the user that it is a valid point to drop.

pDO

The line display object

pDC

The device context on which to draw

bHighlite

If TRUE, draw the point highlighted, otherwise draw it normal
