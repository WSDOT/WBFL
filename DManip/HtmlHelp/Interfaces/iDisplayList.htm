iDisplayList

Interface implemented by DisplayList objects. Used to manage a list of display objects

void SetDisplayMgr(iDisplayMgr* dispMgr)
void GetDisplayMgr(iDisplayMgr** dispMgr)
Sets/Gets the display manager object associated with this list.

dispMgr

The display manager

void SetID(long id)
long GetID()
Sets/Gets the display list identifier. Display lists must be given unique identifiers

id

The display list identifier

void AddDisplayObject(iDisplayObject* pDO)
Adds a display object to the display list

pDO

Reference to the display object

void GetDisplayObject(long index,iDisplayObject** pDO)
Gets the display object and the specified index

index

Index of the subject display object

pDO

Reference to the display object

void FindDisplayObject(long id,iDisplayObject** pDO)
Searches the display list for a display object with the specified id

id

Identifier to search for

pDO

Reference to the display object

void RemoveDisplayObject(long key,BOOL bByIndex)
Removes a display object from the display list

key

Index or ID of the display object to be removed

bByIndex

If TRUE, removes key is taken to be an index, otherwise it is taken to be a display object identifier

void Clear()
Removes all display object from the list

long GetDisplayObjectCount()
Returns the number of display objects in the display list

void FindDisplayObject(CPoint point,iDisplayObject** dispObj)
Searches the display list for the first display object that occupies the specified logical point.

point

A point in logical space

dispObj

The display object. Null if not found

void FindDisplayObject(IPoint2d* point,iDisplayObject** dispObj)
Searches the display list for the first display object that occupies the specified world point

point

A point in world space

dispObj

The display object. Null if not found

void FindDisplayObjects(CRect rect,DisplayObjectContainer* dispObjs)
Searches the display list for all display objects that touch the specified logical space rectangle.

rect

A rectangle in logical space

dispObjs

A container of display objects

void DrawDisplayObjects(CDC* pDC)
Draw all of the display objects on the specified device context

pDC

A device context

void HideDisplayObjects(BOOL bHide)
Hides the display objects in this list

bHide

If TRUE, the display objects are hidden, otherwise they are visible
