iDraggable

Implemented by display objects that can participate in a drag/drop operation.

void SetDragData(iDragData* dd)
void GetDragData(iDragData** dd)
Sets/Gets the drag data object associated with this display object

dd

A drag data object

UINT Format()
Returns the clipboard format for this display object.

void PrepareDrag(iDragDataSink* pSink)
Called by the framework to prepare a draggable object for drag drop operation. After saving display object specific data, calls PrepareDrag on the associated DragData object.

pSink

The drag data sink

void OnDrop(iDragDataSource* pSource)
Called by the framework when the draggable object has been dropped. The draggable object reads the data is stored in PrepareDrag and then calls OnDrop on the associated DragData object.

pSource

The drag data source

void OnDragMoved(ISize2d* offset)
Called by the framework when a draggable display object is moved within a DisplayView.

offset

The amount, in world space, that the display object was moved

void OnMoved()
Called by the framework when a draggable display object is moved from one DisplayView to another.

void OnCopied()
Called by the framework when a draggable display object is copied from one DisplayView to another, or copied within a DisplayView

void DragDragImage(CDC* pDC,IPoint2d* dragStart,IPoint2d* dragPoint)
Called by the framework whenever the display object needs to be drawn during a drag/drop operation.

pDC

The device context on which to draw

dragStart

The starting point of the drag/drop operation, in world space

dragPoint

The current location of the cursor, in world space
