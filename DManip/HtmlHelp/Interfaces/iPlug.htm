iPlug

Interface implemented by Plug objects. A Plug object is attached to a Socket object to form a connection between Connectable objects and Connector objects.

void Notify(iSocket* socket)
Called by the framework when an event occurs on a socket. The plug forwards this event to all of its event sinks

socket

The socket where the event occured

void OnRemove(iSocket* socket)
Called when a socket is removed from a connectable object. This gives the plug the opportunity to disconnect from the socket before the socket is destroyed

socket

The socket that is being removed

void Register(iPlugEvents* pEventSink)
Registers a plug event sink with this plug object

pEventSink

A sink for plug events

void SetSocket(iSocket* socket)
void GetSocket(iSocket** socket)
Sets/Gets the socket this plug is associated with.

socket

The socket
