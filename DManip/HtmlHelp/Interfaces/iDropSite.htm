iDropSite

Interface implemented by DropSite objects. Client applications implement drop site objects and associate them with display objects that can have other display objects dropped on then.

DROPEFFECT CanDrop(COleDataObject* pDataObject,DWORD dwKeyState,IPoint2d* point)
Called by the framework when a display object is dragged over another. The drop site indicates whether the display objects can be dropped by returning a drop effect

pDataObject

The data object for the drag/drop operation

dwKeyState

The state of the keys

point

The world space coordinate of the cursor

void OnDropped(COleDataObject* pDataObject,DROPEFFECT dropEffect,IPoint2d* point)
Called by the framework when display objects are dropped on the drop site.

pDataObject

The data object for the drag/drop operation

dropEffect

The drop effect

point

The world space coordinate where the drop occured

void SetDisplayObject(iDisplayObject* dispObj)
void GetDisplayObject(iDisplayObject** dispObj)
Associated a display object with this drop site

dispObj

The display object that is associted with this drop site

void Highlite(CDC* pDC,BOOL bHighlite)
Call by the framework when the drop site needs to be highlighted

pDC

The device context on which to draw

bHighlite

If TRUE, the drop site needs to be highlighted
