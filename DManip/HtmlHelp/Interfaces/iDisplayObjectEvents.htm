iDisplayObjectEvents

This interface is implemented by objects wishing to receive display object events

void OnLButtonDown(iDisplayObject* pDO,UINT nFlags,CPoint point)
Called by the framework when a LButtonDown event occurs over a display object.

void OnLButtonUp(iDisplayObject* pDO,UINT nFlags,CPoint point)
Called by the framework when a LButtonDown event occurs over a display object.

void OnLButtonDblClk(iDisplayObject* pDO,UINT nFlags,CPoint point)
Called by the framework when a LButtonDblClk event occurs over a display object.

void OnRButtonDown(iDisplayObject* pDO,UINT nFlags,CPoint point)
Called by the framework when a RButtonDown event occurs over a display object.

void OnRButtonUp(iDisplayObject* pDO,UINT nFlags,CPoint point)
Called by the framework when a RButtonDown event occurs over a display object.

void OnRButtonDblClk(iDisplayObject* pDO,UINT nFlags,CPoint point)
Called by the framework when a RButtonDblClk event occurs over a display object.

void OnKeyDown(iDisplayObject* pDO,UINT nChar,UINT nRepCnt,UINT nFlags)
Called by the framework when a display object is selected and a KeyDown event occurs.

void OnChanged(iDisplayObject* pDO)
Called by the framework when data managed by the display object is changed. For example, if the text in an Editable Text box is changed, the OnChanged event is fired.

void OnDragMoved(iDisplayObject* pDO,ISize2d* offset)
Called by the framework when a display object is moved within a DisplayView.

pDO

The display object that was moved

offset

The amount the display object was moved in world space

void OnMoved(iDisplayObject* pDO)
Called by the framework when a display object is moved from one display view to another by a drag and drop operation.

void OnCopied(iDisplayObject* pDO)
Called by the framework when a display object is copied from one display view to another by a drag and drop operation.

void OnSelect(iDisplayObject* pDO)
Called by the framework when a display object is selected.

void OnUnselect(iDisplayObject* pDO)
Called by the framework when a display object is unselected.
