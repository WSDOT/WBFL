iDimensionLine

Interface for DimensionLine objects

iDisplayObject
  ^
  |
  +----- iDimensionLine

void SetAngle(double angle)
double GetAngle()
Sets/Gets the angle of the dimension line. The angle defines the slope of the dimension line, measured from the X direction.

angle

Angle of the dimension line measured in radians

void SetWitnessLength(double l)
double GetWitnessLength()
Sets/Gets the length of the witness line in twips.

l

Length of the witness line in twips

void SetWitnessOffset(double wOffset)
double GetWitnessOffset()
Sets/Gets the witness line offset in twips. The witness line offset is the distance from the connector point to the start of the witness line.

wOffset

Witness line offset in twips

void SetArrowHeadSize(CSize size)
CSize GetArrowHeadSize()
Sets/Gets the arrow head size in twips. The cx parameter of size is the width of the arrowhead. The cy parameter is the length of the arrowhead

size

The size of the arrow head in twips

void EnableAutoText(BOOL bEnable)
Enables/disables auto text. When enabled, the dimension line text is automatically updated as the end points of the dimension line move.

bEnable

Boolean value indicating if auto text is to be enabled or disabled.

BOOL IsAutoTextEnabled()
Returns TRUE if auto text is enabled.

void SetTextBlock(iTextBlock* textBlock)
void GetTextBlock(iTextBlock** textBlock)
Associates a text block with the dimension line object. The text block displays the text of the dimension line.

textBlock

A text block object
