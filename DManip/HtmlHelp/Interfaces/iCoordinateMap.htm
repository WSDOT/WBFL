CSize GetTextWindowExtent()
Returns the entire logical extents (paintable window, in most cases) in twips. This function can be used in conjunction with the text space coordinate transforms to lay out items on the display canvas accurately.

Note that this function uses the logical extents as set by iMapping::SetLogicalExt. If you do not use the entire device extents as the logical extents, you may not get what you want.
