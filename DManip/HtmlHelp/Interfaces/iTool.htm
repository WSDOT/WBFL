iTool

Interface implemented by tools. Tools are objects in a toolbar that can be dragged and dropped onto the DisplayView.

void SetID(UINT id)
UINT GetID()
Set/Get the identifier for the tool. The tool palette is implemented with a CDialogBar object. The tool identifier is the control id.

id

Tool identifier.

CString GetToolTipText()
