iDisplayMgrEvents

This interface is implemented by objects wishing to intercept events going to the canvas. This allows you to catch events if they are not handled by any DisplayObjects on the canvas.

void OnLButtonDown(iDisplayMgr* pDO,UINT nFlags,CPoint point)
Called by the framework when a LButtonDown event occurs over a display object.

void OnLButtonDblClk(iDisplayMgr* pDO,UINT nFlags,CPoint point)
Called by the framework when a LButtonDblClk event occurs over a display object.

void OnKeyDown(iDisplayMgr* pDO,UINT nChar,UINT nRepCnt,UINT nFlags)
Called by the framework when a display object is selected and a KeyDown event occurs.
