iInplaceEditable

Interface implemented by display objects that present an in-place editing interface such as a text edit box.

void DoInplaceEdit()
Called by the framework to start the inplace editing task.

void CreateEditControl()
Called by the framework when the editing interface needs to be created and displayed to the user.

void DestroyEditControl()
Called by the framework when the editing interface needs to be destroyed.

BOOL ValidateData()
Called by the framework when the edited data is to be validated. Return FALSE if the data is invalid and editing needs to continue.

void OnDataChanged()
Called by the framework when the edited data has changed
