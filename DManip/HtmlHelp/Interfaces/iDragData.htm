iDragData

Implemented by the client application to provide data used in OLE data transfer

UINT Format()
Returns and integer that identifies the data format

BOOL PrepareForDrag(iDisplayObject* pDO,iDragDataSink* pSink)
Called by the framework just before drag/drop is about to begin. Write what ever data you want to be included in the data transfer to the DragDataSink object. Returns TRUE if successful.

pDO

The display object that is about to be dragged

pSink

The drag data sink where data is stored

void OnDrop(iDisplayObject* pDO,iDragDataSource* pSource)
Called by the framework when the display object is dropped. This is your opportunity to read the data you stored in PrepareForDrag.

pDO

The display object that was dropped

pSource

A DragDataSource object containing the data your saved in PrepareForDrag
