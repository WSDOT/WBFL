iDisplayObjectFactory

Interface implemented by display object factories. Most applications need customized versions of display objects because the display objects they have additional information with them. When the framework needs a new display object it uses the factory to create one. The factory gives you the opportunity to initialize the display object. An example of when the framework needs to create display object is during a drag and drop operation. The framework needs temporary display object to drag around the screen.

void Create(CLIPFORMAT cfFormat,COleDataObject* pDataObject,iDisplayObject** dispObj)
Called by the framework to create a display object.

cfFormat

Clipboard format identifying the display object

pDataObject

Data object that contains the display objects data

dispObj

The newly created display object
