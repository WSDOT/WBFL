iDisplayObject

Interface supported by all display objects

void SetID(long ID)
long GetID()
Sets/Gets the display object identifier. The display object ID must be unique within a display list.

ID

The display object identifier

void SetDisplayList(iDisplayList* list)
void GetDisplayList(iDisplayList** list)
Sets/Gets the display list that this display object belongs to.

list

The display list

void Visible(BOOL bVisible)
Sets the visibility of the display object

bVisible

If TRUE, the display object is visible

BOOL IsVisible()
Returns TRUE if the display object is visible

void Draw(CDC* pDC)
Draws the display object onto a device context

pDC

The device context on which to draw

void Highlite(CDC* pDC,BOOL bHighlite)
Draws the display object highlited. This typically occurs during a drag and drop operation when an object is being dragged over the display object.

pDC

The device context on which to draw

bHighlite

If TRUE, the display object is drawn highlited

BOOL HitTest(CPoint point)
Returns TRUE if the display object contains the specified logical point

point

A point in logical space

CRect GetBoundingBox()
Returns the rectangle, in logical space, that completely surrounds the display object

void GetBoundingBox(IRect2d** rect)
Returns the rectangle, in world space, that completely surrounds the display object

BOOL TouchesRect(CRect r)
Returns TRUE if the display object touches the specified rectangle in logical space

r

A rectangle in logical space

void Select(BOOL bSelect)
Selects/unselects the display object

bSelect

If TRUE, the display object is selected

BOOL IsSelected()
Returns TRUE if the display object is selected

void SetToolTipText(LPCSTR lpszToolTipText)
CString GetToolTipText()
Sets/Gets the tool tip text for this display object

lpszToolTipText

The tool tip text

void OnLButtonDown(UINT nFlags,CPoint point)
Called by the framework when a LButtonDown event occurs over this display object.

void OnLButtonDblClk(UINT nFlags,CPoint point)
Called by the framework when a LButtonDblClk event occurs over this display object.

void OnKeyDown(UINT nChar,UINT nRepCnt,UINT nFlags)
Called by the framework when this display object is selected and a KeyDown event occurs.

void RegisterEventSink(iDisplayObjectEvents* eventSink)
Registers an event sink with this display object. The event sink receives notifications when events occur on this display object.

eventSink

The event sink

void UnregisterEventSink()
Unregisters the event sink

void GetEventSink(iDisplayObjectEvents** eventSink)
Returns the event sink

eventSink

The event sink

void SetDropSite(iDropSite* dropSite)
void GetDropSite(iDropSite** dropSite)
Sets/Gets the drop site for this display object. If a display object has a drop site associated with it, display objects can be drop on it. The drop site object determines under what conditions various display objects can be drop, the user feedback that needs to be displayed when an object is being dragged of this display object, and what to do with the objects that are dropped.

dropSite

The drop site
