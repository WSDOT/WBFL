Coordinate Systems

In order to display the real world objects, represented with floating points numbers, on an integer display space coordinate mappings must be done. The WBFL DManip framework works with four coordinate systems, world space, text space, logical space, and device space.

World Space Coordinates

World space coordinates are the real world coordinates used represent geometry of something in the real world, like a bridge.

Text Space Coordinates

Text space coordinates provide a standard (x positive right, y positive up).

Logical Space Coordinates

The logical space coordinates are the same as Logical Coordinates in the Windows GDI.

Device Space Coordinates

The device space coordinates are the same as Device Coordinates in the Windows GDI.

Coordinate Mapping

Points can be mapped betwen World Space and Logical Space coordinates using the iCoordinateMap interface. This interface can be obtained from the Display Manager object.
