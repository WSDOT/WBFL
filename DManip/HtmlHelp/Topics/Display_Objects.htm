Display Objects

Display objects are what this framework is all about. Display objects are the entities on the computer screen that the user interacts with. Interactions include clicking, dragging, and double clicking. These user interactions are all mapped to application specific operations.

Everything you see in the Display View is a display object. Some of the display objects are Point, Line, and Text Block. Some of the display objects are connectable and some are connectors.

Display View

The display view's job is to act as a canvas. It does all the work necessary to be a view. All of the work dealing with interactions are delegated to the display manager.

Display Manager

The display manager controls the display and coordinates the user interactions by forwarding events, executing tasks, and managing display lists.
