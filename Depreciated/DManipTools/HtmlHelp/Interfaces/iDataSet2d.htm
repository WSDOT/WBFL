iDataSet2d

Interface for describing a collection of 2d Data.

void get_Item(long idx, IPoint2d** pVal)
Returns the point at the given index

pVal

The point to be retrieved. Returned by reference

idx

The index of the desired point

void get_Count(long *pVal)
Returns the number of points in the data set

pVal

Number of points

void Add(IPoint2d *pVal)
Append a point to the dataset.

pVal

New point to be added (by reference)

 

void Insert(long index, IPoint2d *pVal)

Insert a new point at the selected location

pVal

New point to be added (by ref)

index

Location in the collection where the point is to be added. Must be in the range (0-size)

void Remove(long idx)
Returns the point at the specified location

idx

location of point to be removed

void Clear()

Remove all Points from the data set collection
