iLegendEntry

Interface for defining entries that are to be placed in a legend display object.

void get_Name(BSTR *pVal)

Get the name for the legend entry. Th entry name is put into the legend next to its symbol.

pVal

String that is the name for this entry

void Draw(CDC* pDC, CRect& drawRect, BOOL beingDragged)
The legend display object uses this call to draw the symbol for the entry in the given rectangle.

pDC

CDC object that the entry is to use for drawing

drawRect

Rectangle, in logical coordinates where the drawing is to take place

beindDragged

Boolean value to indicate wheter the item is being dragged. If this is true, don't draw anything (such as text) that does not work with ROPCode draw/erasing

HRESULT GetClassID(CLSID *pClassID)
Get the CLSID this LegendEntry object

pClassID

This value is used to dynamically recreate the legend entry during drag/drop operations. This is identical to the IPersist interface in COM

 

HRESULT WriteDragData(iDragDataSink* pSink, UINT cfFormat)
Write legend entry drag data to the data sink on prepare for dragging

pSink

The data sink to be written to

cfFormat

The clipboard format value that refers to the sink

 

HRESULT ReadDragData(iDragDataSource* pSource, UINT cfFormat)
 

Read legend entry data after a drag operation

pSource

The data sink to be read from

cfFormat

The clipboard format value that refers to the source
