iGraphXyDisplayObject

Interface for describing a 2d X-Y graph.

void get_GraphBounds)( IRect2d* *pVal)
void put_GraphBounds)( IRect2d* pVal)

Get/set the location of the graph in world coordinates. Note that this has nothing to do with how data sets are mapped onto the graph surface.

pVal

Rectangle (by ref) that defines the location of the graph on the canvas in world coordinates

void get_Title(BSTR *pVal)
void put_Title(BSTR *pVal)

Get/set the title for the graph. If not blank, the title is placed above the graph.

pVal

String that is the title of the graph

void DataCount(long* cnt)

Get the number of data sets (as described by iGraphXyDataProvider) on this graph

cnt

Number of iGraphXyDataProviders

void DataCount(long* cnt)

Get the number of data sets (as described by iGraphXyDataProvider) on this graph

cnt

Number of iGraphXyDataProviders

 

HRESULT GetData(long idx, iGraphXyDataProvider** data)

Get the ith iGraphXyDataProvider (data set) on this graph

data

The iGraphXyDataProvider at the desired location

idx

Index of the desired iGraphXyDataProvider

 

HRESULT Commit()
 

Commit the current data to the graph and redraw all curves. Note that graph data will not be drawn or updated until this call is made.

 

 

HRESULT ClearData()
 

Clear all data sets from the graph

void get_NumberOfMajorIncrements)(long *pVal)
void put_NumberOfMajorIncrements)(long Val)

Get/set the number of increments where axis values will be put along the graph's x and y axes. Note that this value is only a recommendation - the graph will use this as a starting point to choose the best possible value.

pVal

Number of increments along X and Y where axis values are to be placed

void get_DisplayAxisValues(BOOL *pVal)
void put_DisplayAxisValues(BOOL Val)

 
Get/set wheter values are to be drawn along graph axes

pVal

If not FALSE, values are placed along graph axes at major axis increments

void get_DoDisplayGrid(BOOL *pVal)
void put_DoDisplayGrid(BOOL Val)

 
Get/set wheter a grid is to be overlaid onto the graph

pVal

If not FALSE, a grid is overlaid onto the graph at major axis increment locations
