ProfileElement Object

An object that represents an element in a vertical profile. A ProfileElement can represent a ProfilePoint or a VertCurve object. ProfileElement objects are stored in Profile objects to define the vertical profile of a roadway.

Properties

StructuredStorage

Returns a reference to an IStructuredStorage2 interface

Type

Returns the type of object assigned to the Value property

Value

Sets/Gets the object associated with this ProfileElement

Methods

Clone

Creates a copy of this object

Events

OnProfileElementChanged

Fires when the ProfileElement changes
