_EnumKeys Property (AlignmentCollection Object)

Summary:

Returns an interface to enumerate the keys in the collection

Syntax:

HRESULT get__EnumKeys(IEnumKeys** ppEnum);

Element

Description

ppEnum

Pointer to an IEnumKeys interface

Remarks:

This is a restricted property and is not available to Automation clients such as Visual Basic or JavaScript
