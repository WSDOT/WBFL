OnAlignmentAdded Event (AlignmentCollection Object)

Summary:

Fires when an Alignment object is added to the collection

Syntax:

object.OnAlignmentAdded(key,objAlignment)

Element

Description

object

The object receiving the event

key

The lookup key for the Alignment object

objAlignment

The Alignment object that was added to the collection
