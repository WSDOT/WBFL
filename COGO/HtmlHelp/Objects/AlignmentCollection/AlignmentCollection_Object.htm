AlignmentCollection Object

A collection that holds Alignment objects. Alignment objects are stored in this collection by keys.

Properties

Count

Returns the number of Alignment objects in this collection

Factory

Set/Get the AlignmentFactory

Item

Set/Get an item in the collection

StructuredStorage

Returns a reference to an IStructuredStorage2 interface

Methods

Add

Adds an Alignment object to the collection by creating one with the specified key

AddEx

Adds an Alignment object to the collection

Clear

Removes all the Alignment objects from the collection

Clone

Creates a copy of this object

FindKey

Finds a key for an Alignment object

Key

Returns the key for a certain position in the collection

Remove

Removes an Alignment object from the collection

Events

OnAlignmentAdded

Fires when an Alignment object is added to the collection

OnAlignmentChanged

Fires when one of the Alignment objects in the collection changes

OnAlignmentRemoved

Fires when an Alignment object is removed from the collection

OnAlignmentsCleared

Fires when the collection is cleared

OnProfileChanged

Fires when a profile associated with one of the Alignment objects in the collection changes

Restricted Properties

_EnumAlignments

Returns an interface to enumerate Alignment objects

_EnumKeys

Returns an interface to enumerate the keys in the collection
