Value Property (AlignmentElement Object)

Summary:

Sets/Gets the object associated with this AlignmentElement

Syntax:

object.Value = objDisp
objDisp = object.Value

Element

Description

objDisp

A Point, LineSegment, or HorzCurve object

object

This object

Remarks:

The Type property can be used to determine what type of object is assigned to this property
