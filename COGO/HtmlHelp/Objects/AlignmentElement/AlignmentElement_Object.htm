AlignmentElement Object

An object that represents an element in a horizontal alignment. An AlignmentElement can represent a Point, LineSegment, or HorzCurve object. AlignmentElements are stored in Alignment objects to define a horizontal roadway alignment.

Properties

Type

Returns the type of object assigned to the Value property

Value

Sets/Gets the object associated with this AlignmentElement

StructuredStorage

Returns a reference to an IStructuredStorage2 interface

Methods

Clone

Creates a copy of this object

Events

OnAlignmentElementChanged

Fires when the AlignmentElement changes
