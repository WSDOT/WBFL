Direction Object

An object that represents a direction. A direction can be a bearing or an azimuth.

Properties

Degree

Get/Set the degree portion of a bearing in DMS format

EWDirection

Get/Set the East/West portion of a bearing

Minute

Get/Set the minute portion of a bearing in DMS format

NSDirection

Get/Set the North/South portion of a bearing

Second

Returns the second portion of a bearing in DMS format

StructuredStorage

Returns a reference to an IStructuredStorage2 interface

Value

Get/Set the value of the bearing

Methods

AngleBetween

Computes the angle between this direction and another

Clone

Creates a copy of this object

FromAzimuth

Sets the value of the direction from an azimuth

FromDMS

Sets the value of the direction from DMS values

FromString

Sets the value of the direction from a text string

Increment

Creates a new direction that is equal to this direction plus a increment

IncrementBy

Increments this direction by a specified angle
