CrossSection Object

A CrossSection object represents the cross section of a roadway surface. A CrossSection object is defined by the location of its crown point away from the profile grade line (CrownPointOffset), and its Left and Right Cross Slopes.

Crown Point Offsets to the right of the profile grade line are positive. Cross slopes are positive if the roadway surface elevation increases towards the edge of the roadway.

Properties

CrownPointOffset

Gets/Sets the crown point offset

LeftCrownSlope

Get/Sets the left crown slope

RightCrownSlope

Gets/Sets the right crown slope

Station

Gets/Sets the station of this cross section

StructuredStorage

Returns a reference to an IStructuredStorage2 interface

Methods

Clone

Creates a copy of this object

Events

OnCrossSectionChanged

Fires when the description of the cross section changes.

OnCrossSectionMoved

Fires when the location of the cross section changes
