CubicSpline Object

An object that represents an element of a path modeled as a cubic spline. A spline consists of at least two points and the tangent direction at the start and end points.

Using a series of coordinate points and the direction of the tangents at the first and last point to model a smooth curve modeled by piecewise cubic segments.

This objects uses "by value" semantics for the points and directions.

Properties

EndDirection

Gets/Sets the direction of the tangent at the end of the spline

EndPoint

Gets the end point

Length

Returns the length of the spline

Point

Returns a spline control point

PointCount

Returns the number of points in the spline

PointFactory

Gets/Sets the point factory

Points

Returns all the spline control points

StartDirection

Gets/Sets the direction of the tangent at the start of the spline

StartPoint

Returns the start point

StructuredStorage

Returns a reference to an IStructuredStorage2 interface

Methods

AddPoint

Add a point to the spline

AddPointEx

Add a point object to the spline

AddPoints

Adds a collection of points to the spline

Bearing

Computes the bearing of a line tangent to the spline at a specified distance from the start

Clear

Removes all points from the spline

Clone

Creates a copy of this object

DistanceFromStart

Computes the distance along the spline from the start point to the intersection of a point projected onto the spline and the spline

Evaluate

Evaluates the spline as a function in a Cartesian coordinate space

Intersect

Determines the points where a line intersects the spline

Normal

Computes the bearing of a line normal to the spline at a specified distance from the start

PointOnSpline

Determines the coordinate of a point on the spline at a specified distance from the start

ProjectPoint

Projects a point onto the spline such that a line perpendicular to the spline passes through the point

Slope

Treating the spline as a function in a Cartesian coordinate space, this method computes the first derivative.

Events

OnSplineChanged

Fires when the shape of the spline changes
