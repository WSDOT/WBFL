Intersect Method (CubicSpline Object)

Summary:

Determines the points where a line intersects the spline

Syntax:

objPoints = object.Intersect(objLine,bProjectBack,bProjectAhead)

Element

Description

objPoints

A collection of the intersection points

objLine

The line to intersect the spline with

bProjectBack

Projects the back tangent and computes intersection points

bProjectAhead

Projects the ahead tangent and computes intersection points

object

This object
