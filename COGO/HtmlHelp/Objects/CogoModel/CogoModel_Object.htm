CogoModel Object

A CogoModel object is used to store, create, and manage coordinate geometry elements such as Points, HorzCurves, LineSegments, Profiles, and Alignments

Properties

AlignmentFactory

Sets/Gets the AlignmentFactory object

Alignments

Returns the Alignments collection

Divide

Returns an interface used to create points by dividing distances, lines, and curves

HorzCurveFactory

Sets/Gets the HorzCurveFactory object

HorzCurves

Returns the HorzCurves collection

Intersect

Returns an interface used to create points by intersecting lines and curves

LineSegmentFactory

Sets/Gets the LineSegmentFactory object

LineSegments

Returns the LineSegments collection

Locate

Returns an interface used to locate points and lines

Measure

Returns an interface used to measure angles, distances, and areas

PathFactory

Sets/Gets the PathFactory object

Paths

Returns the Paths collection

PointFactory

Sets/Gets the PointFactory object

Points

Returns the Points collection

ProfilePointFactory

Sets/Gets the ProfilePointFactory object

ProfilePoints

Returns the ProfilePoints collection

Project

Returns an interface used to locate points by projection

Tangent

Returns an interface used to locate points tangent to circles

StructuredStorage

Returns a reference to an IStructuredStorage2 interface

VertCurveFactory

Sets/Gets the VertCurveFactory object

VertCurves

Returns the VertCurves collection

Methods

Clear

Clears the entire COGO model

Clone

Creates a copy of this object

Events

OnAlignmentAdded

Fires when an Alignment object is added to the model

OnAlignmentChanged

Fires when one of the Alignment objects in the model is changed

OnAlignmentRemoved

Fires when an Alignment object is removed from the model

OnAlignmentsCleared

Fires when the Alignments collection is cleared

OnHorzCurveAdded

Fires when a HorzCurve object is added to the model

OnHorzCurveChanged

OnHorzCurveRemoved

OnHorzCurvesCleared

Fires when the HorzCurves collection is cleared

OnLineSegmentAdded

Fires when a LineSegment object is added to the model

OnLineSegmentChanged

Fires when one of the LineSegment objects in the model is changed

OnLineSegmentRemoved

Fires when a LineSegment object is removed from the model

OnLineSegmentsCleared

Fires when the LineSegments collection is cleared

OnPathAdded

Fires when a Path object is added to the model

OnPathChanged

Fires when one of the Path objects in the model is changed

OnPathRemoved

Fires when a Path object is removed from the model

OnPathsCleared

Fires when the Paths collection is cleared

OnPointAdded

Fires when a Point object is added to the model

OnPointChanged

Fires when one of the Point objects in the model changes

OnPointRemoved

Fires when a Point object is removed from the model

OnPointsCleared

Fires when the Points colleciton is cleared

Fires when a Profile object, that is associated with an Alignment object in this model, changes

OnProfilePointRemoved

Interfaces

 

IDivide

Interface used to create points by dividing distances, lines, and arcs

 

IIntersect

Interface used to locate points by intersecting lines and curves

 

ILocate

Interface used to locate points in a coordinate geometry space

 

IMeasure

Interface used to measure angles, distances, and areas

 

IProject

Interface used to locate points by projection

 

IStructuredStorage2

Interface used to store and load the cogo model

 

ITangent

Interface used to locate points tangent to circles
