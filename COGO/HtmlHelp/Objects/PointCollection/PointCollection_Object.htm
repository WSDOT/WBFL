PointCollection Object

A collection that holds Point objects. Point objects are stored in this collection by keys.

Properties

Count

Returns the number of Point objects in this collection

Factory

Set/Get the PointFactory

Item

Set/Get an item in the collection

StructuredStorage

Returns a reference to an IStructuredStorage2 interface

Methods

Add

Adds a Point object to the collection by creating one with the specified key

AddEx

Adds a Point object to the collection

Clear

Removes all the Point objects from the collection

Clone

Creates a copy of this object

FindKey

Finds a key for a Point object

Key

Returns the key for a certain position in the collection

Remove

Removes a Point object from the collection

Events

OnPointAdded

Fires when a Point object is added to the collection

OnPointChanged

Fires when one of the Point objects in the collection changes

OnPointRemoved

Fires when a Point object is removed from the collection

OnPointsCleared

Fires when the collection is cleared

Restricted Properties

_EnumPoints

Returns an interface to enumerate Point objects

_EnumKeys

Returns an interface to enumerate the keys in the collection
