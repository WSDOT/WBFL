Bearing Method (Alignment Object)

Summary:

Computes the bearing of the alignment at a given station

Syntax:

objDir = object.Bearing(varSta)

Element

Description

objDir

A Direction object representing the bearing

varSta

The station where the bearing is to be computed

object

This object

Remarks:

The bearing computed by this method is in the direction of increasing station. If the station occurs at the an angle point in the alignment, the bearing immediately before the station is computed.
