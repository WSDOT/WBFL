Alignment Object

An object that represents a horizontal roadway alignment. Alignment extends the Path object be adding a reference station at the start of the path and adding geometric computations based on stationing.

Properties

Count

Returns the number of PathElement objects in this Path

Item

Gets/Sets an item in the collection

Profile

Gets/Sets the Profile object

RefStation

Get/Set the alignment reference station

StructuredStorage

Returns a reference to an IStructuredStorage2 interface

Methods

Add

Adds a PathElement to the path

AddEx

Adds a Point, LineSegment, or HorzCurve object to the path

Bearing

Computes the bearing of the path at a given station

Clear

Removes all PathElement objects from the path

Clone

Creates a copy of this object

Insert

Inserts a PathElement at the specified position

InsertEx

Inserts a Point, LineSegment, or HorzCurve object into the path at the specified position

LocatePoint

Creates a Point object on the path at a given station

Normal

Computes the normal to the path at a given station

Offset

Computes the station and offset for a given point

ProjectPoint

Projects a point onto the path

Remove

Removes a PathElement from the path

Events

OnPathChanged

Fires when the path changes

OnProfileChanged

Fires when the profile associated with this path changes

Restricted Properties

_EnumPathElements

Returns an interface to enumerate PathElements objects
