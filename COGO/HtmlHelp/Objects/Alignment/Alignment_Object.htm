Alignment Object

An object that represents a horizontal roadway alignment. Alignment extends the Path object be adding a reference station at the start of the path and adding geometric computations based on stationing.

Properties

Count

Returns the number of PathElement objects in this Path

Item

Gets/Sets an item in the collection

Length

Returns the length of the alignment

Profile

Gets the Profile object

RefStation

Get/Set the alignment reference station

StructuredStorage

Returns a reference to an IStructuredStorage2 interface

StationEquations

Gets the StationEquationCollection object

PointFactory

Gets/Sets the PointFactory object

Methods

Add

Adds a PathElement to the path

AddEx

Adds a Point, LineSegment, or HorzCurve object to the path

Bearing

Computes the bearing of the path at a given station

Clear

Removes all PathElement objects from the path

Clone

Creates a copy of this object

CreateConnectedAlignment

Creates a copy of this Alignment with connected PathElements

CreateConnectedPath

Creates a Path object based on this Alignment with connected PathElements

CreateParallelAlignment

Creates a new Alignment object that is parallel to this one

CreateParallelPath

Creates a new path object that is parallel to this alignment

CreateSubAlignment

Creates a new Alignment object that is that copy of this Alignment between two stations

CreateSubPath

Creates a new Path object based on this Alignment between two stations

DistanceBetweenStations

Computes the distance between two stations

GetDirection

Returns the direction of a line at the specified station

Insert

Inserts a PathElement at the specified position

InsertEx

Inserts a Point, LineSegment, or HorzCurve object into the path at the specified position

Intersect

Finds the intersection of a line with the alignment

IntersectEx

Finds the intersection of a line with the alignment

LocatePoint

Creates a Point object on the path at a given station

LocateCrownPoint2D

Finds the point where a line intersects the crown point

LocateCrownPoint3D

Finds the point and elevation where a line intersects the crown point

Move

Moves the Alignment

Normal

Computes the normal to the path at a given station

Offset

Computes the station and offset for a given point

ProjectPoint

Projects a point onto the path

Remove

Removes a PathElement from the path

Events

OnAlignmentChanged

Fires when the path changes

OnProfileChanged

Fires when the profile associated with this path changes

OnStationEquationsChanged

Fires when the station equations change

Restricted Properties

_EnumPathElements

Returns an interface to enumerate PathElements objects
