CreateParallelPath Method (Alignment Object)

Summary:

Creates a new path object that is parallel to this alignment

Syntax:

path = object.CreateParallelPath(offset)

Element

Description

offset

Offset from the alignment to the new path

path

The new path

object

This object

Remarks:

If offset is less than zero, the new path is to the left of the alignment.

Horizontal curves are degraded to points if the offset is on the same side of the Alignment as the center point and the offset exceeds the radius.

Cubic splines are degraded to a point if lines normal to the spline at the control points intersect on the same side of the Alignment as the offset and the offset distance exceeds the distance from the spline to the intersection point.
