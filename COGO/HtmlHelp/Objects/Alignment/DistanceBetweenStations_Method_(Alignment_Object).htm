DistanceBetweenStations Method (Alignment Object)

Summary:

Computes the distance between two stations

Syntax:

dblDist = object.DistanceBetweenStations(varStation1,varStation2)

Element

Description

dblDist

Distance between station 1 and station 2

varStation1

Station 1

varStation2

Station 2

object

This object

Remarks:

This method essentially computes dist = (station2 - station1).
