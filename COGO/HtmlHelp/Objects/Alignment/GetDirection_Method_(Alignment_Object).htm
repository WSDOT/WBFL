GetDirection Method (Alignment Object)

Summary:

Returns the direction of a line at the specified station

Syntax:

objDirection = object.GetDirection(varStation,varOrientation)

Element

Description

objDirection

A Direction object

varStation

Station where the direction is to be computed

varOrientation

Orientation of the line whose direction is to be determined

object

This object

Remarks:

If varOrientation is the keyword Normal, the direction of a line normal to the alignment at the specified station is returned.

If varOrientation is a skew angle, the direction of the skewed line at the specified station is returned

If varOrientation is a bearing, the direction of the bearing is returned.
