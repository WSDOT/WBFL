Intersect Method (Alignment Object)

Summary:

Finds the point where a line intersects the alignment

Syntax:

objPoint = object.Intersect(objLine,objNearest)

Element

Description

objPoint

The intersection point

objLine

A line that intersects the alignment

objNearest

When multiple intersections are found, the point nearest this point is returned

object

This object
