LocateCrownPoint3D Method (Alignment Object)

Summary:

Finds the point and elevation where a line intersects the crown point

Syntax:

objPoint3D = object.LocateCrownPoint3D(varStation,varDirection)

Element

Description

objPoint3D

The point

varStation

Station where the crown point is to located from

varDirection

Direction of the line that intersects the crown point

object

This object
