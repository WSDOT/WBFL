HorzCurveCollection Object

A collection that holds HorzCurve objects. HorzCurve objects are stored in this collection by keys.

Properties

Count

Returns the number of HorzCurve objects in this collection

Factory

Set/Get the HorzCurveFactory

Item

Set/Get an item in the collection

StructuredStorage

Returns a reference to an IStructuredStorage2 interface

Methods

Add

Adds a HorzCurve object to the collection by creating one with the specified key

AddEx

Adds a HorzCurve object to the collection

Clear

Removes all the HorzCurve objects from the collection

Clone

Creates a copy of this object

FindKey

Finds a key for a HorzCurve object

Key

Returns the key for a certain position in the collection

Remove

Removes a HorzCurve object from the collection

Events

OnHorzCurveAdded

Fires when a HorzCurve object is added to the collection

OnHorzCurveChanged

Fires when one of the HorzCurve objects in the collection changes

OnHorzCurveRemoved

Fires when a HorzCurve object is removed from the collection

OnHorzCurvesCleared

Fires when the collection is cleared

Restricted Properties

_EnumHorzCurves

Returns an interface to enumerate HorzCurve objects

_EnumKeys

Returns an interface to enumerate the keys in the collection
