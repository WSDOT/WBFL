_EnumHorzCurves Property (HorzCurveCollection Object)

Summary:

Returns an interface to enumerate HorzCurve objects

Syntax:

HRESULT get__EnumHorzCurves(IEnumHorzCurves** ppEnum);

Element

Description

ppEnum

Pointer to an IEnumHorzCurves interface

Remarks:
