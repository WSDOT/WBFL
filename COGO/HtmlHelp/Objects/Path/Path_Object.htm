Path Object

An object that represents a geometric path in plan view. A path consists of PathElement objects, which are: Point objects, LineSegment objects, CubicSpline objects, and HorzCurve objects. PathElement objects are positioned end to end in the order they are defined. Straight lines are used to connect the PathElement objects together. The tangents at the beginning and end of the path are projected as straight lines.

If a Path does not contain any PathElement objects, it is assumed that the path begins at N 0.00, E 0.00, and is a straight line bearing N 90 E.

Properties

Count

Returns the number of PathElement objects in this Path

Item

Gets/Sets an item in the collection

StructuredStorage

Returns a reference to an IStructuredStorage2 interface

Methods

Add

Adds a PathElement to the path

AddEx

Adds a Point, LineSegment, CubicSpline, or HorzCurve object to the path

Bearing

Computes the bearing of the path at a given distance

Clear

Removes all PathElement objects from the path

Clone

Creates a copy of this object

CreateParallelPath

Creates a new path object that is parallel to this one

Insert

Inserts a PathElement at the specified position

InsertEx

Inserts a Point, LineSegment, CubicSpline, or HorzCurve object into the path at the specified position

LocatePoint

Creates a Point object on the path at a specified location

Move

Moves the path

Normal

Computes the normal to the path at a given distance

Offset

Computes the distance along the path and the offset from the path for a given point

ProjectPoint

Projects a point onto the path

Remove

Removes a PathElement from the path

Events

OnPathChanged

Fires when the path changes

OnProfileChanged

Fires when the profile associated with this path changes

Restricted Properties

_EnumPathElements

Returns an interface to enumerate PathElements objects
