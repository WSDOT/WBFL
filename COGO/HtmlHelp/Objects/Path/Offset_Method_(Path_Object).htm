Offset Method (Path Object)

Summary:

Computes the distance along the path and the offset from the path for a given point

Syntax:

object.Offset(objPnt,dblDist,dblOffset)

Element

Description

dblDist

The distance from the start of the path

dblOffset

The distance from the point to the path

objPnt

The point whose offset and location is to be computed

object

This object
