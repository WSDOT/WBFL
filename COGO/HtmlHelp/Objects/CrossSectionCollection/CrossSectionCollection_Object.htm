CrossSectionCollection Object

A collection that holds CrossSection objects. CrossSection objects are sorted based on the value of their station property.

This collection is also used from computing crown slopes at specified stations. The crown slopes and the crown point offset are linerally interperlated between the CrossSection objects stored in the collection.

If the collection is empty, all cross sections will be assumed to have a crown point offset of 0.0, and left and right crown slopes of 0.00.

Properties

Count

Returns the number of CrossSection objects in this collection

Factory

Set/Get the CrossSectionFactory

Item

Gets an item in the collection

StructuredStorage

Returns a reference to an IStructuredStorage2 interface

Methods

Add

Adds a CrossSection object to the collection

AddEx

Creates a new CrossSection object and adds it to the collection

Clear

Removes all the CrossSection objects from the collection

Clone

Creates a copy of this object

CreateCrossSection

Creates a cross section object for a specified station

CrownSlope

Computes the crown slope at a specified station and offset

LeftCrownSlope

Computes the crown slope on the left side of the crown point for a specified station

Remove

Removes a CrossSection object from the collection

RightCrownSlope

Computes the crown slope on the right side of the crown point for a specified station

Events

OnCrossSectionAdded

Fires when a CrossSection object is added to the collection

OnCrossSectionChanged

Fires when one of the CrossSection objects in the collection changes

OnCrossSectionRemoved

Fires when a CrossSection object is removed from the collection

OnCrossSectionsCleared

Fires when the collection is cleared
