VertCurveCollection Object

A collection that holds VertCurve objects. VertCurve objects are stored in this collection by keys.

Properties

Count

Returns the number of VertCurve objects in this collection

Factory

Set/Get the VertCurveFactory

Item

Set/Get an item in the collection

StructuredStorage

Returns a reference to an IStructuredStorage2 interface

Methods

Add

Adds a VertCurve object to the collection by creating one with the specified key

AddEx

Adds a VertCurve object to the collection

Clear

Removes all the VertCurve objects from the collection

Clone

Creates a copy of this object

FindKey

Finds a key for a VertCurve object

Key

Returns the key for a certain position in the collection

Remove

Removes a VertCurve object from the collection

Events

OnVertCurveAdded

Fires when a VertCurve object is added to the collection

OnVertCurveChanged

Fires when one of the VertCurve objects in the collection changes

OnVertCurveRemoved

Fires when a VertCurve object is removed from the collection

OnVertCurvesCleared

Fires when the collection is cleared

Restricted Properties

_EnumVertCurves

Returns an interface to enumerate VertCurve objects

_EnumKeys

Returns an interface to enumerate the keys in the collection
