HorzCurve Object

An object that represents a horizontal curve. A horizontal curve is defined by a point on the back tangent, the intersection point of the back and forward tangents, a point on the forward tangent, and the radius. Entry and/or exit clothoid spirals can be added to the curve by defining their length. Input values are denoted in red in the figure below.
