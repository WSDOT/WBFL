Offset Method (HorzCurve Object)

Summary:

Offsets the curve from its present location

Syntax:

object.Offset(dblDx,dblDy)

Element

Description

dblDx

Distance to offset the curve in the X (East-West) direction

dblDy

Distance to offset the curve in the Y (North-South) direction

object

This object
