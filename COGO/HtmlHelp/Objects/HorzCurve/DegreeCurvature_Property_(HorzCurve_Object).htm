DegreeCurvature Property (HorzCurve Object)

Summary:

Returns the degree of curvature of the circular curve

Syntax:

objDC = object.DegreeCurvature(dblD,dcType)

Element

Description

objDC

An Angle object that represents the degree of curvature

dblD

The distance upon which the degree of curvature is based

dcType

Defines the method by which the degree of curvature is computed

object

This object

Remarks:

The degree of curvature can be computed by either the arc or chord method.

Arc Method: dc = D/R

Chord Method: dc = 2*arcSin(D/2R)
