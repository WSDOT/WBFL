IncrementBy Method (StationEquationCollection Object)

Summary:

Increments the location of a station

Syntax:

object.Increment(objStation,dblInc)

Element

Description

objStation

The station to be incremented

dblInc

The increment amount

object

This object

Remarks:

If the station object is normalized, the result will also be normalized. If the station object consists of a station equation zone index and a station value, the result will be in the same form.
