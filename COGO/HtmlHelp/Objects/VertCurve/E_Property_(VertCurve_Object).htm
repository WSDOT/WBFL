E Property (VertCurve Object)

Summary:

Returns the vertical distance at any point on the curve to the tangent grade

Syntax:

dblE = object.E(dblT)

Element

Description

dblE

The vertical distance from the curve to the tangent

dblT

The distance from the start of the curve

object

This object

Remarks:

For dblT < L1, e = h(t/L1)2  otherwise e = h((L-t)/L2)2
