VertCurve Object

An object that represents a vertical curve. A vertical curve is defined by a Point on Back Grade (PBG), point of vertical intersection (PVI), a Point on Forward Grade (PFG), and the half-curve lengths. For non-symetrical vertical curves enter different values for L1 and L2. Input values are denoted in red in the figure below.

Properties

A

Returns the algebraic difference in gradients

BVC

Returns the Begin Vertical Curve point

E

Returns the vertical distance at any point on the curve to the tangent grade

EntryGrade

Returns the entry grade (g1)

EVC

Returns the End Vertical Curve point

ExitGrade

Returns the exit grade (g2)

H

Returns the vertical distance from the curve to the VPI

HighPoint

Returns the highest point on the vertical curve

K

Returns the vertical curve constant

L1

Sets/Gets the Length of the entry half of the vertical curve

L2

Sets/Gets the Length of the exit half of the vertical curve

Length

Returns the length of the vertical curve

LowPoint

Returns the lowest point on the vertical curve

PBG

Sets/Gets the Point on Back Grade

PFG

Sets/Gets the Point on Forward Grade

ProfilePointFactory

Sets/Gets the ProfilePointFactory

PVI

Sets/Gets the Point of Vertical Intersection

StructuredStorage

Returns a reference to an IStructuredStorage2 interface

Methods

Clone

Creates a copy of this object

Elevation

Computes the elevation at a given station

Grade

Computes the grade at a given station

Events

OnVertCurveChanged

Fires when the curve changes
