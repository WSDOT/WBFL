Profile Object

An object that represents the profile along the survey line of a roadway alignment. A profile consists of ProfileElement objects, which are ProfilePoint objects and VertCurve objects. ProfileElement objects are sorted by station. Straight grades are used to connect the ProfileElement objects together. The grades at the beginning and end of the profile are projected as straight grades.

If a Profile does not contain any ProfileElement objects, it is assumed that Station 0+00 is located at Elevation 0.00 and the grade is 0.00%.

Properties

Count

Returns the number of ProfileElement objects in this Profile

CrossSections

Returns a CrossSections collection

Item

Gets/Sets a ProfileElement

StructuredStorage

Returns a reference to an IStructuredStorage2 interface

Methods

Add

Adds a ProfileElement to the profile

AddEx

Adds a ProfilePoint or VertCurve object to the profile

Clear

Removes all ProfileElement objects from the profile

Clone

Creates a copy of this object

CrownSlope

Computes the crown slope at a specified station and offset

Elevation

Computes the profile elevation at a specified station

Grade

Computes the profile grade at a specified station

LeftCrownSlope

Computes the left crown slope at a specified station

Remove

Removes a ProfileElement from the profile

RightCrownSlope

Computes the right crown slope at a specified station

Events

OnProfileChanged

Fires when the profile changes

Restricted Properties

_EnumProfileElements

Returns an interface to enumerate ProfileElements objects
