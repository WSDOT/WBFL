Introduction

The WBFL Coordinate Geometry Library is an Automation compatible COGO library. With the components in this library you can, perform planar coordinate geometry operations and compute alignment geometry including stations, offsets, profile elevations, and crown slopes. This library is intended for use with small COGO models.

To get started using the the WBFL COGO Library, read the Concepts section and study the Examples.

Warning

This is a preview release of this library. The documentation, implementation, and interfaces are subject to change. Please take this into account when developing code against this library.
