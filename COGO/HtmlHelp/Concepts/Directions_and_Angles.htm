Angles, Directions and Stations

The WBFL COGO Library offers a very flexible format for specifing angles, directions and stations. Angles, and Drections can be specified in a method as a Direction (or Angle) object, a floating point value, or a text string. Stations may be specified as a Station object our a floating point value. The tables below describe how to specify directions and angles.

Angle

Method

Description

Example

Floating Point Value

A counter-clockwise angle, measured from the X axis, in radians

model.Locate.ByDistAngle(idNew,idFrom,idTo,dblDist,0.785398,dblOffset)

The angle value is p/4, or 45° 00' 00" R.

Angle object

An Angle object is used

model.Locate.ByDistAngle(idNew,idFrom,idTo,dblDist,objAngle,dblOffset)

The angle is specified with an ANgle object

Text string

A text string in the one of the following formats:

{[+|-]}ddd.ddd

ddd.ddd {[L|R|l|r]}

ddd mm ss.s {[L|R|l|r}]

{[+|-]}ddd mm ss.s

model.Locate.ByDistAngle(idNew,idFrom,idTo,dblDist,"45 00 00 L",dblOffset)

The angle is specified with a text string

Direction

Method

Description

Example

Floating Point Value

A counter-clockwise angle, measured from the X axis, in radians

model.Locate.ByDistDir(idNew,dblDist,0.785398,dblOffset)

The direction value is p/4, or N 45° 00' 00" E.

Direction Object

A Direction object is used

model.Locate.ByDistDir(idNew,dblDist,objDir,dblOffset)

The direction is specified with a Direction object

Text string

A text string in the following format:

[N|S|n|s] dd {mm {ss.s}} [E|W|e|w]

model.Locate.ByDistDir(idNew,dlbDist,"N 45 00 00 E",dblOffset)

The direction is specified with a text string

Station

Method

Description

Example

Floating Point Value

A value representing a station

point = alignment.LocatPoint(123.45,objDir)

Station object

A Station object is used

point = alignment.LocatePoint(objStation,objDir)

Notation of Text String Descriptions

Items between {} are optional. In dd {mm {ss.s}} the mm and ss.s parts are optional, but the mm part is required if the ss.s part is used.

[ ] indicates you must choose one from the list of items seperated by |. In [N|S|n|s] you must select one of N, S, n, or s.
