Angle Method (IMeasure Interface)

Summary:

Measures the angle formed by three non-coincident points

Syntax:

objAngle = object.Angle(idFromPnt,idVertex,idToPnt)

Element

Description

objAngle

The angle

idFromPnt

ID of the point the angle is measured from

idVertex

ID of the point at the vertex of the angle

idToPnt

ID of the point the angle is measured to

object

The object implementing the IMeasure interface

Remarks:

The counter-clockwise angle is computed
