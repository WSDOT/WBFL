Circles Method (IIntersect Interface)

Summary:

Finds the intersection of two circles

Syntax:

bFound = object.Circles(idNew,idCenter1,dblRadius1,idCenter2,dblRadius2,idNearest)

Element

Description

bFound

Returns True if an intersection was found

idNew

ID of the new point

idCenter

ID of the center point of the first circle

dblRadius1

Radius of the first circle

idCenter2

ID of the center point of the second circle

dblRadius2

Radius of the second circle

idNearest

ID of the point nearest the intersection

object

The object implementing the IIntersect interface
