LineSegmentCircle Method (IIntersect2 Interface)

Summary:

Finds the intersection of a stored line segment and a circle

Syntax:

objPnt = object.LineSegmentCircle(LineSeg,dblOffset,pntCenter,dblRadius,pntNearest)

Element

Description

objPnt

Intersection point (Nothing if not found)

LineSegment

The line segment

dblOffset

Offset from the line segment

pntCenter

Center point of the circle

dblRadius

Radius of the circle

pntNearest

Point nearest the intersection point

object

The object implementing the IIntersect2 interface
