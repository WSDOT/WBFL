Lines Method (IIntersect2 Interface)

Summary:

Finds the intersection of two stored line segments

Syntax:

objPnt = object.Lines(Line1,dblOffset1,Line2,dblOffset2)

Element

Description

objPnt

The interesction point (Nothing if not found)

Line1

First line

dblOffset1

Offset from the first line

Line2

Second line passes

dblOffset2

Offset from the second line

object

The object implementing the IIntersect2 interface
