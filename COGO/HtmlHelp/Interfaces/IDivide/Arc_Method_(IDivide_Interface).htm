Arc Method (IDivide Interface)

Summary:

Divides an arc into equal lengths

Syntax:

object.Arc(firstID,idInc,fromID,vertexID,toID,nParts)

Element

Description

firstID

ID of the first point

idInc

Point ID increment

fromID

ID of the point from which the arc is measured

vertexID

ID of the vertex point

toID

ID of the point to which the arc is measured

nParts

Number of parts to divide the arc into

object

The object implementing the IDivide interface

Remarks:

This method will divide an arc into nParts equally spaced points. nParts-2 points will be created. The end points of the arc are not duplicated. Points will not be generated if points with the same ID already exist in the CogoModel.
