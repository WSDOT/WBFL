Point Method (ITangent Interface)

Summary:

Locates point tangent to a circle

Syntax:

object.Point(idNew,idCenter,radius,idPoint,tangent)

Element

Description

idNew

ID of the tangent point

idCenter

ID of the center of the circle

radius

Radius of the circle

idPoint

ID of the point

tangent

Used to indicate which of the two tangent lines is used to locate the tangent points. This is a constant of type TangentSignType.

object

The object implementing the ITangent interface

Remarks:

If tangent is tsCW then the points indicated in the figure are located. If tangent is tsCCW, then the tangent points on the opposite tangent are located.
