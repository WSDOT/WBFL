HorzCurve Method (IDivide2 Interface)

Summary:

Divides a HorzCurve object

Syntax:

objPoints =object.HorzCurve(curve,nParts)

Element

Description

curve

A HorzCurve object

nParts

Number of parts to divide the curve into

objPoints

A PointCollection containing the points

object

The object implementing the IDivide2 interface

Remarks:

This method will divide a horizontal curve into nParts equally spaced points. nParts-2 points will be created. The end points of the horizontal curve (TS and ST) are not duplicated.
