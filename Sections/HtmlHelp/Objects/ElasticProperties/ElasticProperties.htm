ElasticProperties

Represents the elastic properties of a section including axial stiffness (EA) and flexural stiffness (EI). Elastic properties can be transformed into different coordinate systems using the parallel axis theorem. Elastic properties can also be transformed into shape properties.

Properties

Centroid

Location of the centroid

CoordinateSystem

Coordinate system in which the elastic properties are measured

EA

Axial stiffness

EI11

Maximum flexural stiffness

EI12Max

Flexural stiffness corresponding to the maximum product of inertia

EI12Min

Flexural stiffness corresponding to the minimum product of inertia

EI22

Minimum flexural stiffness

EIxx

Flexural stiffness about the X axis in the coordinate coordiante system

EIxy

Flexural stiffness corresponding to the product of inertia for the current coordinate system

EIyy

Flexural stiffness about the Y axis in the coordinate coordiante system

Orientation

Orientation of the current coordinate system

Origin

Origin of the current coordinate system

PrincipleDirection

Orientation of the principle coordinate system

StructuredStorage2

Returns a reference to an IStructuredStorage2 interface

Xleft

Location of the left edge of the section in the current coordinate system

Xright

Location of the right edge of the section in the current coordinate system

Ybottom

Location of the bottom edge of the section in the current coordinate system

Ytop

Location of the top edge of the section in the current coordinate system

Methods

AddProperties

Adds the properties of another ElasticProperties object to this object

TransformProperties

Transforms the elastic properties into equivalent shape properties
