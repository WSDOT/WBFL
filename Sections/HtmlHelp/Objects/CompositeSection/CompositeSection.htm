CompositeSection

A generic cross section composed of CompositeSectionItem objects.

Properties

Count

Returns the number of CompositeSectionItem objects

Item

Returns a CompositeSectionItem object

Section

Returns a reference to an ISection interface

StructuredStorage

Returns a reference to an IStructuredStorage interface

XYPosition

Returns a reference to an IXYPosition interface

Methods

AddSection

Creates a composite section item

AddSectionEx

Adds a composite section item using a CompositeSectionItem object

Remove

Removes a CompositeSectionItem
