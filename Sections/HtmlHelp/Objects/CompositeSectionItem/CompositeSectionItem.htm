CompositeSectionItem

An item in a CompositeSection object.

Properties

Density

Density of this section item's material

E

Modulus of elasticity of this section item's material

Shape

The shape of this section item

Structural

Indicates if this piece of the section is consider to be structural

StructuredStorage

Returns a reference to an IStructuredStorage2 interface

Void

Indicates if this section item represents a void in the overall cross section
