SacrificialDepth Property (CompositeBeam Object)

Summary:

Get/Set the sacrificial depth

Syntax:

object.SacrificialDepth = fltDepth
fltDepth = object.SacrificialDepth

Element

Description

fltDepth

Sacrificial depth

object

This object

Remarks:

The sacrificial depth is that portion of the roadway slab that will wear down overtime, reducing the overall depth of the slab. The sacrificial depth is considered to be non-structural.

If the sacrificial depth is set greater than the gross slab depth, the gross slab depth is made equal to the previous effective slab depth (gross depth less the previous sacrificial depth) plus the new sacrifical depth. For example if the gross slab depth is 8 and the sacrifical depth is 1, and you set the sacrificial depth to 9, the gross slab depth will be made (8-1)+9=16.
