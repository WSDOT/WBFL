GrossSlabDepth Property (CompositeBeam Object)

Summary:

Get/Set the gross slab depth

Syntax:

object.GrossSlabDepth = fltDepth
fltDepth = object.GrossSlabDepth

Element

Description

fltDepth

Gross slab depth

object

This object

Remarks:

The gross slab depth is the total depth of the slab, including the sacrificial wearing surface depth. The height of the haunch is not included in the gross slab depth
