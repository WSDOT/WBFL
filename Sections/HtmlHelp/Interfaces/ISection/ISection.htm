ISection

The ISection interface defines properties and methods applicable to all Section objects

Properties

BoundingBox

Returns the rectangle that bounds the section

ElasticProperties

Returns the elastic properties for the section

MassProperties

Returns the mass properties for the section

Methods

ClipIn

Creates a new section by removing all portions of this section that are outside the bounds of a specified rectangle

ClipWithLine

Creates a new section by removing all portions of this section that are on the right side of a specified line

Clone

Creates a clone of this section
