Introduction

The WBFL Sections Library is a library for computing elastic stiffness and mass properties of bridge cross sections. Elastic properties are generally stiffness properties such as EI and EA. Mass properties are limited to mass per unit length. Properties can be computed for non-homogeneous sections. Section properties can be transformed into equivalent properties of a homogeneous material.
