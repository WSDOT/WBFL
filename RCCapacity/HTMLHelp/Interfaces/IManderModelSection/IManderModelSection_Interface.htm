IManderModelSection Interface

Interface implemented by a confined reinforced concrete section. This interface provides information to objects implementing the IManderModel interface.

The Mander Model is a mathematical model of the stress-strain relationship of reinforced concrete.

Properties

ConfinementEffectivenessCoefficient

Returns the confinement effectiveness coefficient

TransvReinforcementRatio

Returns the transverse reinforcement ratio

TransvYieldStrength

Set/Get the yield strength of the transverse reinforcement
