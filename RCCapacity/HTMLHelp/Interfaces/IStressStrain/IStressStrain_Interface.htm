IStressStrain Interface

Interface implemented by a material model. This interface is used to describe the stress-strain relationship of a material.

Properties

ModulusOfElasticity

Returns the modulus of elasticity

Name

Sets/Gets the material name

StrainAtPeakStress

Returns the strain corresponding to the peak stress

YieldStrain

Returns the strain at yield

Methods

ComputeStress

Computes the stress in the material for a given strain

StrainLimits

Returns the strain limits for the materail
