StrainLimits Method (IStressStrain Interface)

Summary:

Returns the minimum and maximum strain for the material

Syntax:

object.StrainLimits([out]dblMin,[out]dblMax)

Element

Description

dblMin

Compressive strain limit

dblMax

Tensile strain limit

object

An object implementing the IStressStrain interface
