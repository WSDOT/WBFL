InitSlice Method (GeneralSectionSlice Object)

Summary:

Initializes this object with its various properties

Syntax:

object.InitSlice(objShape,dblA,dblCGx,dblCGy,dblStrain,dblFgStress,dblBgStress)

Element

Description

objShape

The shape of the slice

dblA

Area of the slice

dblCGx

X-ordinate of the slice centroid location

dblCGY

Y-ordinate of the slice centroid location

dblStrain

Strain at the CG of the slice

dblFgStress

Stress in the foreground material

dblBgStress

Stress in the background material

object

This object
