RCBeam2 Object

This object represents a basic reinforced concrete beam section. RCBeam2 is exactly like RCBeam except the reinforcement can be located in any number of arbitrary layers.

Properties

b

Gets/Sets the width of the flange

bw

Gets/Sets the width of the web

Eps

Gets/Sets the modulus of elasticity for prestressing steel

Es

Gets/Sets the modulus of elasticity for the mild reinforcing steel

FcBeam

Gets/Sets the ultimate concrete strength for the beam/web

FcSlab

Gets/Sets the ultimate concrete strength for the slab

fpe

Gets/Sets the effective stress in the prestressing steel after all losses

fpu

Gets/Sets the ultimate strength of the prestressing steel

fpy

Gets/Sets the yield strength of the prestressing steel

fy

Gets/Sets the yield strength of the mild reinforcing steel

h

Gets/Sets the overhall height of the section

hf

Gets/Sets the height of the flange

RebarLayerCount

Gets the number of rebar layers

RebarLayerDepth

Gets the depth of a rebar layer

RebarLayerSteel

Gets the area of steel for a rebar layer

RebarLayerDevFactor

Gets the development length factor for a rebar layer

StrandLayerCount

Gets the number of strand layers

StrandLayerDepth

Gets the depth of a strand layer

StrandLayerSteel

Gets the area of steel for a strand layer

StrandLayerDevFactor

Gets the development length factor for a strand layer

Methods

AddRebarLayer

Adds a layer of rebar

AddStrandLayer

Adds a layer of prestressing strand

ClearRebarLayers

Removes all rebar layers

ClearStrandLayers

Removes all strand layers

GetRebarLayer

Gets the depth and area of steel for a rebar layer

GetStrandLayer

Gets the depth and area of steel for a strand layer

RemoveRebarLayer

Removes a rebar layer

RemoveStrandLayer

Removes a strand layer
