AddRebarLayer Method (RCBeam2 Object)

Summary:

Adds a layer of rebar

Syntax:

object.AddRebarLayer(ds, As, devFactor)

Element

Description

object

This RCBeam2 object

ds

Depth to the reinforcing steel measured from the top of the beam

As

Area of reinforcement

devFactor

Development length factor

Remarks:

The development length factor is a measure of how developed the reinforcement is at ultimate capacity. A value of 0.0 would indicate the reinforcement is not developed at all. A value of 1.0 indicates the reinforcement is fully developed. A value between 0 and 1 indicates the reinforcement is not fully developed, but there is some degree of bond.

The development length factor is multiplied with fpy to determine the maximum stress the reinforcement can carry.
