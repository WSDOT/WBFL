MomentCurvatureSolution Object

This object represents the results of a moment curvature analysis of a general section

Properties

MaxMoment

Returns the maximum moment

Moment

Returns a moment value for a curvature step

Curvature

Returns the curvature at a curvature step

StrainPlane

Returns the strain plane for a curvature step

PointCount

Returns the number of curvature steps
