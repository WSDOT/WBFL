MomentCapacitySolution Object

This object represents the solution to a general biaxial section analysis. The location of the neutral axis is varied until the position that satisfies axial load equilibrium is found. This object is created by analyzing a general section using the MomentCapacitySolver object.

Properties

CompressionResultant

Returns the magnitude of the resultant compressive force

CompressionResultantLocation

Returns the location of the resultant compressive force

Fz

Returns the magnitude of the resultant axial force

GeneralSectionSolution

Returns a GeneralSectionSolution object

Mx

Returns the magnitude of the resultant moment about the X-axis

My

Returns the magnitude of the resultant moment about the Y-axis

NeutralAxis

Returns an object representing the neutral axis

NeutralAxisDirection

Returns the orientation of the neutral axis

StrainPlane

Returns the strain plane for the equilibrium condition

TensionResultant

Returns the magnitude of the resultant tensile force

TensionResultantLocation

Returns the location of the resultant tensile force

Methods
