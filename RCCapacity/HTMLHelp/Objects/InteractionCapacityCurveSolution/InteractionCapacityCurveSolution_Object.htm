InteractionCapacityCurveSolution Object

This object represents an interaction capacity curve. The curve can be either P-M for a fixed orientation of the neutral axis or Mx-My for a fixed axial load. The specific type will depend on the capacity solver object used to create this object (See AxialInteractuveCurveSolver and MomentInteractionCurveSolver).

Properties

MomentCapacitySolution

Returns a moment capacity solution object

SolutionPointCount

Returns the number of solution points

Methods

AddSolutionPoint

Adds a solution point to the curve
