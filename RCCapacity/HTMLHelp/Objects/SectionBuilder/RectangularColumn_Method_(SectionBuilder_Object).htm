RectangularColumn Method (SectionBuilder Object)

Summary:

Creates a GeneralSection object that represents a rectangular reinforced concrete column

Syntax:

objSection = object.RectangularColumn(dblH,dblW,dblCover,nBarsH,nBarsW,dblAs,objConcrete,objRebar)

Element

Description

objSection

The GeneralSection object

dblH

The height of the column

dblW

The width of the column

dblCover

Cover measured from the face of the column to the center of the bars

nBarsH

The number of bars in the H direction (N2)

nBarsW

The number of bars in the W direction (N1)

dblAs

The area of one reinforcing bar

objConcrete

An object implementing the IStressStrain interface that represents the concrete material

objRebar

An object implementing the IStressStrain interface that represents the reinforcing steel material

object

This object

Remarks:

For the nBarsH (N2) and nBarsW (N1) parameters, use the total number of bars in each direction. This will seem like the bars at the corners are being "doubled". This is taken into consideration and the bars are only one bar is located in each corner.
