AddSolutionPoint Method (InteractionCurveSolution Object)

Summary:

Adds a moment curvature solution for a given neutral axis direction

Syntax:

object.AddSolutionPoint(dblNA,objMC)

Element

Description

dblNA

The neutral axis direction of this solution point

objMC

The moment curvature solution for this solution point

object

This object

Remarks:

Solution points need not be entered in ascending neutral axis order. Solution points are sorted by neutral axis direction.
