InteractionCurveSolution Object

This object represents the biaxial moment interaction curve for a particular axial load. This object is created by solving an interaction curve problem using the InteractionCurveSolver object.

Properties

MomentCurvatureSolution

Returns a moment curvature solution

NeutralAxisDirection

Returns a neutral axis direction

SolutionPointCount

Returns the number points on the interaction curve

Methods

AddSolutionPoint

Adds a moment curvature solution for a given neutral axis direction
