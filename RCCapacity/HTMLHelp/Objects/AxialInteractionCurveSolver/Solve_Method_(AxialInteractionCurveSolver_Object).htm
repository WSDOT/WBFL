Solve Method (AxialInteractionCurveSolver Object)

Summary:

Computes the axial load - bending moment interaction

Syntax:

objSolution = object.Solve(dblNA,nFzSteps)

Element

Description

object

This object

dblNA

The orientation of the neutral axis

nFzSteps

Number of axial load steps (minimum of 3)

objSolution

An InteractionCapacityCurveSolution object

Remarks:

The neutral axis is measured in radians, with 0 being in the positive X direction. The neutral axis angle increases in the counter clockwise direction
