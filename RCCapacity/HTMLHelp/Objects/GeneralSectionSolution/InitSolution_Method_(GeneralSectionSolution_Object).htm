InitSolution Method (GeneralSectionSolution Object)

Summary:

Initializes the solution objects with its various properties

Syntax:

object.InitSolution(dblFz,dblMx,dblMy,objNA,objCGComp,dblC,objCGTens,dblT,arrSlices)

Element

Description

dblFz

Resultant axial force

dblMx

Resultant moment about the X-axis

dblMy

Resultant moment about the Y-axis

objNA

A Line2d object representing the neutral axis

objCGComp

A Point2d object representing the location of the compression resultant

dblC

The magnitude of the compression resultant

objCGTens

A Point2d object representing the location of the tension resultant

dblT

The magnitude of the tensile resultant

arrSlices

An array of GeneralSectionSlice objects

object

This object
