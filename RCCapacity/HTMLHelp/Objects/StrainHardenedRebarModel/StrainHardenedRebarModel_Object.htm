StrainHardenedRebarModel Object

This object represents the stress-strain relationship for mild reinforcing steel. This object implements the IStressStrain interface. This stress-strain relationship is assumed to be elastic up to the yield point and then non-linear as strain hardening is considered.

Properties

efr

Sets/Gets the fracture strain

Es

Sets/Gets the modulus of elasticity

esh

Sets/Gets the strain hardening strain

fu

Sets/Gets the ultimate stress

fy

Sets/Gets the yield stress
