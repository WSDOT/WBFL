PCISolver Object

This object implements the IRCSolver, IRCSolver2, and IRCSolver2Ex interfaces to solve reinforced concrete moment capacity problems. The method of solution based on the PCI Bridge Design Manual.

This object was developed to investigate the validity of the PCI BDM average b1 analysis method. Seguirant, Brice, Khaleghi (2004) report that this analysis method is unreliable for many cases including composite precast beams of high strength concrete. It is not recommended that this solver be used for bridge designs.

Properties

Tolerance

Gets/Sets the convergence tolerance

IsCflangeOverhangOnly

Indicates the solution parameter Cflange is to be for the overhang portion of the flange only

StrandModel

Gets/Sets the material model for computing stresses in the prestressing strands

RebarModel

Gets/Sets the material model for computing stresses in the mild reinforcement

UnitMode

Gets/Sets the specification unit mode
