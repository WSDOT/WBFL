GeneralSection Object

This object represents a general cross section of a structural member subjected to axial load and biaxial bending. This object typically represents a reinforced concrete section, but in general it can be any section consisting of any material with well defined material properties.

Properties

ShapeCount

Returns the number of shapes in the section

Shape

Returns a shape

ForegroundMaterial

Returns the foreground material for a shape

BackgroundMaterial

Returns the background material for a shape

Methods

AddShape

Adds a shape to the section
