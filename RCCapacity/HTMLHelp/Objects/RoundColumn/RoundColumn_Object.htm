RoundColumn Object

This object represents a circular reinforced column. The reinforcing in the column is assumed to be uniformly distributed around the cross section as shown below. Input and output for this object are in consistent units.

Properties

As

Gets/Sets the total area of mild reinforcing steel

Diameter

Gets/Sets the diameter of the column

Cover

Gets/Sets the cover to the center of the reinforcing

fc

Gets/Sets the concrete compressive strength

fy

Gets/Sets the yield strength of the mild reinforcing steel

Es

Gets/Sets the modulus of elasticity for the mild reinforcing steel

Methods

ComputeInteraction

Computes the interaction values
