RebarDevLengthFactor Property (NLSolver Object)

Summary:

Gets/Sets the development length factor for rebar

Syntax:

factor = object.RebarDevLengthFactor
object.RebarDevLengthFactor = factor

Element

Description

factor

Strand development length factor. In the range of 0.0 to 1.0

object

This NLSolver object

Remarks:

The development length factor is a measure of how developed the rebar are at ultimate capacity. A value of 0.0 would indicate the rebar are not developed. A factor of 1.0 indicates the rebar are fully developed. The development length factor is multiplied with fpy to determine the maximum usable rebar stress. The stress used to compute the tensile force does not exceed this value.
