NLSolver Object

This object implements both the IRCSolver, IRCSolver2, and IRCSolver2Ex interfaces to solve reinforced concrete moment capacity problems. The method of solution is to divide the cross section into small strips to numerically integrate the stress field over the cross section.

Properties

IsCflangeOverhangOnly

Indicates the solution parameter Cflange is to be for the overhang portion of the flange only

Slices

Gets/Sets the number of slices into which to divide the section

Tolerance

Gets/Sets the convergence tolerance

SlabConcreteModel

Gets/Sets the material model for computing stresses in the slab concrete

BeamConcreteModel

Gets/Sets the material model for computing stresses in the girder concrete

StrandModel

Gets/Sets the material model for computing stresses in the prestressing strands

StrandDevLengthFactor

Gets/Sets the development length factor for prestressing strands

RebarModel

Gets/Sets the material model for computing stresses in the mild reinforcement

RebarDevLengthFactor

Gets/Sets the development length factor for rebar
