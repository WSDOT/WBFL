PlasticCentroid Property (MomentCapacitySolver Object)

Summary:

Returns the plastic centroid of the section

Syntax:

pntCG = object.PlasticCentroid

Element

Description

pntCG

A Point2d object that represents the location of the plastic centroid in the global coordinate space

object

This object

Remarks:

The plastic centroid is computed by determining the location of the resultant force when the entire section is strained uniformly in compression
