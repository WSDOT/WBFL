MomentCapacitySolver Object

This object computes the external forces of a general cross section giving an axial load, angle of the neutral axis, and rotation of the section. The location of the neutral axis is varied until equilibrium is achieved.

Properties

Section

Sets/Get the section to be analyzed

Slices

Sets/Gets the number of integration slices

AxialTolerance

Sets/Gets the axial load tolerance

MaxIterations

Sets/Gets the maximum number of iterations

PlasticCentroid

Returns the plastic centroid of the section

Methods

Solve

Computes the capacity of the section

CompressionLimit

Computes the force state for the maximum compression force that can be applied to the section

TensionLimit

Computes the force state for the maximum tensile force that can be applied to the section
