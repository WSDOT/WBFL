TensionLimit (MomentCapacitySolver Object)

Summary:

Computes the force state for the maximum tensile force that can be applied to the section

Syntax:

object.TensionLimit([out]dblFz,[out]dblMx,[out]dblMz,[out]dbleo)

Element

Description

object

This object

dblFz

The maximum tensile force

dblMx

The corresponding moment about the X-axis

dblMy

The corresponding moment about the Y-axis

dbleo

The strain at (0,0)
