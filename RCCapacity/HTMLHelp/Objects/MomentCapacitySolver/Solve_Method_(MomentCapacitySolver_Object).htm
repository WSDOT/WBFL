Solve Method (MomentCapacitySolver Object)

Summary:

Computes the capacity of the section

Syntax:

objSolution = object.Solve(dblPz,dblAngle,dblK,smMethod)

Element

Description

object

This object

objSolution

A MomentCapacitySolution object

dblPz

The externally applied axial force

dblAngle

The angle of the neutral axis (radians)

dblK

The curvature of the section (radians) or strain at the extreme compression fiber (interpretation depends on the smMethod parameter)

smMethod

The solution method
