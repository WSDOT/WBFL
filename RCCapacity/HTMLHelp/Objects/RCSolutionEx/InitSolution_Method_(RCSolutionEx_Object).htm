InitSolution Method (RCSolutionEx Object)

Summary:

Initializes the solution object

Syntax:

object.InitSolution(Mn,c,fs,fps,Cflange,Cweb,T,Yflange,Yweb)

Element

Description

Mn

Nominal moment capacity

c

Depth to neutral axis

fs

Array of stresses in the rebar

fps

Array of stress in the prestressing steel

Cflange

Resultant compression force in flange

Cweb

Resutlant compression force in web

Yflange

Depth to resultant compression force in flange

Yweb

Depth to resultant compression force in the web

object

This RCSolutionEx object
