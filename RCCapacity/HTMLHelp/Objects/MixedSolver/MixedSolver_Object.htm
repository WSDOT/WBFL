MixedSolver Object

This object implements the IRCSolver interface to solve reinforced concrete moment capacity problems. The method of solution is a combination of the strength equations from the AASHTO Standard Specifications and the stress in the prestresssing steel at ultimate (fps) from the AASHTO LRFD specifications. This, albiet strange, solver is a proposed method for refining the AASHTO LRFD specifications.

Properties

IsCflangeOverhangOnly

Indicates the solution parameter Cflange is to be for the overhang portion of the flange only
