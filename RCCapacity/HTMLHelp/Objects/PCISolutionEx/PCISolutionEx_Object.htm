PCISolutionEx Object

This object represents the solution to a moment capacity analysis. This is a specialization of the RCSolution object that includes parameters particular to the PCI BDM solution method.

Properties

AvgBeta1

Returns the average b1 value.

Cflange

Returns the compression force in the flange

Cweb

Returns the compression force in the web

fs

Returns the stresses in the rebar

fps

Returns the stresses in the prestressing strands

Mn

Returns the nominal moment capacity of the section

NeutralAxisDepth

Returns the depth to the neutral axis

T

Returns the resultant tensile force

Yflange

Returns the depth to the resultant compression force in the flange

Yweb

Returns the depth to the resultant compression force in the web

Methods

InitSolution

Initializes the solution object
