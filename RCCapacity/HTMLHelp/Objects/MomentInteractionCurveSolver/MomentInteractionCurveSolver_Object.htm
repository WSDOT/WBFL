MomentInteractionCurveSolver Object

This object represents an object that creates the data for a Mx-My Biaxial Bending interaction diagram for a fixed magnitude of axial force.

Properties

AxialTolerance

Sets/Gets the axial load tolerance

MaxIterations

Sets/Gets the maximum number of iterations

Section

Sets/Gets the GeneralSection object that will be analyzed

Slices

Sets/Gets the number of integration slices

Methods

CompressionLimit

Computes the force state for the maximum compression force that can be applied to the section

TensionLimit

Computes the force state for the maximum tensile force that can be applied to the section

Solve

Computes the axial-moment interaction
