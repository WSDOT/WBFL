Solve Method (GeneralSectionSolver Object)

Summary:

Solves the general section analysis problem

Syntax:

objSolution = object.Solve(objStrainPlane)

Element

Description

object

This object

objStrainPlane

A Plane3d object representing the strain plane

objSolution

A GeneralSectionSolution object representing the solution of the analysis

Remarks:

The z axis of the Plane3d object is considered to be the strain axis. Plane3d is defined in the WBFL Geometry library
