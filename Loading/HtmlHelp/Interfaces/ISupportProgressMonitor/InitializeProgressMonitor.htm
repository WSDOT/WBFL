InitializeProgressMonitor Method (ISupportProgressMonitor Object)

Summary:

This method sets the IProgressMonitor interface to a process whose progress is to be be tracked

Syntax:

ISupportProgressMonitor.InitializeProgressMonitor( [in]objProgressMonitor, [in]intCookie )

Element

Description

objProgressMonitor

The IProgressMonitor interface

intCookie

The cookie that will be used to track this process. The process must save this cookie and use it when it sends progress messages

ISupportProgressMonitor

The ISupportProgressMonitor object
