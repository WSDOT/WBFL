WasCancelled Property (IProgressMonitor Object)

Summary:

Check to see if the current process was cancelled

Syntax:

boolWasCancelled = IProgressMonitor.WasCancelled

Element

Description

boolWasCancelled

If true, the process should be cancelled

IProgressMonitor

The IProgressMonitor object

Remarks:

Implementors of this interface should check this value frequently and must react to this message and work to cancel the process if this method returns true.
