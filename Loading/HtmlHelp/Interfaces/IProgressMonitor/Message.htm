Message Property (IProgressMonitor Object)

Summary:

Set the Message to be displayed by the progress display

Syntax:

IProgressMonitor.Message ( intCookie, bstrMessage )

 

Element

Description

bstrMessage

The Message string

intCookie

An integer value that identifies the sender of this message

IProgressMonitor

The IProgressMonitor object
