GaugeValue Property (IProgressMonitor Object)

Summary:

Set the value of the progress gauge from 0 to 100

Syntax:

IProgressMonitor.GaugeValue ( intCookie, intGaugeValue )

 

Element

Description

intCookie

An integer value that identifies the sender of this message

intGaugeValue

The GaugeValue

IProgressMonitor

The IProgressMonitor object

Remarks:

Limits on the gauge are zero to 100.
