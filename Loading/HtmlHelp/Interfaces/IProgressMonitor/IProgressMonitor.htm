IProgressMonitor

An Interface to be used by processes for displaying their status. The properties allow a process to display its percentage complete, display a message about what it is doing, and check to see if a user has cancelled the process.

Properties

GaugeValue

Set the value of the progress gauge from 0 to 100

Message

Set the Message to be displayed by the progress display

WasCancelled

Check to see if the current process was cancelled

 

Remarks

Processes should check the WasCancelled property regularly and stop in a timely manner if the property is true.
