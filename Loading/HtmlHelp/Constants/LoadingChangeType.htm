Loading ChangeType

Used as a parameter from an event. Designates what type of change took place during the event.

Constant

Value

Description

chgDescription

1

A textual description changed

chgLiveLoad

2

A vehicle changed

chgUserItemData

3

User-defined ItemData changed

chgLoading

4

A load group or load case changed

chgCombination

5

A load combination changed
