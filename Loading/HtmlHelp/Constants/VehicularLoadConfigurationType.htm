VehicularLoadConfigurationType

Designates how truck and lane results are to be combined for a VehicularLoadConfiguration.

Constant

Value

Description

vlcDefault

0

Default for system, if used a property of VehicularLoadConfiguration, defaults to vlcTruckPlusLane.

vlcTruckOnly

1

Apply results of truck (axles) only.

vlcLaneOnly

2

Apply results of lane load only

vlcTruckPlusLane

3

Apply sum of results of truck plus lane load

vlcTruckLaneEnvelope

4

Apply envelope of results of truck plus lane load

vlcSidewalkOnly

5

Apply results of sidewalk load only
