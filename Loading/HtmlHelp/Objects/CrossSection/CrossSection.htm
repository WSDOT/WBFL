CrossSection

Represents the prismatic structural section properties along a Segment.

Properties

Depth

Total depth of this CrossSection

EADefl

Axial stiffness used for deflection calculations

EAForce

Axial stiffness used for force calculations

EIDefl

Bending stiffness used for deflection calculations

EIForce

Bending stiffness used for force calculations

StressPoints

Collection of StressPoints for this CrossSection

ThermalCoeff

Coefficient of thermal expansion

Methods

Clone

Creates a clone of this CrossSection

GetStiffness

Gets all stiffness parameters in a single call

SetStiffness

Sets all stiffness parameters in a single call

Events

OnCrossSectionChanged

Fired whenever a change is made to the CrossSection or its parts

Interfaces

These interfaces are obtained via QueryInterface.
