GetStiffness Method (CrossSection Object)

Summary:

Gets all stiffness properties in a single call

 

Syntax:

CrossSection.GetStiffness([out] dblEAForce, [out] dblEIForce, [out] dblEADefl, [out] dblEIDefl)

 

Element

Description

dblEAForce

Axial stiffness to be used for force response calculation

dblEIForce

Bending stiffness to be used for force response calculation

dblEADefl

Axial stiffness to be used for deflection response calculation

dblEIDefl

Axial stiffness to be used for deflection response calculation

CrossSection

The CrossSection object
