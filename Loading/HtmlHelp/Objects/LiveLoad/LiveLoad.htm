LiveLoad

The top-level categories of LiveLoadModels

Properties

Each of the properties in the table below provide a LiveLoadModel for the category given. In reality, these categories are arbitrary - they simply serve as placeholders to allow LiveLoadModels to be mapped into LoadCombinations. There is a one-to-one mapping between these categories and the LiveLoadModelType enums.
