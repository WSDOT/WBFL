AddLoadCaseFactor Method (LoadCombination Object)

Summary:

Add a LoadCase, and the factors that are to be applied to it, to the LoadCombination

Syntax:

LoadCombination.AddLoadCaseFactor([in]bstrLoadCase, [in]dblMinFactor, [in]dblMaxFactor)

Element

Description

bstrLoadCase

Name of LoadCase to be added

dblMinFactor

Factor to be applied when response value does not maximize the effect being considered

dblMaxFactor

Factor to be applied when response value maximizes the effect being considered

LoadCombination

The LoadCombination object

Remarks:

Refer to LRFD 3.4.1 for more discussion on max and min factors

If you only have one factor, set the min and max factors to the same value.
