SetG Method (DistributionFactor Object)

Summary:

Set all distribution factors in one call

 

Syntax:

DistributionFactor.SetG( dblPMSgl, dblPMMul, dblNMSgl, dblNMMul, dblVSgl, dblVMul, dblDSgl, dblDMul, dblRSgl, dblRMul, dblTSgl, dblFat)

 

Element

Description

dblPMSgl

Distribution factor for positive moment - single lane

dblPMMul

Distribution factor for positive moment - multiple lanes loaded

dblNMSgl

Distribution factor for negative moment - single lane

dblNMMul

Distribution factor for negative moment - multiple lanes loaded

dblVSgl

Distribution factor for shear - single lane

dblVMul

Distribution factor for shear - multiple lanes loaded

dblDSgl

Distribution factor for deflection - single lane

dblDMul

Distribution factor for deflection - multiple lanes loaded

dblRSgl

Distribution factor for reactions - single lane

dblRMul

Distribution factor for reactions - multiple lanes loaded

dblTSgl

Distribution factor for rotation - single lane

dblFat

Distribution factor for fatigue loads - used to calculate stresses

DistributionFactor

The DistributionFactor object

Remarks:

All values must be positive
