LiveLoadModel

Represents a collection of Truck/Lane combinations (VehicularLoads) whose results are enveloped.

Properties

Each of the properties in the table below provide a LiveLoadModelModel for the category given. In reality, these categories are arbitrary - they simply serve as placeholders to allow LiveLoadModelModels to be mapped into LoadCombinations. There is a one-to-one mapping between these categories and the LiveLoadModelType enums.

Name

Name for live load model - used in reports

DistributionFactorType

Constant of type DistributionFactorType indicating whether to use single lane or multiple lane distribution factors in the analysis

VehicularLoads

Collection of truck/lanes to be enveloped for this live load model

Methods

Clear

Clears all data from this LiveLoadModel

Clone

Make a deep clone

Events

OnLiveLoadModelChanged

Fired whenever a change is made to the LiveLoadModel

Interfaces

These interfaces are obtained from the LiveLoadModel via QueryInterface

IStructuredStorage2

Persistence of LBAM data
