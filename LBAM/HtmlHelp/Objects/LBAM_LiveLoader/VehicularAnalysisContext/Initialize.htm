Initialize Method (IVehicularAnalysisContext Object)

Summary:

Initialize the context with the interfaces it needs

Syntax:

IVehicularAnalysisContext.Initialize( objILBAMModel, objInfluenceLineResponse, objIContraflexureResponse, objAnalysisPOIs, objGetDistributionFactors, objGetStressPoints )

Element

Description

objILBAMModel

The LBAMModel that is to be analyzed

objInfluenceLineResponse

IInfluenceLineResponse object that was initialized for the model

objIContraflexureResponse

IIContraflexureResponse object that was initialized for the model

objAnalysisPOIs

IAnalysisPOIs object that was initialized for the model

objGetDistributionFactors

IGetDistributionFactors object that was initialized for the model

objGetStressPoints

IGetStressPoints object that was initialized for the model

IVehicularAnalysisContext

The IVehicularAnalysisContext object

Remarks:

Note that all of the interfaces passed into this method must be initialized to the same model. Really strange things are likely to happen if they are not.
