LiveLoadModelResponse

This interface computes the enveloped results for a specified LiveLoadModel. The result is computed by enveloping the enveloped responses for all of the VehicularLoad's contained by the LiveLoadModel.

Results are be requested for multiple POI's (or Support's) and are returned in a collection that is in the same order as the request.
