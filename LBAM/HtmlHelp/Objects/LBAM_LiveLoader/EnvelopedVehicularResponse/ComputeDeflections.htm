ComputeDeflections Method (EnvelopedVehicularResponse Object)

Summary:

Compute the optimized deflection response for the given LiveLoadModel

Syntax:

objResults = EnvelopedVehicularResponse.ComputeDeflections( [in]varPOIs, [in]bstrStage, [in]intLiveLoadModelType, [in]intVehicleIndex, [in]intForceEffect, [in]intOptimization, [in]intVehicularLoadConfiguration, [in]boolApplyImpact, [in]boolApplyDistribution, [in]boolComputePlacement)

Element

Description

objResults

LiveLoadModelSectionResults collection containing results at POIs

varPOIs

ILngArray of POI ID's (longs) where results are to be computed

bstrStage

Name of Stage when results are to be computed

intLiveLoadModelType

LiveLoadModelType describing which live load model to use

intVehicleIndex

Index of VehicularLoad in LiveLoadModel to be optimized

intForceEffect

ForceEffectType of force to be computed/optimized

intOptimization

OptmizationType - whether to minimize or maximize the force effect

intVehicularLoadConfiguration

VehicularLoadConfigurationType describing how to combine truck and lane loads

boolApplyImpact

If true, apply live load impact factors

boolApplyDistribution

If true, apply live load distribution factors. If false, use 1.0

boolComputePlacement

If true, create a LiveLoadConfiguration object that describes the truck/lane configuration that produced the optimized response and make the configuration a part of the results

EnvelopedVehicularResponse

The EnvelopedVehicularResponse object

Remarks:

Deflections are computed in global coordinates

Results collection is in same order as varPOIs array
