ComputeSupportDeflections Method (BasicVehicularResponse Object)

Summary:

Compute deflections at supports for the given live load configuration

Syntax:

objResults = BasicVehicularResponse.ComputeSupportDeflections( [in]varSupports, [in]bstrStage, [in]objLiveLoadConfiguration )

Element

Description

objResults

Result3Ds collection containing results at POIs

varSupports

ILngArray of Support id's (longs) where results are to be computed

bstrStage

Name of Stage when results are to be computed

objLiveLoadConfiguration

LiveLoadConfiguration that describes live load placement

BasicVehicularResponse

The BasicVehicularResponse object

Remarks:

Deflections are computed in global coordinates

Results collection is in same order as varSupports array
