LiveLoadConfiguration

This object represents a snapshot of the application of a live load on the LBAMModel. It gives all pertinant information as to how and where truck and lane loads are applied. It is generally an optional part of a live load results object.

Properties

AxleConfig

An array of longs that indicates which truck axles are inactive for the current configuration.

DistributionFactorType

The the type of distribution factor that was applied for this vehicular load

DoApplyImpact

True if impact was applied to this live load response

ForceEffect

The force effect that was optimized for this live load response

IsApplicable

If true, the live load was applicable for the given configuration

LiveLoadModel

The type of live load model used for this configuration

Optimization

Type of optimization used for this configuration

TruckDirection

Direction that truck is travelling

TruckPosition

Position of front axle of truck measured in LBAM global coordinates

VariableSpacing

Value of variable spacing used to apply the truck load

VehicleIndex

The index of the VehicularLoad in the LiveLoadModel that was applied

VehicularLoadConfiguration

Describes what components of the VehicularLoad that were applied for this configuration

Methods

Clone

Create a clone of this configuration

Events

OnLiveLoadConfigurationChanged

Fired when a configuration changes

Interfaces

These interfaces are obtained via QueryInterface

IStructuredStorage2

Persistence of LBAM data
