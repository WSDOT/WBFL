SpanPOIIncrement Property (IAnalysisPOIs)

Summary:

This value defines the minumum number of increments between POI's placed along Spans. This in turn defines the maximum POI spacing, which is the span length divided by this property. If user-defined POI's do not meet this requirement, then the analysis engine will automatically generate POI's in order to meet minimum spacings.

Syntax:

intSpanPOIIncrement = IAnalysisPOIs.SpanPOIIncrement
IAnalysisPOIs.SpanPOIIncrement = intSpanPOIIncrement

 

Element

Description

intSpanPOIIncrement

The minumum number of increments between POI's placed along Spans.

IAnalysisPOIs

The IAnalysisPOIs object

Remarks:

The default is 10. This means that 11 POI's are placed along every Span

The minimum value is 1, which means a POI at each end of every Span. However you should be aware that the spacing between points along influence lines is governed by POI spacing. If you do not have enough POI's you will not get good influence lines for live load analyses.

Automatically-generated POI's are given negative ID's and are only generated if there are not enough user-defined POI's to meet maximum spacing requirements.
