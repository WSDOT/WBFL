ComputeAreaInRegions Method (InfluenceLine Object)

Summary:

Compute area under the influence line between given regions

Syntax:

dblArea = InfluenceLine.ComputeAreaInRegions varRegions

Element

Description

varRegions

IDblArray array of doubles that defines regions. This array must have an even number of values.

dblArea

The computed area

InfluenceLine

The InfluenceLine object

Remarks:

Regions must be within the bounds of the InfluenceLine
