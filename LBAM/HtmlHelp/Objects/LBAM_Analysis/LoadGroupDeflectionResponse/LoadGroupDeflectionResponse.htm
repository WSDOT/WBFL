LoadGroupDeflectionResponse

This object implements the ILoadGroupResponse interface. The properties used to compute stiffnesses are the Deflection properties for the LBAMModel.

Results are be requested for multiple POI's (or Support's) and are returned in a collection that is in the same order as the request.

Interfaces

This interfaces may be obtained via QueryInterface from the LoadGroupForceResponse object

ILoadGroupResponse

Compute the reponse due to member loads in LoadGroups

IAnalysisPOIs

Defines the minimum spacing between POI's that are to be placed on the superstructure for analysis

IContraflexureResponse

Computes the zones along the superstructure where a negative moment response occurs due to a unit uniform load applied along the entire superstructure. These zones are called contraflexure zones.

DependOnLBAM

Used to Set/Get the LBAMModel associated with this response object.

IGetActiveLoadGroups

Interface used to determine which LoadGroups actually have loads

IGetCrossSection

Interface to determine the cross section at a given location

IGetDistributionFactors

Interface to determine the live load distributon factors at a given location

IGetFemForLoadGroupResponse

Interface to save the generated FEM model used to analyze the LBAMModel to a file

IGetStressPoints

Interface to determine the stress points at a given location

IInfluenceLineResponse

Interface to compute influence lines for a given location and Stage
