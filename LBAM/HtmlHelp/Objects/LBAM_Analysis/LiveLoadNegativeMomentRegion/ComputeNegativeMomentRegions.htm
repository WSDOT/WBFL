ComputeNegativeMomentRegions Method (ILiveLoadNegativeMomentRegion)

Summary:

Compute the regions of negative moment, as defined by LRFD, along the superstructure

Syntax:

varLocations = LiveLoadNegativeMomentRegion.ComputeContraflexureLocations ([in] bstrStage )

Element

Description

varLocations

IDblArray array of doubles describing boundaries of negative moment regions along the superstructure.

bstrStage

Stage in which regions are to be computed

LiveLoadNegativeMomentRegion

The LiveLoadNegativeMomentRegion object

Remarks:

These regions are used to determine where truck/train negative moment results are applicable as per LRFD 3.6.1.3.

Note that this is similar to, but not the same as, the contraflexure regions across the structure. For example, results are not computed on the outside half of exterior spans that do not have any positive uniform load moment.

Pairs of values in array define the starts and ends of the negative moment regions.

If array is empty, there is no negative moment along the superstructure due to a unit uniform load
