IsPOIInNegativeLiveLoadMomentZone Property (ILiveLoadNegativeMomentRegion)

Summary:

Test if the given POI is in zones where negative moment due to VehicularLoad's that have their Applicability property set to llaContraflexure

Syntax:

boolIsInZone = LiveLoadNegativeMomentRegion.IsPOIInContraflexureZone([in]intPoiID, [in] bstrStage)

Element

Description

boolIsInZone

Boolean value that is true if the POI lies within the contraflexure zone

intPoiID

ID of POI that is to be checked

bstrStage

Stage in which contraflexure locations are to be computed

LiveLoadNegativeMomentRegion

The LiveLoadNegativeMomentRegion object
