ComputeDeflectionInfluenceLine Method (IInfluenceLineResponse)

Summary:

Compute influence line for deflection response at the given POI

Syntax:

InfluenceLineResponse.ComputeDeflectionInfluenceLine( [in]intPOIID, [in]bstrStage, [in]intForceEffect, [out]objLeftInfluenceLine, [out]objRightInfluenceLine)

Element

Description

intPOIID

ID of POI for which response is desired

bstrStage

Stage for reponse

intForceEffect

ForceEffectType for reponse

objLeftInfluenceLine

The influence line either at, or immediately to the left of the POI in question

objRightInfluenceLine

The influence line immediately to the right of the POI in question. This object is only computed if there is a need to have two unique influence lines at the location in question, otherwise it is NULL.

IInfluenceLineResponse

The IInfluenceLineResponse object

 

Remarks:

 

This method will only return a right influence line if it is needed at the location in question. Otherwise it will be NULL and the left influence line will be the only valid result object. This is done for efficiency reasons because most of the time, the left influence line is the only one needed.
