GetActiveLoadGroups Method (IGetActiveLoadGroups)

Summary:

Returns a list of names of LoadGroups that actually have loads

Syntax:

varActiveLoadGroups = GetActiveLoadGroups.GetActiveLoadGroups

 

Element

Description

varActiveLoadGroups

IBstrArray containing the names of the LoadGroups that actually have loads.

GetActiveLoadGroups

The GetActiveLoadGroups object

Remarks:

This method searches through applied loads and LoadGroups for all Stages in the model to see which LoadGroups have applied member or support loads
