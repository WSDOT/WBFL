LBAMModelEnveloper

Object that allows you to envelope the combined responses from two or more LBAMModels.

NOTE: All models that are enveloped using this object must have matching POI's. The POI's must have identical ID's and be located at the same location within the model - Otherwise, you will not get the results you expect.

Note that the Initialize method must be called once, and only once, before using this object
