InitializeProgressMonitor4E Method (ISupportProgressMonitorForEngine Object)

Summary:

Intialize the analysis components of the LBAMAnalysisEngine with a progress monitoring interface

Syntax:

ISupportProgressMonitorForEngine.InitializeProgressMonitor4E([in]objProgressMonitor, [in]intLoadGroupCookie, [in]intLiveLoadCookie, [in]intLoadCombinationCookie)

Element

Description

objProgressMonitor

The progress monitor object that is to be wired to the analysis engine

intLoadGroupCookie

Cookie to be given the the components that compute load group responses.

intLiveLoadCookie

Cookie to be given the the components that compute live load responses.

intLoadCombinationCookie

Cookie to be given the the components that compute load combination responses.

ISupportProgressMonitorForEngine

The ISupportProgressMonitorForEngine object

Remarks:

The cookies can be used to filter or categorize progress messages coming back from the analysis engines
