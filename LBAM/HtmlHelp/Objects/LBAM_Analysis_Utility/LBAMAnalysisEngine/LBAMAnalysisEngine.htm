LBAMAnalysisEngine

Object that provides a simple way of initializing and creating the interfaces necessary to perform a complete analysis of an LBAMModel. It is recommended to use this object for all of your analysis needs unless you plan to modify the LBAM analysis engines themselves.

Note that the Initialize method must be called once, and only once, before using this object
