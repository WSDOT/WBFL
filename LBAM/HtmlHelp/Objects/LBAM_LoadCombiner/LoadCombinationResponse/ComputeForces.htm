ComputeForces Method (LoadCombinationResponse Object)

Summary:

Compute optimized force responses for the given load combination

Syntax:

objResults = LoadCombinationResponse.ComputeForces(bstrCombination, varPOIs, bstrStage, intOrientation, intResultsSummationType, intForceEffectType, intOptimizationType, boolIncludeLiveLoad, boolIncludeImpact, boolComputeConfig)

Element

Description

objResults

LoadCombinationSectionResults collection containing results at POIs

bstrCombination

Name of LoadCombination for which results are requested.

varPOIs

ILngArray of POI id's (longs) where results are to be computed

bstrStage

Name of Stage when results are to be computed

intOrientation

ResultsOrientation defining orientation of results

intResultsSummationType

ResultsSummationType that defines how results are to be summed over stages

intForceEffectType

ForceEffectType to be optimized

intOptimizationType

OptmizationType that defines whether we are minimizing or maxmizing

boolIncludeLiveLoad

If true, include live load responses in combination. If false, don't

boolIncludeImpact

If true, include impact in combined response, otherwise don't

boolComputeConfig

If true, Create a LoadCombinationResultConfiguration and include it with the results

LoadCombinationResponse

The LoadCombinationResponse object

Remarks:

Results collection is in same order as varPOIs array
