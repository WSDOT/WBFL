LoadCombinationResultConfiguration

This object represents a snapshot of the application of a load combination on the LBAMModel. It gives all pertinant information as to the factors that were used and how and where truck and lane loads are applied. It is returned by load combination optmization routines, so it can be used along with the ConcurrentLoadCombinationResponse interface to compute concurrent responses.

Properties

LoadCaseFactorCount

The number of LoadCase/Factors for this combination configuration

LiveLoadConfiguration

The object that describes the configuration of the live load that was used to optmize the response for this load combination

LiveLoadFactor

The load combination factor that was applied to the live load portion of the combination

SummationType

Defines whether results were summed over stages or incremental for the given stage

Methods

AddLoadCaseFactor

Add a LoadCase, and the factor that was applied to it, to the configuration

GetLoadCaseFactor

Get the ith LoadCase, and the factor that was applied to it, from the configuration
