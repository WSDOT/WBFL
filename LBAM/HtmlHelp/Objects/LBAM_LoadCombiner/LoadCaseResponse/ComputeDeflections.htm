ComputeDeflections Method (LoadCaseResponse Object)

Summary:

Compute deflections for the given LoadCase

Syntax:

objResults = LoadCaseResponse.ComputeDeflections([in]bstrLoadCase, [in]varPOIs, [in]bstrStage, [in]intResultsSummationType)

Element

Description

objResults

SectionResult3Ds collection containing results at POIs

bstrLoadCase

Name of the LoadCase that results are to be computed for

varPOIs

ILngArray of POI id's (longs) where results are to be computed

bstrStage

Name of Stage when results are to be computed

intResultsSummationType

How results are to be summed over stages

LoadCaseResponse

The LoadCaseResponse object

Remarks:

Deflections are computed in global coordinates

Results collection is in same order as varPOIs array

Also See:
