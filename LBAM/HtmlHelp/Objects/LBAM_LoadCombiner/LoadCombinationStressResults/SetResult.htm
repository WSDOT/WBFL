SetResult Method (LoadCombinationStressResults Object)

Summary:

Set a result in the LoadCombinationStressResults collection

Syntax:

LoadCombinationStressResults.SetResult( [in]intIndex, [in]objLeftStressResult, [in]objLeftConfiguration,[in]objRightStressResult, [in]objRightConfiguration)

Element

Description

intIndex

Index (order) within LoadCombinationStressResults collection of item desired

objLeftStressResult

StressResult to the left of the section

objLeftConfiguration

The left LoadCombinationResultConfiguration

objLeftStressResult

StressResult to the right of the section

objLeftConfiguration

The right LoadCombinationResultConfiguration

LoadCombinationStressResults

The LoadCombinationStressResults object

Remarks:

The objConfiguration object can be NULL if no configuration information was requested
