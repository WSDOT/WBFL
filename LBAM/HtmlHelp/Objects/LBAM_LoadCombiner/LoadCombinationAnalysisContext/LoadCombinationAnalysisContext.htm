LoadCombinationAnalysisContext

This object is used to supply needed interfaces and events to live load analysis objects. Analysis objects are initialized to a LoadCombinationAnalysisContext object via the IDependOnLoadCombinationAnalysisContext interface. This means that you can use live load analysis objects (engines) to analyze models other than the LBAMModel simply by creating your own context object that implements the interfaces and events listed below. However, this LoadCombinationAnalysisContext is LBAM-dependent

Properties

 

 

Methods

Initialize

Initialize the context with the interfaces it requires from the LBAM

Events

OnResultsChanged

Fired when the reponses that the combination depends on change.

OnCombinationChanged

Fired when load combination data changes

Interfaces

This interfaces may be obtained via QueryInterface from the LoadCombinationAnalysisContext object.

IAnalysisPOIs

Defines the minimum spacing between POI's that are to be placed on the superstructure for analysis

IContraflexureResponse

Computes the zones along the superstructure where a negative moment response occurs due to a unit uniform load applied along the entire superstructure. These zones are called contraflexure zones.

ISupportLocations

Used to determine the locations of supports and superstructure dimensions

ILiveLoad

Describes the applied live loads

IGetStressPoints

Interface to determine the cross section at a given location

IGetDistributionFactors

Interface to determine the live load distributon factors at a given location

IGetStressPoints

Interface to determine the stress points at a given location

IInfluenceLineResponse

Interface to compute influence lines for a given location and Stage
