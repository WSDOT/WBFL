LeftSupport Method (In Model)

Summary:

Returns the Support on the left hand side of the specified span

 

Syntax:

objSupport = Model.LeftSupport(intSpan)

 

Element

Description

objSupport

The Support

Model

The Model object

intSpan

Zero-based index of the Span where the desired Support connects to.

Remarks:

Returns NULL if Support does not exist.
