ComputeLocation Method (In Model)

Summary:

Computes the a location in global coordinates based on a local location along a member

 

Syntax:

Model.ComputeLocation( [in] intMbrID, [in] mbrType, [in] dblMbrLocation, [out]dblXLoc, [out]dblYLoc)

 

Element

Description

intMbrID

Index of member where location is to be calculated. For Spans, SuperstructureMembers, and Supports, this represents the zero-based location from the left end of the model. For TemporarySupports, this is the unique ID of the support.

mbrType

Type of member (of type MemberType) where location is to be computed.

dblMbrLocation

Distance, measured from start of member, along member where location is to be computed. Positive values indicate placement in base units. Negative values indicate fractional spacing and may range from -1.0 to 0.0.

dblXLoc

X coordinate of location measured in global model coordinates

dblYLoc

Y coordinate of location measured in global model coordinates

Model

The Model object

Remarks:

Refer to Sign Conventions for a description of global model coordinates
