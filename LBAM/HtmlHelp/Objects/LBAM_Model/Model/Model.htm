LBAMModel

This is the top-level model for the LBAM system. It serves as the front door for access to all LBAM functionality.

Properties

DistributionFactors

Collection of distribution factors along the model

Spans

Collection of Span members in the model

SuperstructureMembers

Collection of members making which describe the properties of the superstructure.

Supports

Collection of Supports in the model

POIs

Collection of POI's in the model

Stages

Collection of Stages in the model

LoadCases

Collection of LoadCases's that are associated with this LBAM

LoadGroups

Collection of LoadGroup's that are associated with this LBAM

LiveLoad

Live loads applied to this LBAM

LoadCombinations

Collection of load combinations in this model

PointLoads

Collection of PointLoads applied to this Model

DistributedLoads

Collection of DistributedLoads applied to this Model

TemperatureLoads

Collection of TemperatureLoads applied to this Model

SettlementLoads

Collection of support SettlementLoads applied to this Model

StrainLoads

Collection of StrainLoads Applied to this Model

Methods

Clear

Clear all members, loads, POI's, and results from the model. Sets everything back to default conditions

Clone

Clones entire model including geometry, loading, and load combinations

ComputeLocation

Computes the X-Y coordinates of a location on the LBAM

RightSupport

Returns the support on the right hand side of a span

LeftSupport

Returns the support on the left hand side of a span

RightSpan

Returns the span on the right hand side of a support

LeftSpan

Returns the span on the left hand side of a support

RemoveStage

Removes a given Stage and all data associated with the Stage

Events

OnModelChanged

Fired whenever a change is made to the LBAM

Interfaces

These interfaces are obtained from the Model via QueryInterface

IStructuredStorage2

Persistence of LBAM data
