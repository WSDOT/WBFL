LoadCase

LoadCases provide a way to accumulate the reponses of multiple LoadGroups in an LBAM. For example, you might have LoadCases with the names "Interior Barrier Loads" and "Exterior Barrier Loads and you would like to group them all together into a LoadCase to represent the weight of all traffic barriers on the structure. You would then use this LoadCase when using an analysis engine to request the sum of repsonses due to barrier loads.

LoadCases may then be assigned in LoadCombinations where they are factored and summed together to create limit states. Refer to the section on Load Combination Hierarchy for more information

Properties

Name

Name of the LoadCase. This name is used to refer to the LoadCase

Description

A text description of the LoadCase - used for reporting purposes

ItemData

VARIANT data member that can be used to store user-specific data.

LoadGroupCount

Total number of LoadGroups in this LoadCase

Methods

AddLoadGroup

Adds a LoadGroup to this LoadCase

Clone

Creates a clone of this LoadCase

GetLoadGroup

Retrieve a LoadGroup by index

RemoveLoadGroup

Remove a LoadGroup by name

RemoveLoadGroupByIndex

Remove a LoadGroup using index

SetLoadGroup

Replace an existing LoadGroup

Events

OnLoadCaseRenamed

Fired when a LoadCase is renamed

OnLoadCaseChanged

Fired whenever a change is made to the LoadCase or its parts

These interfaces are obtained via QueryInterface.
