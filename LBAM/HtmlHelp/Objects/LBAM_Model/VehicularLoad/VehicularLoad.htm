VehicularLoad

A combination of a vehicle (truck) and a lane load

Properties

Applicability

Determines whether results are applicable for entire superstructure or only in contraflexure zones.

Axles

Collection of axles associated with this VehicularLoad

Configuration

Determines how truck and lane/sidewalk results are to be combined.

IMLane

Impact factor for lane load

IMTruck

Impact factor for truck load

LaneFactor

Arbitrary factor for lane results

LaneLoad

Value of uniform lane load

Name

Name of this VehicularLoad

SidewalkLoad

Value of uniform sidewalk load

TruckFactor

Arbitrary factor for truck results

UseNotional

Boolean to determine if notional truck rules are to be applied

VariableAxle

Index of variable axle

VariableMaxSpacing

Maximum spacing of variable axle, if present

Methods

Clone

Makes a deep clone of this VehicularLoad

Events

OnVehicularLoadChanged

Fired whenever a change is made to this VehicularLoad

Interfaces

These interfaces are obtained from the VehicularLoad via QueryInterface

IStructuredStorage2

Persistence of data
