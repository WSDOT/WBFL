BottomOffset Property (TemporarySupport Object)

Summary:

Offset of bottom of column for inclined columns

 

Syntax:

dblOffset = TemporarySupport.BottomOffset
TemporarySupport.BottomOffset = dblOffset

 

Element

Description

dblOffset

Offset of bottom of column. Positive values mean column bottom is offset up-station (to right).

TemporarySupport

The TemporarySupport object

Remarks:

Has no effect on zero-length TemporarySupports.

The absolute value of the offset must be less than the length of the TemporarySupport.
