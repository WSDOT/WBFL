TemporarySupport

A support that can be placed at any location along a Span. Otherwise, they are essentially the same as Support objects except that they are given a StageRemoved property which allows them to be used to model falsework, or any other types of supports that are used during construction and removed prior to service conditions.

Each TemporarySupport in an LBAM must have a unique integer ID. This ID is used to refer to the TemporarySupport when requesting structural responses at it or applying loads and POIs to it. Normal Support objects are referred to by their sequence in the Supports collection. Hence, TemporarySupport ID's must not lie in the range from zero to the number of Supports minus one.

TemporarySupports are made up of prismatic Segments that describe the cross section along their length. Hence, the TemporarySupport acts as a container for its Segments.

Properties

_EnumElements

Returns a fast enumerator (hidden,restricted)

BottomOffset

Offset of bottom of column for inclined columns

BoundaryCondition

Boundary condition at the bottom of the support

ID

Unique integer ID of the TemporarySupport

Length

The total length of the TemporarySupport

LoadModifier

Factor to be applied to all load combinations for this Support

Location

Location of the TemporarySupport measured from the start of the Span it is contained in.

Segment

Gets or puts a Segment to a particular Stage/location

SegmentLength

Gets the length of all Segments for a specified Stage

SegmentCount

Gets the number of Segments for a specified Stage

StageRemoved

Name of the Stage in which this TemporarySupport is removed from the LBAM

TopRelease

If true, a hinge is placed at the top of the column (fixed-length members only)

IsSymmetrical

Flag that forces the section properties of this TemporarySupport to be symmetrical about its length

Methods

AddSegment

Appends a Segment to the end

InsertSegment

Inserts a Segment at a specified location

MoveSegmentTo

Moves a Segment from one location to another

CopySegmentTo

Copies a Segment from one location to another

RemoveSegment

Removes a specific Segment

ReverseSegments

Reverses the order of all Segments

GetSegmentForMemberLocation

Gets a Segment at a specified distance along the TemporarySupport

GetSegmentsForStage

Gets a collection of all Segments, as they are input, corresponding to a specified Stage

GetMemberSegments

Gets a collection of Segments for a specified Stage. The contents of this collection is computed based on the Length of the TemporarySupport and the state of the IsSymmetrical flag.

Clone

Clones entire TemporarySupport and its Segments. Makes deep copy.

RemoveStage

Removes all data related to a specified stage for this TemporarySupport.

Events

OnTemporarySupportChanged

Fired whenever a change is made to the TemporarySupport
