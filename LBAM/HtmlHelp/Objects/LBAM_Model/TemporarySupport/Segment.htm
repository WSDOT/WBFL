Segment Property (TemporarySupport Object)

Summary:

Gets/Sets a Segment at a given position

 

Syntax:

objSegment = TemporarySupport.Segment(strStage, intRelPosition)
TemporarySupport.Segment(strStage, intRelPosition) = objSegment

 

Element

Description

strStage

Stage where Segment occurs

intRelPosition

Position (index) in collection where Segment is located.

objSegment

The Segment

TemporarySupport

The TemporarySupport object

Remarks:

Returns E_INVALIDARG or LBAM_E_NO_SEGMENTS_FOR_STAGE if Segment does not exist in collection.
