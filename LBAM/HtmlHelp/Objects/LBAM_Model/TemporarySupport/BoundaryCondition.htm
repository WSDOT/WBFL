BoundaryCondition Property (TemporarySupport Object)

Summary:

Specifies the connectivity of the bottom of the support to the superstructure, or the support condition for an idealized (zero-length) support

 

Syntax:

varBoundaryCondition = Support.BoundaryCondition
Support.BoundaryCondition = varBoundaryCondition

 

Element

Description

varBoundaryCondition

The Boundary Condition at the bottom of the column

Support

The TemporarySupport object

Remarks:

The bcRoller option will cause instability for fixed length supports if the top of the support is pinned.

For zero-length supports, the bcFixed option will effectively lock the span ends framing into the support.

Supports are bcPinned by default.
