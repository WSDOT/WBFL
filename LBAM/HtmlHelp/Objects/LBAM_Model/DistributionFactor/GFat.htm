GetGFat / SetGFat Methods (DistributionFactor Object)

Summary:

Distribution factor for fatigue loads - used to calculate stresses

 

Syntax:

DistributionFactor.GetGFat( dblFatM, dblFatV)

DistributionFactor.SetGFat( dblFatM, dblFatV)

 

Element

Description

dblGFatM

The Distribution factor for fatigue moment - used to calculate stresses

dblGFatV

The Distribution factor for fatigue shear force

DistributionFactor

The DistributionFactor object
