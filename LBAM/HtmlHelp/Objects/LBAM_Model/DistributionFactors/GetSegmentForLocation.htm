GetSegmentForLocation Method (DistributionFactors Object)

Summary:

Returns DistributionFactorSegment(s) based on a location measured from the start of the superstructure

 

Syntax:

DistributionFactors.GetSegmentForLocation (dblLocation, objLeftSegment, objRightSegment)

 

Element

Description

dblLocation

Distance, measured from start of the superstructure. The start of the superstructure is the left end of the left overhang, or if there is no overhang, the left support.

Positive values indicate placement in base units. Negative values indicate fractional spacing and may range from -1.0 to 0.0.

Note that this value will be 0.0 if objRightSegmentItem is not NULL

objLeftSegmentItem

If dblLocation is a a junction between two segments, this is the DistributionFactorSegment on the left side. If not at a junction between two segments, this is the DistributionFactorSegment at the location.

objRightSegmentItem

DistributionFactorSegment containing Segment at right of dblLocation. NOTE:This is only non-NULL if the location is at a junction between two segments.

DistributionFactors

This collection

Remarks:

Segment locations are as computed by model, and not necessarily exactly as input. Segment layout rules are used to compute the locations of the segments based on input data and symmetry.

objRightSegmentItem will be NULL if the location is not at the junction between two segments.
