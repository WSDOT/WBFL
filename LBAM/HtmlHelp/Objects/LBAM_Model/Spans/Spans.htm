Spans

This collection interface is used to access the Spans in a Model. This collection behaves like a sequential list of geometric segments ordered from left to right.

  • Supports must be provided at the ends of each span (i.e., the number of Supports must be equal to the number of Spans, plus one).

  • The collection supports the standard IDispatch-based collection access interface as well as access to a type-safe, fast vtable interface through a fast enumerator.

  • Access to the list of Spans is zero-based.

  • Spans can be inserted, moved and removed from the sequence.

Properties

Count

Number of Spans in collection

Item

Returns a Span from the collection

_EnumElements

Returns a fast enumerator (hidden,restricted)

Length

Computes the total length of all Spans in the LBAM

Methods

Add

Adds (appends) a Span to the end of the collection

Insert

Inserts a Span to a specified location in the collection

MoveTo

Move a Span from one location to another

CopyTo

Copy a given Span to a new location

Remove

Remove a Span

Reverse

Reverses the order of the Spans in a collection

Events

OnSpansChanged

Fired whenever a Span in the collection is changed

OnSpansAdded

Fired whenever a Span is added to the collection

OnSpansBeforeRemove

Fired just before a Span is removed from the collection

OnSpansMoveTo

Fired whenever MoveTo is called

OnSpansCopyTo

Fired whenever CopyTo is called

OnSpansReverse

Fired whenever Reverse is called

Interfaces

These interfaces are obtained via QueryInterface

IStructuredStorage2

Persistence of LBAM data
