GetLiveLoadModel Method (LoadCombination Object)

Summary:

Returns one of the types of live load models associated with this combination

Syntax:

llmt = LoadCombination.GetLiveLoadModel(index)
 

Element

Description

llmt

One of the live load model types associated with this combination

index

Index of the live load model to return

LoadCombination

The LoadCombination object
