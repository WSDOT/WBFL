Insert Method (VehicularLoads Object)

Summary:

Insert a VehicularLoad into the collection at the desired location

 

Syntax:

VehicularLoads.Insert (intIndex, objVL)

 

Element

Description

intIndex

Location in collection where VehicularLoad is to be inserted

objVL

The VehicularLoad to be inserted

VehicularLoads

The VehicularLoads object

Remarks:

Sequence is zero-based

Other VehicularLoad(s) are shifted to the right

Returns E_INVALIDARG if index is out of bounds (cannot have gaps)
