Remove Method (VehicularLoads Object)

Summary:

Removes a VehicularLoad from the sequence

 

Syntax:

VehicularLoads.Remove(intIndex)

 

Element

Description

intIndex

Location of VehicularLoad within collection

VehicularLoads

The VehicularLoads object

Remarks:

Returns E_INVALIDARG if an index is specified that is out of the bounds of the collection.

Other VehicularLoads are shifted to the left
