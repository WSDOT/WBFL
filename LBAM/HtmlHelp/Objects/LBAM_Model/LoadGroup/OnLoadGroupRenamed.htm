OnLoadGroupRenamed Event (LoadGroup Object)

Summary:

Fired when a LoadGroup is renamed

 

Syntax:

OnLoadGroupRenamed(strOldName, strNewName)

 

Element

Description

strOldName

Name before the change

strNewName

Name after the change

Remarks:

All LoadGroups in an LBAM must have unique names.

No error checking is performed during this call to see if you have created a LoadGroup with a duplicate name.
