C++ Specific (Fast) Enumerator Interfaces

All of the collections in the LBAM engine support access to a special hidden, restricted enumerator that provides high-performance access to collection elements. The enumerator is obtained through a call to the _EnumElements method on the collection.

Each enumerator has a type that is specific to the type of element that is contained in the collection. Enumerators are given the name IEnumThing, where Thing is the type of element in the collection (e.g. IEnumJoints, IEnumPointLoads, ...).

A prototype of the enumerator interface for a hypothetical collection of Things is given as follows:
