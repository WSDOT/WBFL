OnAxlesCopyTo Event (Axles Object)

Summary:

Fired when CopyTo is called

 

Syntax:

OnAxlesCopyTo(int FromIndex, intToIndex)

 

Element

Description

intFromIndex

Location where Axle is being moved from

intToIndex

Location where Axle is being moved to

Axles

The Axles object
