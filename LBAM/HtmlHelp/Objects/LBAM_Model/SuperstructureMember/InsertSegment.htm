InsertSegment Method (SuperstructureMember Object)

Summary:

Inserts a Segment into the collection at the given Stage/Location

 

Syntax:

SuperstructureMember.InsertSegment([in] strStage, [in] intRelPosition, [in] objSegment)

 

Element

Description

strStage

Stage where segment is to be applied

intRelPosition

Relative position (index) where Segment is to be placed in collection

objSegment

The Segment to be inserted

SuperstructureMember

The SuperstructureMember object

Remarks:

Returns E_INVALIDARG if intRelPosition is out of bounds

Existing Segments in collection are shifted to right to make room for new Segment
