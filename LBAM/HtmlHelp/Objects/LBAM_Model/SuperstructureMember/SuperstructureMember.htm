SuperstructureMember

A individual member (girder) that describes the properties of the main structural beams along the superstructure.

SuperstructureMembers are made up of prismatic Segments that describe the cross section along their length. Hence, the SuperstructureMember acts as a container for its Segments.

Properties

_EnumElements

Returns a fast enumerator (hidden,restricted)

Length

The total length of the SuperstructureMember

Segment

Gets or puts a Segment to a particular Stage/location

SegmentLength

Gets the length of all Segments for a specified Stage

SegmentCount

Gets the number of Segments for a specified Stage

IsSymmetrical

Flag that forces the section properties of this SuperstructureMember to be symmetrical about its length

Methods

AddSegment

Appends a Segment to the end

InsertSegment

Inserts a Segment at a specified location

MoveSegmentTo

Moves a Segment from one location to another

CopySegmentTo

Copies a Segment from one location to another

RemoveSegment

Removes a specific Segment

ReverseSegments

Reverses the order of all Segments

GetSegmentForMemberLocation

Gets a Segment at a specified distance along the SuperstructureMember

GetSegmentsForStage

Gets a collection of all Segments, as they are input, corresponding to a specified Stage

GetMemberSegments

Gets a collection of Segments for a specified Stage. The contents of this collection is computed based on the Length of the SuperstructureMember and the state of the IsSymmetrical flag.

Clone

Clones entire SuperstructureMember and its Segments. Makes deep copy.

GetEndRelease

Gets the end release (hinge) properties for this SuperstructureMember

SetEndRelease

Sets the end release (hinge) properties for this SuperstructureMember

RemoveStage

Removes all data related to a specified stage for this SuperstructureMember.

Events

OnSuperstructureMemberChanged

Fired whenever a change is made to the SuperstructureMember

Interfaces

These interfaces are obtained via QueryInterface

IStructuredStorage2

Persistence of LBAM data
