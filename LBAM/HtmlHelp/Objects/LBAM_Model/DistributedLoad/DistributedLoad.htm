DistributedLoad

Represents a trapezoidally distributed load that can be placed along a member in an LBAM

Properties

Direction

Direction load is to be applied in

EndLocation

Location of end of DistributedLoad along member

MemberID

ID of member this DistributedLoad is attached to

MemberType

Type of member DistributedLoad is attached to

Orientation

Orientation load is to be applied at

StartLocation

Location of start of DistributedLoad along member

WEnd

Load value at start

WStart

Load value at end

Methods

Clone

Creates a clone of this DistributedLoad

Events

OnDistributedLoadChanged

Fired whenever a change is made to the DistributedLoad or its parts

Interfaces

These interfaces are obtained via QueryInterface.

IStructuredStorage2

Persistence
