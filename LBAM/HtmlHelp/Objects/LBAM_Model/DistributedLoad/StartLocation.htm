StartLocation Property (DistributedLoad Object)

Summary:

Location of the start of the DistributedLoad with respect to the start of the member it is attached to

 

Syntax:

dblLocation = DistributedLoad.StartLocation
DistributedLoad.StartLocation = dblLocation

 

Element

Description

dblLocation

Distance, measured from start of member, along member where start of DistributedLoad is to be found. Positive values indicate placement in base units. Negative values indicate fractional spacing and may range from -1.0 to 0.0.

DistributedLoad

The DistributedLoad object

Remarks:

DistributedLoads must be located within the bounds of the member to which they are attached. If a DistributedLoad is located off of the member, it will result in an error at analysis time.

DistributedLoads cannot be attached to a zero-length supports.
