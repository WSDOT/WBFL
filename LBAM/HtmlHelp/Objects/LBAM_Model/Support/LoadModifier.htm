LoadModifier Property (Support Object)

Summary:

Factor to be applied to all load combinations for this Support. For LRFD, it is equal to the product of the h factors as specified in article 1.3.2.

 

Syntax:

dblLoadModifier = Support.LoadModifier( varLoadModifierType )
Support.LoadModifier( varLoadModifierType ) = dblLoadModifier

 

Element

Description

dblLoadModifier

The LoadModifier

varLoadModifierType

The type of load modifier to be assigned

Support

The Support object

Remarks:

Equal to 1.0 by default
