GetSegmentForMemberLocation Method (Support Object)

Summary:

Returns Segment(s) based on a location measured from the start of a member

 

Syntax:

Support.GetSegmentForMemberLocation (strStage, dblLocation, objLeftSegmentItem, objRightSegmentItem)

 

Element

Description

strStage

Name of Stage where you want to find Segment.

dblLocation

Distance, measured from start of member, along member where Segment is to be found. Positive values indicate placement in base units. Negative values indicate fractional spacing and may range from -1.0 to 0.0.

objLeftSegmentItem

SegmentItem containing Segment at the location (if not at a junction between two segments), or at left side of the location that you are looking for.

objRightSegmentItem

SegmentItem containing Segment at right of dblLocation. NOTE:This is only non-NULL if the location is at a junction between two segments.

Support

This Support

Remarks:

Segment locations are as computed by model, and not necessarily exactly as input. Segment layout rules are used to compute the locations of the segments based on input data and symmetry.

objLeftSegmentItem will be NULL if no Segment exists at the location/Stage. This typically means that no Segments exist in this Stage.

objRightSegmentItem will be NULL if the location is not at the junction between two segments.
