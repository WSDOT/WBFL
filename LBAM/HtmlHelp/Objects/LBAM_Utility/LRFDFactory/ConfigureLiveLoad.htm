ConfigureLiveLoad Method (LRFDFactory Object)

Summary:

Configure the live loads in an LBAMModel so they conform to the AASHTO LRFD Specifications

Syntax:

factory.ConfigureLiveLoad ( objLBAMModel, boolIncludeLowBoy, intSpecificationUnitsType, objUnitServer)

Element

Description

objLBAMModel

The LBAMModel that the live loads are to be configured for

boolIncludeLowBoy

If true, include the LRFD "Low Boy" truck in the VehicularLoads for the Design truck.

intSpecificationUnitsType

Version of the specification that live load parameters are to be taken from

objUnitServer

A UnitServer object that has base units which are consistent with the units used to build the LBAMModel

factory

The LRFDFactory object

Remarks:

Refer to LRFD 3.6.1.1

The DesignTruck is configured with: The Design truck and Lane, the Design Tandem and Lane, the Dual Truck Train and Lane, and optionally, the "Low Boy"

The LRFD Fatigue truck is also configured

All pre-existing live loads are deleted
