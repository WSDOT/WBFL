GeneratePOIsOnSuperstructure Method (LRFDFactory Object)

Summary:

Generate POI's across superstructure at a fixed increment of each span length

Syntax:

intLastID = factory.GeneratePOIsOnSuperstructure([in]objLBAMModel, [in]intStartID, [in]intIncrement)

Element

Description

intLastID

The ID of the last POI that was generated. This can be used along with intStartID to determine the number of POI's that were generated

objLBAMModel

The LBAMModel on which POI's are to be generated

intStartID

The ID of the first POI to be generated. ID's are incremented by one for Subsequent POI's created

intIncrement

The POI spacing increment for all spans on the superstructure (e.g. if equal to 10, there will be 11 POI's generated on each span at equal spacing).

factory

The LRFDFactory object

Remarks:

If an overhang exists, this method places a POI at each end of the overhang
