ConfigureFatigueLiveLoad Method (LRFDFactory Object)

Summary:

Configure the fatigue live load in an LBAMModel so they conform to the AASHTO LRFD Specifications

Syntax:

factory.ConfigureFatigueLiveLoad ( objLBAMModel, dblImpactTruck,dblImpactLane, intSpecificationUnitsType, objUnitServer)

Element

Description

objLBAMModel

The LBAMModel that the live loads are to be configured for

dblImpactTruck

Impact factor for trucks

dblImpactLane

Impact factor for lane loads

intSpecificationUnitsType

Version of the specification that live load parameters are to be taken from

objUnitServer

A UnitServer object that has base units which are consistent with the units used to build the LBAMModel

factory

The LRFDFactory object

Remarks:

Refer to LRFD 3.6.1.1

Impact factors are generally greater than 1.0. For example, for 15% impact the factor would be 1.15

All pre-existing live loads are deleted
