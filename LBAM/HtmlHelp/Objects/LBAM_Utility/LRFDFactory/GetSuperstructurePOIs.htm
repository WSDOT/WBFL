GetSuperstructurePOIs Method (LRFDFactory Object)

Summary:

Returns lists of the ID's and locations of all POI's along the superstructure sorted by location.

Syntax:

factory.GetSuperstructurePOIs ([in]objLBAMModel, [out]varPoiIDs, [out]varPoiLocations)

Element

Description

objLBAMModel

The LBAMModel for which POI's are to be sorted

varPoiIDs

ILngArray array of longs containing POI ID's as sorted by location along superstructure

varPoiLocations

IDblArray array of doubles containing the locations of each POI in global coordinates

factory

The LRFDFactory object

Remarks:

Both user-defined and internally-generated POI's are included. Internally-generated POI's have negative ID's.
