ConfigureLoadCombinations Method (LRFDFactory Object)

Summary:

Configure the LBAMModel's LoadCombinations collection with the LRFD load combination table

Syntax:

factoryl.ConfigureLoadCombinations objModel

Element

Description

objModel

The LBAMModel whose LoadCombinations are to be configured

factory

The LRFDFactory object

Remarks:

Refer to LRFD Section 3.4.1 for details

The complete LRFD load combinations table is populated.

Any pre-existing LoadCombinations are deleted

All load case and limit state names are designated with capital letters (e.g. DC, DW, LLIM, STRENGTH-I, SERVICE-III). The LBAM is a case-sensitive system.

Before load case and load combination results are requested, you must define loads and load groups. You must map your load groups into the LRFD load cases (DC, DW, etc)
