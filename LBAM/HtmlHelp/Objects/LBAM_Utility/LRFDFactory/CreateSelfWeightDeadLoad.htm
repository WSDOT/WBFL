CreateSelfWeightDeadLoad Method (LRFDFactory Object)

Summary:

Generates self-weight dead loads

Syntax:

LBAMGeneralUtil.CreateSelfWeightDeadLoad ( objLBAMModel, stage,lgName,unitWgt,E)

Element

Description

objLBAMModel

The LBAMModel that the live loads are to be configured for

stage

Name of the stage when the dead load is applied

lgName

Name of load group to add the loads to

unitWgt

Unit weight of the structural material

E

Modulus of elasticity of the structural material (see Remarks for more information)

factory

The LRFDFactory object

Remarks:

This utility function is only applicable for models consisting of members whose material is uniform throughout. That is, all cross sections have the same unit weight and modulus of elasticity.

Modulus of elasticity is a required input because section properties are defined by stiffness (EA and EI). In order to compute the weight of a member per unit length the modulus of elasticity has to be divided out of the axial stiffness. w = (unit weight)*(EA)/E.
