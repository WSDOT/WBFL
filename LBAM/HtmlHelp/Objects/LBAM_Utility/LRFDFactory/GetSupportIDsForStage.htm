GetSupportIDsForStage Method (LRFDFactory Object)

Summary:

Get an array of the ID's of all supports in the LBAMModel

Syntax:

objSupportIDs = factory.GetSupportIDsForStage ( objLBAMModel, bstrStage)

Element

Description

objLBAMModel

LBAMModel being queried

bstrStage

Name of Stage in question

varSupportIDs

ILngArray of longs containing the id's of all supports in the LBAMModel

factory

The LRFDFactory

Remarks:

Returns indexes of permanent Support's as well as TemporarySupport's that exist in the Stage
