CreateSimpleModel Method (LRFDFactory Object)

Summary:

Create an entire LBAM Model in a single call

Syntax:

objLBAM = factory.CreateSimpleModel( varSpanLengths, dblE, dblA, dblI, dblDepth )

Element

Description

objLBAM

The LBAMModel that we create

varSpanLengths

IDblArray array of doubles that define the lengths of the Span's

dblE

Elastic modulus of prismatic SuperstructureMember

dblA

Cross sectional area of prismatic SuperstructureMember

dblI

Moment of inertia of prismatic SuperstructureMember

dblDepth

Depth of prismatic SuperstructureMember

factor

The LRFDFactory object

Remarks:

Model has a single Stage named "Stage 1"

Model has a single SuperstructureMember

Model has zero-length supports. Support 0 is pinned and the rest are on rollers

Model has no overhangs
