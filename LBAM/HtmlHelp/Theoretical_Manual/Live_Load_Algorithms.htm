Live Load Algorithms

The current truck placement algorithm used by the LBAM Live Loader is very simple and is implemented by the BruteForceVehicularResponse object. This algorithm simply places every axle of the truck on every POI (internal and user-defined) in the model, with the truck travelling in both directions.

If this algorithm is not adequate for your needs, it is possible to replace it by creating a new live loader strategy object that implements the IEnvelopedVehicularResponse and the other interfaces as BruteForceVehicularResponse. This new strategy can then be used by theLiveLoadModelResponse object by initializing it via the IEnvelopingStrategy interface.

Internal POI generation is controlled by the IAnalysisPOIs interface of the LoadGroupForceResponse and LoadGroupDeflectionResponse objects.
