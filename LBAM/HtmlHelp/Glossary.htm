Glossary

We understand that computer people often seem to be speaking in their own language. Too bad we don't get paid by the acronym.

API

Application Programming Interface.

COM

Component Object Model. This is Microsoft's software architecture that defines how to create software components and how they interact. All of the public interfaces for the LBAM system are based on the COM architecture.

FEM2D

WBFL - 2D beam analysis engine

LBAM

Longitudinal Brige Analysis Model - Our version of a 2D lollipop stick model of a bridge superstructure

LBAM system

All of the components and documentation that make up the software system used to describe and analysis LBAM's

LRFD

AASHTO LRFD Specifications

VB or VBA

Microsoft Visual Basic or Visual Basic for Applications

WBFL

Washington Bridge Foundation Libraries - The LBAM and FEM2D are parts of this

WSDOT

Washington State Department Of Transportation. The LBAM is brought to you by the WSDOT Bridges and Structures Office.
