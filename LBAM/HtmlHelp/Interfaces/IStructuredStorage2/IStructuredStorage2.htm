IStructuredStorage2

An Interface for persisting model data. This Interface is supported extensively by objects in the LBAM system. In fact, it is easy to save entire LBAMModel to a file if desired.

Methods

Load

Load model data from an object that inplements IStructuredLoad2

Save

Save model data to an object that inplements IStructuredSave2

 

Remarks

IStructuredLoad2 and IStructuredSave2 are defined in the WBFLTools library. Default objects implementing simple file-based storage are also contained in the WBFL. Refer to the WBFLTools documentation for more information.
