ILngArray

Zero-based, one-dimensional array of longs.

Properties

Count

The number of items in the array

Item

Set/Get a particular item in the array

Methods

Add

Load the object from persistence

Clear

Clear all items from the collection

Clone

Make a clone of this array

Find

Find the first item in the array that matches a given value

Insert

Insert an item into the array

ReDim

Redimension the array

Remove

Remove an item from the array

Reserve

Save the object to persistence
