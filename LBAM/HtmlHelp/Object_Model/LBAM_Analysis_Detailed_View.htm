LBAM Analysis - Detailed View

The object models in this section present a more detailed view of the LBAM Analysis engines as compared with the previous section. It is expected that most users of the LBAM system will use the LBAMAnalysisEngine object which provides a simplified way to initialize and use the analysis components. Hence, this section can safely be ingored by most users. However, if you want to make modifications to the analysis system, the object models in this section will prove to be valuable.
