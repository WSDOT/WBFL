Object Model To Describe Combined Loading

This portion of the model shows objects that define load combinations. Each member load in the model is assigned to a LoadGroup. Each LoadCase may then reference one or more LoadGroups. The results for a LoadCase is simply the sum of the results of its LoadGroups. Each LoadCombination references one or more LoadCases along with a factor that the LoadCase results are multiplied by.
