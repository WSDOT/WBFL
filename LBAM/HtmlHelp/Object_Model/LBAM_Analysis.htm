LBAM Analysis Engine

The LBAMAnalysisEngine object provides a simplified way of creating and accessing all of the default analysis engines provided with the LBAM system. It does so by aggegating all of the default engines into one place and providing a single initialization method for its use.

Most users of the LBAM system will find that this object provides all of the analysis capabilities that they need. However, the trade-off is that it conceals some of the advanced capabilities of the system, such as the ability to add your own custom live load algorithm (aka, strategy). If you need access to these advanced features, consult the Programmer's Guide section of this book and the LBAM source code.
