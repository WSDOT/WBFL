LBAM Live Load Engine

This model shows the primary objects in the LBAM live loader and their dependencies.

Note that the VehicularAnalysisContext provides only the interfaces that the live load system requires from the LBAMModel and the LBAMAnalysis objects. The purpose for this design is to decouple the live loader from the rest of the system in order to make it possible for the live loader to be used for systems other than the LBAMModel.
