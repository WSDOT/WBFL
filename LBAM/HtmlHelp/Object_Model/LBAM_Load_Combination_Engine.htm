LBAM Load Combination Engine

This model shows the primary objects in the LBAM load combination package and their dependencies.

Note that the LoadCombinationAnalysisContext provides only the interfaces that the system requires from the LBAMModel, analysis, and live load packages. The purpose for this design is to decouple the load combiner from the rest of the LBAM system. This makes it possible for it to be reused in other systems. Refer to the LoadCombinationTest folder in the main source distribution for an example of the load combiner being used with an alternative context object.
