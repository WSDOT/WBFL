LBAM Analysis

This object model provides an overview of the main objects in the LBAM Analysis package. Note that nearly all of the reponse computation interfaces are derived from a single abstract (non-creatable) implementation class called CLoadGroupResponse.

Note that there are two concrete implementations. Clients may chose which implemetation to use depending on whether reponses based on force properties or deflection properties of the LBAMModel are desired.
