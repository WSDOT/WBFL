Object Model To Describe Live Loading

The LiveLoad object model provides a flexable means to define vehicular live loads. It provides the capabilities to define any of the live loads (variable or fixed axle trucks combined with lane loads) described in the AASHTO LFD and LRFD Specifications. User-defined permit and special live loads may also be defined. The LiveLoadModel's may then be referenced and factored by LoadCombinations.
