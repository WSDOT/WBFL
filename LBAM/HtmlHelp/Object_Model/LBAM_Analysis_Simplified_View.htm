LBAM Analysis - Simplified View

This section contains a simplified (less detailed) version of the object models for the LBAM Analysis Engines. This portion of the documentation is meant for programmers who are going to use, but do not need to modify the analysis engines.

If you plan to modify the analysis engines, or just want to know more about the entire system, you will also need to refer to the Detailed object models section.
