Bibliography

Here is a list of books that we have found useful during the development of the LBAM. Much of the information is also available online at Microsoft's MSDN web site and other places around the Internet.

COM

  • Inside OLE, Kraig Brockscmidt, Microsoft Press, 1999. A very comprehensive book on OLE (COM). Pretty dry, but complete.

  • Inside ATL, George Shepherd and Brad King, Microsoft Press, 1999. The best book that we know of on the Active Template Library (ATL). ATL is used extensively in the LBAM system. ATL is only applicable for C++ users.

Visual Basic

  • Visual Basic 5 From the Ground Up, Gary Cornell, Osbourne Press, 1997. Probably not the best book on Visual Basic, but adequate. A Visual Basic 6 version is now available.

  • VBA for Microsoft Office 2000 Unleashed, Paul McFedries, SAMS, 1999 Good intro and comprehensive discussion on using Visual Basic For Applications in Excel and other Office applications.

Engineering

  • AASHTO LRFD Bridge Design Specificatons 2nd Edition, AASHTO, 2002. The book that made it all possible

  • AASHTO Standard Specificatons For Highway Bridges 17th Edition, AASHTO, 2002.
