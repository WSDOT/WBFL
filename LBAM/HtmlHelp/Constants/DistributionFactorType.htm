DistributionFactorType

Designates type of Distribution factor to be used in applying this LiveLoadModel. Use in concert with the DistributionFactor object

Constant

Value

Description

dftNone

0

No distibution factor is to be applied

dftSingleLane

1

Use the single-lane distribution factor

dftMultipleLane

2

Use the multiple-lane distribution factor

dftEnvelope

3

Use the maximum of single-lane and multiple-lane factors

dftFatigue

4

Use the fatigue distribution factor.
