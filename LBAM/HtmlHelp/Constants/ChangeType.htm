ChangeType

Used as a parameter from an event. Designates what type of change took place during the event.

Note that not every type of object can fire any of these change types. For example, a PointLoad will never change the model stiffness and fire a cgtStiffness change. You will need to study at the object model and use common sense to predict what types of events an object will fire.

Constant

Value

Description

cgtStress

0

A stress factor was changed

cgtStiffness

1

Something changed to affect the overall stiffness of the structure

cgtCombination

2

Something about a load combination changed (factor, table)

cgtPOI

3

A POI was moved (won't fire if DF changes - see next)

cgtDistributionFactor

4

A distribution factor changed

cgtStage

5

A Stage name changed

cgtStageOrder

6

The Stage ordering changed or a Stage was added/deleted

cgtLoads

7

A load, loads collection, or load group changed

cgtDescription

8

A textual description of something was changed

cgtLiveLoad

9

A live load changed

cgtUserItemData

10

User-defined ItemData changed

cgtClear

11

Entire model was wiped

cgtUnknown

12

A bunch of stuff changed
