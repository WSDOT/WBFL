Examples

This section documents some sample programs that you can use to help you learn more about the LBAM system. In fact, the samples are the best way to learn - dig in!

The following Samples are presented:

Name

Description

Type

Location within WBFL folder

Simple VBA Sample

An excel spreadsheet using VBA that builds a simple superstructure model and gets results from an analysis.

Excel/VBA

Examples/Excel/SimpleSample.xls

Live Loader VBA Example

Excel spreadsheet that implements a very flexible live load analysis

Excel/VBA

Examples/Excel/LiveLoader.xls

SDCL

Excel spreadsheet that computes multi-stage LRFD load combination results

Excel/VBA

Examples/Excel/SDCL.xls

Culvert

Excel spreadsheet that demonstrates the analysis of a multi-cell culvert with haunched members

Excel/VBA

Examples/Excel/Culvert.xls

LBAMDumper

A C++ DLL example that dumps all input and results into a text file and a utility program driver for the DLL (LBAMDump)

C++

Examples/Vc/LBAMDumper/LBAMDumper.dsw

Note that you must enable macros in Excel spreadsheets in order for the samples to work.
