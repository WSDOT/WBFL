LBAMDumper

The LBAM Dumper example consists of (1) a DLL that takes an LBAM and dumps its description and analysis results to a text file, and (2) a console application that drives the DLL.

The DLL project is called LBAM Dumper and the console application is called LBAM Dump. Both projects can be found in the Examples folder.

To run LBAMDump, enter the following at a command prompt

LBAMDump input_file output_file <output_details>

where

input_file is the name of the input file. The input file must be a file created from the LBAM itself

output_file is the name of the file where basic output is stored

output_details is an option file where output details are stored. This file can be quite large.
