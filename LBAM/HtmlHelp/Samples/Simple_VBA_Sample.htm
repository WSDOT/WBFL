Simple VBA Sample

This sample is an shown in the SimpleSample.xls Excel spreadsheet. It provides a very simple example of how to use the LBAM in Visual Basic for Applications (VBA). The details of the sample are explained more fully in the Concepts section of this help document. Refer to A Simple Example for more details.

Tip!
From the Excel menu, select the "Tools | Macro | Visual Basic Editor" (or on the Toolbar) to display the VBA editing environment.
