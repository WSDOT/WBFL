Describing TemporarySupports

TemporarySupport objects allow you to model falsework used in the construction stages of a bridge superstructure. Each Span in an LBAM model can have zero, one or many TemporarySupports placed along its length. Every TemporarySupport object has a StageRemoved property which specifies the name of the Stage when it is to be removed from the model.
