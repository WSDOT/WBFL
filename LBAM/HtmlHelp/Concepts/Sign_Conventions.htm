Sign Conventions

Units

There is no specific unit system, or utilities for using units in the LBAM. The only restriction is that you must use a consistent unit system for all values in the LBAM.

With the above said, it is inevitable that you will need to manage unit systems with your application that uses the LBAM. If so, you should consider using the UnitServer object in the UnitServer package in the WBFL. It provides a good solution for managing units across multiple components. Refer the the UnitServer help documentation for more details.

Model (Global) Coordinates

The structural elements (Spans, Supports, Segments) that make up the superstructure are always described from right to left. Likewise, support elements (Segments) are described from bottom to top.

Model coordinates are described in a right-hand coordinate system as shown in the figure below. Note that the datum point is always at left end of the first Span member (aka top of Support 0). This means that locations on the left overhang (if any) have negative X values.

Member Coordinates

The X direction for all members is colinear with the member going from the start toward the end of the member. Spans and SuperStructureMembers run from left-to-right. Supports and TemporarySupports run from bottom-to-top. Positive moment lies along a vector normal to the page pointing toward you (same as global Z). The Y direction can be determined from the right-hand rule based on the X and Z directions.

POI Forces

POI forces are reported at both sides of the location where the POI cuts the member. The forces returned are the externally applied forces required to hold the member in equilibrium. POI forces can be returned in local member coordinates or global coordinates.

 

Note that this convention creates a slight discrepancy when a POI is located at the end of the member: one of the faces does not exist. For instance, the left face does not exist on the starting face of the member. For consistency, the forces returned in these cases will be equal and opposite to the true face at the end of the member.

Reaction Forces

Reaction forces are returned in global coordinates. The forces returned are the forces applied to the model at the support point.

Deflections

Deflections are returned in global coordinates.

Stresses

Positive stresses indicate tension, negative stresses indicate compression.

Strains

Strain loads can be applied to members in the LBAM. Positive axial strains make the member stretch longer. Positive curvature strains make the beam smile.
