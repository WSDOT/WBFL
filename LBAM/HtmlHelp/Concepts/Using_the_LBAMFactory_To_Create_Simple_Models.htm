Using the LRFDFactory To Create Simple Models

The CreateSimpleModel method on the LRFDFactory object can be used to quickly create a single-stage, prismatic, multi-span model with a single call. Models created by CreateSimpleModel can be used as a launching point for creating more complex models. An example of using the CreateSimpleModel call is in the Simple Example chapter in this section of this book.

Note that while CreateSimpleModel provides a way to quickly create a complete LBAM model, it does not provide much flexibility in building the model. Hence, in some situations (like when you have a model with many SuperstructureMembers) it will likely be more work to use CreateSimpleModel than it would be just to build the model from scratch (using API calls). CreateSimpleModel is a handy feature, just don't let it turn into a crutch.
