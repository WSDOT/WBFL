UnitSystemDemo

This sample program demonstrates how to use the unit system, unit mode controller, and display units manager objects. It also demonstrates how to implement the ISupportUnitServer interface in Visual Basic. The source code for this example can be found in x:\ARP\AutoWBFL\Examples\VB\UnitSystem.

This program computes the force in the steel and concrete portions of an axially loaded structural tube. The properties of the concrete are implemented in a class that wants to work in SI units internally. The properties of the steel tube are implemented in a class that wants to work in US units internally. Both the Concrete and SteelTube classes implement the ISupportUnitServer interface so that that main form of the application can work with the Concrete and SteelTube classes without having to worry about unit conversions.

The main form and the input form require input in specific units. The application uses the UnitModeController object to manage the unit mode and the DisplayUnitMgr object to handle the display units.

In the frmMain class, pay particular attention to the Form Load event and the UpdateDisplay methods.
