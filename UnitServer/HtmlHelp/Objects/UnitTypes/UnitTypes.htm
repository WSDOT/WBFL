UnitTypes

This object maintains a collection of UnitType objects.

Properties

Count

Returns the number of items in the collection

Item

Returns an item from the collection

Methods

Add

Adds a new item to the collection

Clear

Clears the collection

Remove

Removes an item from the collection

Events

OnUnitAdded

Fired when a Unit object is added to the collection

OnUnitRemoved

Fired when a Unit object is removed from the collection

OnUnitsCleared

Fired when the collection is cleared

OnUnitTypeAdded

Fired when a unit type object is created

OnUnitTypeRemoved

Fired when a unit type object is removed from the collection

OnUnitTypesCleared

Fired when the collection is cleared

Restricted Properties

_EnumUnitTypes

Returns a unit type enumerator
