UnitModeController

The unit model controller is used to implement a unit mode in an application. The unit mode controller manages the application's unit mode state and provides a pre-build unit mode editing interface.

Properties

UnitMode

Sets/Gets the unit mode

Methods

ShowSelectionDialog

Shows the unit mode selection dialog

ShowSelectionDialogEx

Shows the unit mode selection dialog that supports user defined help topics

Events

OnUnitModeChanged

Fired when the unit mode changes
