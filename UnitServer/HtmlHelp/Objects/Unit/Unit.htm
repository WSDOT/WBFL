Unit

A Unit object represents a specific physical unit of measure such as "feet" or "kips".

Properties

ConvFactor

Returns a conversion factor

PostTerm

Returns a postfix term

PreTerm

Returns a prefix term

Tag

Returns the unit tag

UnitServer

Returns the unit server

UnitSystem

Returns the unit system this unit belongs to

UnitType

Returns the physical unit type for this unit
