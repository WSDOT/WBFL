ConvFactor Property (Unit Object)

Summary:

Returns a conversion factor

Syntax:

dblCF = object.ConvFactor

Element

Description

dblCF

The conversion factor

object

This object

Remarks:

One Unit object in a UnitType collection will have a conversion factor of 1.0. This will be the primary unit for that type. The primary unit for build-in types are "kg", "m", and "sec". This conversion factor is used to convert this unit to the primary unit. For example, for the unit type "Length" the primary unit is "meter". If this Unit object represented "inch" the conversion factor would be multiplied to a value in inches to convert it to meters.
