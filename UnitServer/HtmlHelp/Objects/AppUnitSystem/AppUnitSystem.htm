AppUnitSystem

Application unit system designed to provide unit services to MDI applications.

Properties

UnitServer

Returns the application unit server

DisplayUnitMgr

Returns the display unit manager

Methods

ConvertFromBaseUnits

Converts a number from base units to a specified unit of measure

ConvertToBaseUnits

Converts a number from a specifed unit of measure to base units

Events

OnUnitServerChanged

Fired when ever the unit server changes

OnUpdateDisplay

Fired when ever the display units change
