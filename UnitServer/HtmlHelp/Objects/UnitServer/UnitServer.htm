UnitServer

The unit server object maintains the base units for an application or component. Base units are a consistent sent of units that an application or component uses for its internal computations.

The unit server object also provides unit converson serves to make it easy to convert numbers between units of measure and unit systems. Units can also be converted to and from base units without the caller knowing what the base units are.
