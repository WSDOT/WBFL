SetBaseUnits Method (UnitServer Object)

Summary:

Sets the application or component base units

Syntax:

object.SetBaseUnits(strMass,strLength,strTime,strTemp,strAngle)

Element

Description

strMass

The base unit of mass

strLength

The base unit for length

strTime

The base unit for time

strTemp

The base unit for temperature

strAngle

The base unit for angle

object

This object

Remarks:

This method can only be called once for each instance of the UnitServer object. It is recommended that you call it during the initialization routines of your application or component. Subsequent calls to this method will fail (IDS_E_BASEUNITSSET)
