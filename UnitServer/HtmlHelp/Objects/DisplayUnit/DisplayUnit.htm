DisplayUnit

A DisplayUnit brings together a unit of measure and formatting information to create a formatted display unit string. DisplayUnit objects are members of a DisplayUnits collection.

Properties

Formatter

Sets/Gets the display unit formatter object

Unit

Returns the Unit object

UnitSystem

Returns the system of units

UnitTag

Returns the unit tag

Methods

Format

Creates a formatted display unit string

IsDefault

Indicates if this display unit is the default in the collection

Events

OnFormatChanged

Fired when ever this display unit changes
