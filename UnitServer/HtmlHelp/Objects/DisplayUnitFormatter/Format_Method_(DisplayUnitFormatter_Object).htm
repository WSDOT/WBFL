Format Method (DisplayUnitFormatter Object)

Summary:

Formats a numerical value and a unit tag into a text string

Syntax:

strText = object.Format(dblVal,strTag)

Element

Description

strText

The formatted unit value as a text string

object

This object

dblVal

The converted value

strTag

The unit tag

Remarks:

The text string will be -INF or +INF if the converted value exceeds the floating point capabilities of the computer.
