AnnotatedDisplayUnitFormatter

Display unit formatter that formats a display unit string using special annotations such as feet and inchs or stationing.

Properties

Annotation

String that specifies alternate annotations for the formatted unit value

Justification

Returns the text justification

Multiplier

Multiplier used to convert the right hand part of the formatted unit value

Notation

Returns the numeric format type

OffsetDigits

Specifies the offset from the decimal point to the division between the left and right hand portions of the formatted unit value

Precision

Returns the decimal precision of the formatted unit value

UsesTag

Specifies whether the unit tag is used when formatting the unit value

Width

Returns the minimum width of the numeric portion of the formatted unit value

ZeroTolerance

Returns the largest value that will be taken as zero.

Methods

Format

Formats a numerical value and a unit tag into a text string

FormatSpecifiers

Sets the various parameters used to control the format of the text string

Events

OnFormatChanged

Fires whenever the format specifiers change
