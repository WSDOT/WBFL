DisplayUnitMgr2

The display unit manager object is responsible for managing the display unit groups. This DisplayUnitMgr2 object is typically used with Multiple Document Interface (SDI) applications and is associated with the AppUnitSystem object.

Properties

Count

Returns the number of DisplayUnitGroup objects in the collection

Item

Returns a DisplayUnitGroup object

Methods

Add

Add a new DisplayUnitGroup to the collection

Clear

Removes all DisplayUnitGroups from the collection

Remove

Removes a DisplayUnitGroup from the collection

Events

OnDisplayUnitCreated

Fired when a DisplayUnit is created in one of the DisplayUnitGroups

OnDisplayUnitGroupCleared

Fired when the display units in a DisplayUnitGroup collection are cleared

OnDisplayUnitGroupCreated

Fired when a DisplayUnitGroup is created

OnDisplayUnitGroupRemoved

Fired when a DisplayUnitGroup is removed

OnDisplayUnitGroupsCleared

Fired when the collection of DisplayUnitsGroups is cleared

OnDisplayUnitRemoved

Fired when a DisplayUnit is removed from a DisplayUnitGroup

OnFormatChanged

Fired when the formatting properties of a DisplayUnit are changed.

Restricted Properties

_EnumDisplayUnitGroups

Returns a DisplayUnitGroup enumerator
