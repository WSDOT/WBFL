UnitType

A UnitType object represents a general physical unit of measure such as "Length" or "Force". A physical unit of measure is described by its dimensionality.

Properties

Angle

Returns the dimensionality for angle

Label

Returns the name of this unit type

Length

Returns the dimensionality for length

Mass

Returns the dimensionality for mass

Temperature

Returns the dimensionality for temperature

Time

Returns the dimensionality for time

Units

Returns a collection of Unit objects

UnitServer

Returns the unit server

Events

OnUnitAdded

Fired when a unit of this type is added

OnUnitRemoved

Fired when a unit of this type is removed

OnUnitsCleared

Fired when the collection of units is cleared
