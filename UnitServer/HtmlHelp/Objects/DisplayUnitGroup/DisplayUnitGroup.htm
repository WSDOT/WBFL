DisplayUnitGroup

This object is a collection of DisplayUnit objects. DisplayUnit objects are grouped according to use. For example a display unit group may be named "Long Length" to represent a collection of DisplayUnit objects that have "Length" Unit objects. Furthermore, DisplayUnits objects in the group "Long Length" would have units of measure such as "Mile" and "Kilometer" rather than "inch" or "millimieter".

Properties

Count

Returns the number of DisplayUnit objects in this group

Default

Returns the default DisplayUnit object for the specified unit mode

Item

Returns a DisplayUnit object

Name

Returns the name of this group

UnitType

Returns the name of the unit type for this group

Methods

Add

Adds a new DisplayUnit object to this group

Clear

Removes all DisplayUnit objects from this group

Remove

Removes the speicified DisplayUnit object from this group

Events

OnDisplayUnitCreated

Fired when a DisplayUnit object is created and added to this group

OnDisplayUnitGroupCleared

Fired when all the DisplayUnit objects are cleared from this group

OnDisplayUnitRemoved

Fired when a single DisplayUnit object is removed from this group

OnFormatChanged

Fired when the formatting properties change on one of the DisplayUnit objects in this group

Restricted Properties

_EnumDisplay

Returns a DisplayUnit enumerator
