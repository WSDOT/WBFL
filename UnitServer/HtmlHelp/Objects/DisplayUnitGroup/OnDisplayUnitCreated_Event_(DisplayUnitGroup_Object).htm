OnDisplayUnitCreated Event (DisplayUnitGroup Object)

Summary:

Fired when a DisplayUnit object is created and added to this group

Syntax:

object_OnDisplayUnitCreated(objGroup,objDispUnit)

Element

Description

objGroup

Group in which the DisplayUnit object as added

objDispUnit

The newly created DisplayUnit object
