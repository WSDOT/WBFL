IEnumDisplayUnitGroups

This interface is used to enumerate DisplayUnitGroup objects in a DisplayUnitGroups collection. This interface is not compatible with VB, VBA, or VBS. This interface is primarily used by C/C++ clients.

Methods

Clone

Creates another enumerator that contains the same enumeration state as the current one.

Next

Retrieves a specified number of items in the enumeration sequence.

Reset

Resets the enumeration sequence to the beginning.

Skip

Skips over a specified number of items in the enumeration sequence.
