IUnitConvert2

Interface that exposes methods to perform basic unit conversions. Units are specified using Unit objects. Method calls on this interface are slightly less convienent, but faster than method calls on IUnitConvert

Methods

Convert

Converts a value between two arbitrary units of measure of the same type

ConvertToBaseUnits

Converts a value from the specified unit of measure to the base units

ConvertFromBaseUnits

Converts a value from base units to a specifed unit of measure
