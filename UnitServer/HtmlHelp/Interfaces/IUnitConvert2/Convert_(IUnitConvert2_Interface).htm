Convert (IUnitConvert2 Interface)

Syntax:

dblValue = object.Convert(dblFrom,objFromUnit,objToUnit)

Element

Description

dblValue

The converted value

dblFrom

The value to be converted

objFromUnit

The unit of measure to convert from

objToUnit

The unit of measure to convert to

object

The object implementing this interface
