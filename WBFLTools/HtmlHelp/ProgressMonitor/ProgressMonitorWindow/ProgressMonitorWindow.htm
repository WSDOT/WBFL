ProgressMonitorWindow

An object that can be used to display a window for monitoring the progress of a process. It can also dump progress information out to a file.

Properties

HasCancel

Set this to true if you want the window to have a cancel button

HasGauge

Set this to true if you want the window to display a progress gauge

Methods

Show

Show the progress window

Hide

Hide the progress window

Interfaces

IProgressMonitor

The incoming interface for progress messages and status

ILogProgressToFile

Log the progress to the specified file
