ShowAgain Property (ARP Notice Object)

Set/Get the state of the "Don't show me this again" check box on the Disclaimer step.

C++

Syntax

HRESULT put_ShowAgain(VARIANT_BOOL bShowAgain);

HRESULT get_ShowAgain(VARIANT_BOOL* bShowAgain);

Parameters

bShowAgain

State of the "Dont' show me this again" check box. If VARIANT_TRUE the check box is checked, otherwise it is unchecked

Return Value

S_OK if successful.

Remarks

It is your responsibility to save the state of the "Don't show me this again" check box between uses of this object. This includes saving the state between application runs. The system registry is a good place to save this information.

Example

VARIANT_BOOL bShowAgain = VARIANT_TRUE;
long lResult;
pARPNotice->put_ShowAgain( bShowAgain );
pARPNotice->Show( VARIANT_TRUE, &lResult );
pARPNotice->get_ShowAgain( &bShowAgain );

Visual Basic

Syntax

ShowAgain(bShowAgain As Boolean)

ShowAgain() As Boolean

Parameters

bShowAgain

State of the "Dont' show me this again" check box. If True the check box is checked, otherwise it is unchecked

Return Value

The get property returns the current state of the "Don't show me this again" check box.

Remarks

It is your responsibility to save the state of the "Don't show me this again" check box between uses of this object. This includes saving the state between application runs. The system registry is a good place to save this information.

Example

Dim bShowAgain As Boolean
Dim lResult As Long
bShowAgain = True
arpNotice.ShowAgain = bShowAgain
lResult = arpNotice.Show(True)
bShowAgain = arpNotice.ShowAgain
