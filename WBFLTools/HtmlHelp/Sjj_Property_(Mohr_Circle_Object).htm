Sjj Property (Mohr Circle Object)

Set/Get the stress in the j direction.

C++

Syntax

HRESULT put_Sjj(double Sjj);

HRESULT get_Sjj(double* Sjj);

Parameters

Sjj

Stress in the j direction

Return Value

S_OK if successful.

Remarks

None

Example

double sjj = 10.0;
pMohrCircle->put_Sjj(sjj);
pMohrCircle->get_Sjj(&sjj);

Visual Basic

Syntax

Sjj(Sjj As Double)

Sjj() As Double

Parameters

Sjj

Stress in the j direction

Return Value

The get property returns the current value of the stress in the j direciton.

Remarks

None

Example

Dim sjj As Double
sjj = 10#
mohrCircle.Sjj = sjj
sjj = mohrCircle.Sjj
