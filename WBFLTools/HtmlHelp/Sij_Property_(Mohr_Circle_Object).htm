Sij Property (Mohr Circle Object)

Set/Get the shearing stress.

C++

Syntax

HRESULT put_Sij(double Sij);

HRESULT get_Sij(double* Sij);

Parameters

Sij

Shearing stress

Return Value

S_OK if successful.

Remarks

None

Example

double sij = 10.0;
pMohrCircle->put_Sij(sij);
pMohrCircle->get_Sij(&sij);

Visual Basic

Syntax

Sij(Sij As Double)

Sij() As Double

Parameters

Sij

Shearing stress

Return Value

The get property returns the current value of the shearing stress.

Remarks

None

Example

Dim sij As Double
sij = 10#
mohrCircle.Sij = sij
sjj = mohrCircle.Sjj
