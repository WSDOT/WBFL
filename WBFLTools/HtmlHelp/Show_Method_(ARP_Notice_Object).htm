Show Method (ARP Notice Object)

Displays the Alternate Route Project license notice.

C++

Syntax

HRESULT Show(VARIANT_BOOL bGiveChoice,LicenseType lt,AcceptanceType* accept);

Parameters

bGiveChoice

Specifies if the notice should give the user a choice of accepting the license. If VARIANT_TRUE, the last step of the wizard will have an "I Accept" button and a "Cancel", otherwise it will have a "Finish" button.

lt

Indicates which license to display

accept

Result indicating the user's acceptance of the license agreement. If lResult is atAccept, the user has accepted the license (the "I Accept" or "Finish" button was pressed). If lResult is atReject, the user has not accepted the license (the "Cancel" button was pressed.

Return Value

S_OK if successful.

Remarks

In the event the user does not accept the license, the program must immediately terminate.

Example

AcceptanceType accept;
pARPNotice->Show( VARIANT_TRUE, ltAROSL, &accept );
if ( lResult == atReject )
{
//.. Terminate Application Now!!!
}
