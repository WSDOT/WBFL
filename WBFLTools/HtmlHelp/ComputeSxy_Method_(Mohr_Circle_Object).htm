ComputeSxy Method (Mohr Circle Object)

Computes the shearing stress Sxy at a specified orientation.

C++

Syntax

HRESULT ComputeSxy(double angle,double* sxy);

Parameters

angle

Orientation of the stress plane. Measured counter-clockwise from the positive i direction, in radians.

sxy

Shearing stress

Return Value

S_OK if successful.

Remarks

None

Example

pMohrCircle->put_Sii(10.0);
pMohrCircle->put_Sjj(-20.0);
pMohrCircle->put_Sij(-15.0);
pMohrCircle->get_Smin( &Smin );
double s;
pMohrCircle->ComputeSxy( 3.1415/4, &s );

Visual Basic

Syntax

ComputeState(angle As Double) As Double

Parameters

angle

Orientation of the stress plane. Measured counter-clockwise from the positive i direction, in radians.

Return Value

Shearing stress

Remarks

None

Example

mohrCircle.Sjj = -20#
mohrCircle.Sij = -15#
