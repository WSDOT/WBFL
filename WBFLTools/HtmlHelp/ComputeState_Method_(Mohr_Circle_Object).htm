ComputeState Method (Mohr Circle Object)

Computes the state of stresses at a specified orientation.

This method will not work for VBScript or JavaScript clients. Use ComputeSxx, ComputeSyy, and ComputeSxy instead.

C++

Syntax

HRESULT ComputeState(double angle,double* S11,double* S22,double* S12);

Parameters

angle

Orientation of the stress plane. Measured counter-clockwise from the positive i direction, in radians.

S11

Normal stress in the 1-1 direction

S22

Normal stress in the 2-2 direction

S12

Shearing stress

Return Value

S_OK if successful.

Remarks

None

Example

pMohrCircle->put_Sii(10.0);
pMohrCircle->put_Sjj(-20.0);
pMohrCircle->put_Sij(-15.0);
pMohrCircle->get_Smin( &Smin );
VARIANT s11, s22, s12;
pMohrCircle->ComputeState( 3.1415/4, &s11, &s22, &s12 );
