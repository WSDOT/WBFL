BeginUnit Method

(StructuredLoad2 Interface)

Summary:

Sets the cursor down a level in the unit hierarchy to the unit with the given name. The cursor is placed at the first data member in the Unit.

Syntax:

object.BeginUnit strUnitName

Element

Description

strUnitName

Name of the Unit to be opened

object

This IStructuredLoad2

Remarks:

You can get the version of the current Unit by calling the Version property after you have called BeginUnit
