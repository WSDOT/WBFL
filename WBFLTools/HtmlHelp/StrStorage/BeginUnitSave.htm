BeginUnit Method

(IStructuredSave2 Interface)

Summary:

Creates a new Unit with the given name and sets the cursor down a level in the unit hierarchy to the start of the new unit.

Syntax:

object.BeginUnit strUnitName dlbVersion

Element

Description

strUnitName

Name of the Unit to be opened

dblVersion

Version ID for the newly created Unit

object

This IStructuredSave2

Remarks:

You can get the version of the current Unit by calling the Version property after you have called BeginUnit.

Calls to BeginUnit / EndUnit must be balanced.
