Property

(IStructuredLoad2 Interface)

Summary:

Loads the given variant from the persistence medium.

Syntax:

varProp = object.Property( strPropName )

Element

Description

varProp

Variant to be assigned the property in question

strPropName

Name of the property in question

object

This IStructuredLoad2

Remarks:

This method supports most, but not all of the types available for VARIANTs. Below is a list of the supported types.

Type

Description

VT_BSTR

Variables of type BSTR. BSTR's are the default string type for Visual Basic clients

VT_R4

32 bit (single precision) floating point values

VT_R8

64 bit (double precision) floating point values

VT_I2

Short (16 bit) integers

VT_I4

Long (32 bit) integers

VT_UI2

Short unsigned integers

VT_UI4

Long unsigned integers

VT_BOOL

Booleans of type VARIANT_BOOL

VT_UNKNOWN
VT_DISPATCH

 

COM Objects which support the IUknown or IDispatch interfaces. All objects stored using the Property method must have the following attributes:

  1. They must be creatable (via new in Visual Basic or CoCreateInstance in C++)

  2. They must implement the IPersist interface. This is true in Visual Basic for ActiveX objects that have their "Persistable" property set equal to 1. Implementors C++ objects have to support this method manually, or via ATL's methods.

  3. They must implement the IStructuredStorage2 interface.
