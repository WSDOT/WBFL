Version

(IStructuredLoad2 Interface)

Summary:

Gets the version of the Unit currently at the cursor.

Syntax:

fltVers = object.Version

Element

Description

fltVers

Version of the current Unit.

object

This IStructuredLoad2

Remarks:

Will return zero if the version was not defined
