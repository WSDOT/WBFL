IStructuredStorage2

An interface to be implemented by classes that are to be made persistable using the structured storage framework.

This interface, along with IPersist, must be implemented by objects that wish to support persistence using the Property methods on the IStructuredSave2 and IStructuredLoad2 interfaces.

Methods

Load

Load the object from persistence

Save

Save the object to persistence
