StrandPattern Object

An object that represents a prestressing strand pattern.

 

Pattern Points - Points that define the strand pattern. Pattern points are mapped into StrandPoints. Pattern points can be reflected to create a second strand point mirrored about the Y-axis

 

Strand Point - The actual points where strand can be located in a strand pattern
