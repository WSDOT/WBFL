iLegendDisplayObject

Defines a rectangular legend display object. The legend displays iLegendEntry's laid out in a grid-like arrangement.

void get_Position(IPoint2d* pos,BOOL bRedraw,BOOL bFireEvent)
void put_Position(IPoint2d* *pos)

Get/set the location of the top corner of the Legend.

pos

Position of the top left corner of the legend in world coordinates

bRedraw

If not FALSE, the legend will be redrawn after it is moved.

bFireEvent

If not FALSE, fire an OnMoved event

void get_Title(BSTR *pVal)
void put_Title(BSTR *pVal)

Get/set the title for the Legend. If not blank, the title is placed above the legend.

pVal

String that is the title of the legend box

void get_Font(LOGFONT* pFont)
void put_Font(const LOGFONT& Font)

Get/set the font to be used drawing legend entry titles. A bold version of this font will be used for the Legend title.

pFont

 

A Windows LOGFONT structure defining the font to be used. See the MSDN documentation for CreatePointFontIndirect for the expected format of this structure.

void get_NumEntries(long* count)

Get the number of iLegendEntry currently in the legend

count

Number of entries

void AddEntry)(iLegendEntry* entry)

Append a new iLegendEntry to the legend

entry

entry to be added

HRESULT InsertEntry(long index, iLegendEntry* entry)
Insert a new iLegendEntry to the legend at the given location

index

location in collection where entry is to be inserted

entry

entry to be added

HRESULT get_Entry(long index, iLegendEntry* *entry)

Get iLegendEntry at the given location

index

location in collection where entry is to be inserted

entry

entry to be retrieved

HRESULT RemoveEntry(long index)

Remove an iLegendEntry from the legend

index

location in collection of entry to be removed

void ClearEntries()

Remove all iLegendEntry's

void get_NumRows(long* count)
void put_NumRows(long count)

Get/set the number of rows to be displayed in the legend. If this value is more than the number of entries, blank cells will be shown.

count

 

Number of rows to be shown in legend. The number of columns will be determined based on this value and the number of entries

void get_CellSize(CSize* size)
void put_CellSize(CSize size)

Get/set the size of the cell that holds each legend entry. If this value is smaller than the size returned by GetMinCellSize, the min cell size will be used for drawing.

size

 

Size, in twips, of the legend entry cells in the legend

void get_GetMinCellSize(CSize* size)

Get the minimum allowable size of cells that hold each legend entry. This value is computed based on the current font size and maximum title length of the current legend entries

size

 

Minimum size, in twips, of the legend entry cells in the legend

void get_DoDrawBorder(BOOL *pVal)
void put_DoDrawBorder(BOOL Val)

 
Get/set wheter a border is to be drawn around the legend.

pVal

If not FALSE, a border is drawn around the legend entry area

void get_DoFill(BOOL *pVal)
void put_DoFill(BOOL Val)

 
Get/set wheter the legend is to be filled with the current color, or left to be transparent.

pVal

If not FALSE, the legend is filled using the fill color

void get_FillColor(COLORREF* color)
void put_FillColor(COLORREF color)

Get/set the color that the legend is to be filled with.

color

Fill color

void get_IsDraggable(BOOL *pVal)
void put_IsDraggable(BOOL Val)

 
Get/set wheter the legend can be dragged around the screen by the user.

pVal

If not FALSE, the legend is draggable
