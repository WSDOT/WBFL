iSymbolLegendEntry

Interface to a type of iLegendEntry that uses font symbols to draw itself.

void put_Name(BSTR *pVal)

Set the name for the legend entry. Th entry name is put into the legend next to its symbol.

pVal

String that is the name for this entry

void get_SymbolFontFace(char* *pName)
void put_SymbolFontFace(char* pNamel)

Get/set the name of the Windows font fact that will be used to draw the symbol.

pName

Name of the Windows font face (e.g., "Symbol"). Note that the font face must exist on the target machine

void get_SymbolSize(long *pSize)
void put_SymbolSize(long Size)

Get/set the size of the symbol in 1/10 Points.

Size

Size of the symbol in 1/10 points

void get_SymbolCharacterCode(DWORD* pCode)
void put_SymbolCharacterCode(DWORD Code)

Get/set the ascii code the the symbol to be drawn in the selected font face

Code

Code for symbol to be drawn. A good reference for this is the "Insert | Symbol" dialog in MSFT Word.

If this code is zero, then no symbol will be drawn and the entire legend rectangle will be filled with this entry's color.

void get_Color(COLORREF* color)
void put_Color(COLORREF color)

Get/set the color of the symbol.

color

Symbol color

void get_DoDrawLine(BOOL *pVal)
void put_DoDrawLine(BOOL Val)

 
Get/set wheter a horizontal line is to be drawn through the symbol in the legend.

pVal

If not FALSE, a line is drawn across the middle of the legend entry rectangle.
