iDataPointFactory

Interface for creating data point display objects. This interface is used by a graph to create its data points. At a minimum, the object created must support the iDisplayObject and iConnectable interfaces

HRESULT CreateDataPoint(double dataX, double dataY, double graphX, double graphY, iDisplayObject** dataPoint)

Create a graph data point display object. This factory method is used by the GraphXyDisplayObject to create data points for a iGraphXyDataProvider

dataX

X value of data for the point

dataY

Y value of data for the point

graphX

X value in graph coordinates (computed by the graph)

graphY

Y value in graph coordinates (computed by the graph)

dataPoint

graph data point display object created
