TemporaryPierItemCollection Object

A collection of SuperstructureMember objects.

Properties

Count

Returns the number of temporary piers

Item

Returns a temporary pier item object

Methods

Add

Adds a temporary pier to the collection

Clear

Removes all of the temporary piers

Find

Finds a temporary pier by ID

RemoveByID

Removes a temporary pier by ID

RemoveByIndex

Removes a temporary pier by Index

Events

OnTemporaryPierItemAdded

Fired when a temporary pier is added

OnTemporaryPierItemChanged

Fired when a temporary pier changes

OnTemporaryPierItemRemoved

Fired when a temporary pier is removed
