DeflectionMomentOfInertia Property (SegmentCrossSection Object)

Summary:

Sets/Gets the moment of inertia used for deflection calculations

Syntax:

value = object.DeflectionMomentOfInertia
object.DeflectionMomentOfInertia = value

Element

Description

value

The value of this property

object

This object

Remarks:

While the generic bridge model is not intended to be dependent on a specific bridge code, this property is heavily influenced by LRFD 2.5.2.6.2 which states in part...

"For composite design, the design cross-section should include the entire width of the roadway and the structurally continuous portions of the railings, sidewalks, and median barriers".

This gives rise to the need for a different moment of inertia when computing deflection as it does for computing other structural responses.
