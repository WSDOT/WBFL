GirderSpacing Object

An object that describes the spacing between girder lines and the roadway surface overhangs.

Properties

GirderLineCount

Returns the number of girder lines

GirderLineIndex

Returns the index of a girder line

Measurement

Sets/Gets the method for measuring girder spacing

Overhang

Sets/Gets the roadway surface overhang

Spacing

Sets/Gets the girder line spacing

SuperstructureMembers

Returns a collection of superstructure members

Uniform

Specifies that the spacing is uniform

Width

Returns the total width of the roadway cross section

Methods

Add

Adds girder lines and spaces

Insert

Inserts girder lines and spaces

Remove

Removes girder lines and spaces

Events

OnGirderLineCountChanged

Fires when the number of girder lines change

OnGirderSpacingChanged

Fires when the girder spacing changes
