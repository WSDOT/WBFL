Measurement Property (GirderSpacing Object)

Summary:

Sets/Gets the method for measuring girder spacing

Syntax:

object.Measurement(pierIndex) = measure
measure = object.Measure(pierIndex)

Element

Description

measure

The girder line measurement type

pierIndex

The pier where this applies

object

This object

Remarks:

Girder spacing and overhangs can be measured along the centerline of the pier or normal to the alignment.
