GenericBridge Object

An object that describes a bridge by its geometrical, material, and stiffness properties.

Properties

Alignment

Returns the roadway alignment object

CogoInfo

Returns the coordinate geometry information object

CogoModel

Returns the underlying coordinate geometry model

GirderLineLength

Returns the length of a girder line

GirderLineType

Sets/Gets the girder line type

GirderLineCount

Returns the number of girder lines

GirderSpacing

Returns the girder spacing object

Piers

Returns a collection of pier objects

PointsOfInterest

Returns a collection of points of interest

Spans

Returns a collection of span objects

Stages

Returns a collection of stage objects

SuperstructureMembers

Returns a collection of superstructure member objects

Methods

Clone

Creates a copy of this object

InsertSpanAndPier

Inserts a span and pier into the bridge

Move

Moves the bridge by a specified amount

MoveToStation

Moves a pier to a specified station

RemoveSpanAndPier

Removes a span and pier from the bridge

ResizeSpans

Resizes all of the spans

Events

OnBridgeChanged

Fires when the bridge changes
