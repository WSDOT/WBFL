StressPointCollection Object

A collection of StressPoint objects.

Properties

Count

Returns the number of stress points

Item

Returns a stress point

Methods

Add

Adds a stress point to the collection

Clear

Removes all stress points from the collection

Insert

Inserts a stress point into the collection

Remove

Removes a stress point from the collection

Events

OnStressPointAdded

Fired when a stress point is added

OnStressPointChanged

Fired when a stress point changes

OnStressPointRemoved

Fired when a stress point is removed
