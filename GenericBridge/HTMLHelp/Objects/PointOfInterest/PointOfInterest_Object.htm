PointOfInterest Object

Defines a point in the structure where analysis results are desired. Points of interest can be placed on any member of the bridge model, as defined by the MemberType property. The various identifying properties are used to describe which specific member the point of interest is attached to. The table below shows which indexed properties are applicable for the various member types. Where more than one combination of indices can be used, the unused indicies must be -1.

MemberType

Applicable Index Property

mtSpan

SpanIdx + GirderLineIdx
SpanIdx + GirderLineIdx + SuperstructureMemberIdx
SpanIdx + GirderLineIdx + SuperstructureMemberIdx + SegmentIdx

mtGirderLine

GirderLineIdx
GirderLineIdx + SuperstructureMemberIdx
GirderLineIdx + SuperstructureMemberIdx + SegmentIdx

mtLongitudinalSupport

SupportIdx
SupportIdx + SegmentIdx

mtCrossBeam

SupportIdx
SupportIdx + SegmentIdx

mtColumn

SupportIdx + ColumnIdx
SupportIdx + ColumnIdx + SegmentIdx
