MemberType Property (PointOfInterest Object)

Summary:

Sets/Gets the member type

Syntax:

object.MemberType = mt
mt = object.MemberType

Element

Description

mt

The member type

object

This object

Remarks:

A change in member type is generally associated with modifications to the member index values. Make sure valid member index values are assigned for the member type after changing it. Failure to do so will result in this point of interest becoming unavailable as a means to access analysis results
