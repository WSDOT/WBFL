Pier Object

An object that represents pier.

Properties

Direction

Returns a direction object that represents the orientation of the pier

LinkCount

Returns the number of piers linked to this pier

LinkSource

Returns the pier this pier is linked to

LinkTarget

Returns a pier that is linked to this pier

LongitudinalPierDescription

Returns the longitudinal pier description object

Offset

Sets/Gets the pier offset

Orientation

Sets/Gets the pier orientation

SkewAngle

Returns the skew angle of the pier

Station

Sets/Gets the pier station

TransversePierDescription

Returns the transverse pier description object

Methods

CreateTransversePierDescription

Creates a transverse pier description object

Link

Links another pier to this pier

RemoveTransversePierDescription

Removes the transverse pier description object

Unlink

Unlinks this pier from its source pier

Events

OnPierChanged

Fires when the pier changes
