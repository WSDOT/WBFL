Offset Property (SuperstructureMemberCollection Object)

Summary:

Gets/Sets the collection offset (used to defined cantilevers)

Syntax:

offset = object.Offset(end)
object.Offset(end) = offset

Element

Description

offset

The offset

end

The end of the collection for this offset

object

This object

Remarks:

The beginning and ending of the superstructure member collection may be offset from the first and last pier, respectifully. Offsets positive values. The offset distance creates a cantilever at the start/end of the first/last span
