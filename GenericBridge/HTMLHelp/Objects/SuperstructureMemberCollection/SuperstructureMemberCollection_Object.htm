SuperstructureMemberCollection Object

A collection of SuperstructureMember objects.

Properties

Count

Returns the number of items in the collection

Fractional

Indicates if superstructure member lengths are fractional

Item

Returns an item from the collection

LinkSource

Returns the link source

LinkTarget

Returns a link target

LinkCount

Returns the number of targets linked to this source

MemberLength

Returns the length of all the superstructure members

Offset

Gets/Sets the collection offset (used to defined cantilevers)

Symmetrical

Indicates if the superstructure members are symmetrical about the midpoint of the girder line

Methods

Add

Adds a new superstructure member to the collection

Clear

Clears the collection

CopyTo

Copies a superstructure member

Link

Links a superstructure member colleciton to this collection

MoveTo

Moves a superstructure member to a different position in the collection

Remove

Removes a superstructure member from the collection

Reverse

Reverses the order of superstructure members in the collection

Unlink

Unlinks this collection of its link source

Events

OnSuperstructureMemberAdded

Fired when a superstructure member is added to the collection

OnSuperstructureMemberChanged

Fired when a superstructure member changes

OnSuperstructureMemberCollectionChanged

Fired when the collection changes

OnSuperstructureMemberMoved

Fired when a superstructure member is moved

OnSuperstructureMemberRemoved

Fired when a superstructure member is removed
