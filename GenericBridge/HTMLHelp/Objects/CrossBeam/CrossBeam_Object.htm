CrossBeam Object

An object that represents a cross beam in a transverse pier description model. A cross beam is described by a series of prismatic segments, starting at the left hand side of the beam

Properties

Fractional

Specifies that segment lengths use fractional measure

Length

Returns the centerline length of the cross beam

Segment

Returns a segment

SegmentCount

Returns the total number of segments

SegmentLength

Returns the total length of all the segments

Symmetrical

Specifies that segments are defined symmetrically about the midpoint of the cross beam

Methods

AddSegment

Adds a segment to the cross beam

CopySegmentTo

Inserts a copy of a segment at a specified index

GetMemberSegments

Returns a collection of segments that exactly fit the length of the cross beam

GetSegmentForMemberLocation

Returns a segment at a specifed location along the cross beam

InsertSegment

Inserts a new segment at a specified index

MoveSegmentTo

Moves a segment to a new index

RemoveSegment

Removes a segment from the cross beam

RemoveSegments

Removes all of the segments from the cross beam

ReverseSegments

Reverses the order of the segments

Events

OnCrossBeamChanged

Fired when the cross beam changes
