Column Object

An object that represents a column in a transverse pier description model. A column is described by a series of prismatic segments, starting at the base of the column.

Properties

BaseOffset

Sets/Gets the base offset

Fractional

Specifies that segment lengths use fractional measure

Height

Sets/Gets the column height

Length

Returns the centerline length of the column

Segment

Returns a segment

SegmentCount

Returns the total number of segments

SegmentLength

Returns the total length of all the segments

Symmetrical

Specifies that segments are defined symmetrically about the midpoint of the column

Methods

AddSegment

Adds a segment to the column

CopySegmentTo

Inserts a copy of a segment at a specified index

GetMemberSegments

Returns a collection of segments that exactly fit the length of the column

GetSegmentForMemberLocation

Returns a segment at a specifed location along the column

InsertSegment

Inserts a new segment at a specified index

MoveSegmentTo

Moves a segment to a new index

RemoveSegment

Removes a segment from the column

RemoveSegments

Removes all of the segments from the column

ReverseSegments

Reverses the order of the segments

Events

OnColumnChanged

Fired when the column changes
