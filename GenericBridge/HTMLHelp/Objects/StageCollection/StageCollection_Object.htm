StageCollection Object

A collection of stage objects. The order of a stage within the collection defines when this stage occurs in the life of the bridge. Stages at lower indicies occur earlier.

Properties

Count

Returns the number of stages in the collection

Item

Returns a stage object

Methods

CreateStage

Creates a stage

CreateStageAt

Create a stage at a specified position in the collection

Find

Finds a stage based on its name

MoveStage

Moves a stage to a new position

RemoveByIndex

Removes a stage by index

RemoveByName

Removes a stage by name

Events

OnStageAdded

Fired when a stage is added to the collection

OnStageDescriptionChanged

Fired when the description of a stage changes

OnStageMoved

Fired when a stage is moved within the collection

OnStageRemoved

Fired when a stage is removed from the collection

OnStageRenamed

Fired when a stage is renamed
