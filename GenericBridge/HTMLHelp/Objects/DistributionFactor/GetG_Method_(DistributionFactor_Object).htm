GetG Method (DistributionFactor Object)

Summary:

Gets all of the distribution factors at once

Syntax:

object.GetG(gDMul,gDSgl,gFat,gNMMul,gNMSgl,gPMMul,gPMSgl,gRMul,gRSgl,gTMul,gTSgl,gVMul,gVSgl)

Element

Description

gDMul

Lane fraction for deflection for 2 or more loaded lanes

gDSgl

Lane fraction for deflection for one loaded lane

gFat

Lane fraction for fatigue

gNMMul

Lane fraction for negative moment for 2 or more loaded lanes

gNMSgl

Lane fraction for negative moment for one loaded lane

gPMMul

Lane fraction for positive moment for 2 or more loaded lanes

gPMSgl

Lane fraction for positive moment for one loaded lane

gRMul

Lane fraction for reactions for 2 or move loaded lanes

gRSgl

Lane fraction for reactions for one loaded lane

gVMul

Lane fraction for shear for 2 or more loaded lanes

gVSgl

Lane fraction for shear for one loaded lane

This object
