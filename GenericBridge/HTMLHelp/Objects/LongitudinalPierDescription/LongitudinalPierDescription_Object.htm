LongitudinalPierDescription Object

An object that represent a pier when used for longitudinal analysis

Properties

BaseOffset

Sets/Gets the base offset

Connectivity

Sets/Gets the top of column connectivity

ContinuityStage

Sets/Gets the continuity stage

Fractional

Specifies that segment lengths use fractional measure

Height

Sets/Gets the height of the pier

Length

Returns the length of the pier column

Pier

Returns the pier this model is associated with

Segment

Returns a segment

SegmentCount

Returns the number of segments

Symmetrical

Indicates if the segments are symmetrical about the midheight of the column

Methods

AddSegment

Adds a segment to the m odel

CopySegmentTo

Inserts a copy of a segment at a specified index

GetMemberSegments

Returns a collection of segments that exactly fit the length of the column

GetSegmentForMemberLocation

Returns a segment at a specifed location along the column

InsertSegment

Inserts a new segment at a specified index

MoveSegmentTo

Moves a segment to a new index

RemoveSegment

Removes a segment from the column

RemoveSegments

Removes all of the segments from the column

ReverseSegments

Reverses the order of the segments

Events

OnChanged

Fires when the model changes
