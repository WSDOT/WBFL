SuperstructureMember Object

The structural elements of the superstructure are called superstructure members. These members follow the girder lines along the superstructure member. Superstructure members are defined end to end along a girder line. Superstructure member need not start and end at piers. A superstructure member is comprised of a series of prismatic segments.

Properties

Fractional

Indicates if the segments are defined with fractional measure

Length

Total length of the superstructure member

Segment

Returns a segment

SegmentCount

Returns the number of segments

SegmentLength

Returns the sum of the length of segments

Symmetrical

Indicates if the segments are symmetrical about the midpoint of the superstructure member

Methods

AddSegment

Adds a segment to the member

CopySegmentTo

Inserts a copy of a segment at a specified index

GetEndRelease

Returns the member end release

GetMemberSegments

Returns a collection of segments that exactly fit the length of the member

GetSegmentForMemberLocation

Returns a segment at a specifed location along the member

InsertSegment

Inserts a new segment at a specified index

MoveSegmentTo

Moves a segment to a new index

RemoveSegment

Removes a segment from the member

RemoveSegments

Removes all of the segments from the member

ReverseSegments

Reverses the order of the segments

SetEndRelease

Sets the member end release

Events

OnSuperstructureMemberChanged

Fired when the superstructure member changes
