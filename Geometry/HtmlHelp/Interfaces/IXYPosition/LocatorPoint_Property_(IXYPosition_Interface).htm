LocatorPoint Property (IXYPosition Interface)

Gets/Sets the location of a locator point.

C++

Syntax

HRESULT put_LocatorPoint(LocatorPointType lp,IPoint2d* pnt);

HRESULT get_LocatorPoint(LocatorPointType lp,IPoint2d** pnt);

Parameters

lp

Locator point constant

pnt

Location of locator point

Return Value

S_OK if Successful

Remarks

The put version moves the specified locator point to the location specified by pnt. The get version retrieves the current location of the specified locator point.

Visual Basic

Syntax

LocatorPoint(lp As LocatorPointType,pnt As Point2d)

LocatorPoint(lp As LocatorPointType) As Point2d

Parameters

lp

Locator point constant

pnt

Location of locator point

Return Value

The Get version returns the current location of the specified locator point.
