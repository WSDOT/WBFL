MoveEx Method (IXYPosition Interface)

Moves the object by moving an arbitrary point from one location to another.

C++

Syntax

HRESULT MoveEx(IPoint2d* from,IPoint2d* to);

Parameters

from

The point to be moved

to

The location where from will be moved to.

Return Value

S_OK if Successful

Remarks

None

Visual Basic

Syntax

Move(from As Point2d,to As Point2d)

Parameters

from

The point to be moved

to

The location where from will be moved to.

Return Value

None

Remarks
