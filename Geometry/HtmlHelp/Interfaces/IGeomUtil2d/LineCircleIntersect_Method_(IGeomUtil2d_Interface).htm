LineCircleIntersect Method (IGeomUtil2d Interface)

Finds the intersection of a line and a circle.

C++

Syntax

HRESULT LineCirlceIntersect(ILine2d* l,ICircle* c,VARIANT* p1,VARIANT* p2,short* nIntersect);

Parameters

l

The line

c

The circle

p1

The first intersection point

p2

The section intersection point

nIntersect

Number of intersections found

Return Value

S_OK if Successful

Remarks

nIntersect contains the number of intersection points found. If no intersections were found, p1 and p2 are Nothing. If one intersection is found, p2 is Nothing.

Visual Basic

Syntax

LineCircleIntersect(l As Line2d,c As Circle,p1,p2) As Short

Parameters

l

The line segment

c

The circle

p1

The first intersection point

p2

The section intersection point

Return Value

Returns the number of intersections found

Remarks

If the number of intersections found is zero, p1 and p2 are Nothing. If only one intersection was found, p2 is Nothing.
