CreateParallelLineSegment Method (IGeomUtil2d Interface)

Creates a line segment that is parallel to the specified line segment.

C++

Syntax

HRESULT CreateParallelLineSegment(ILineSegment2d* ls,double dist,ILineSegment2d** pls);

Parameters

ls

The source line segment

dist

Distance to offset the parallel line segment from the source line segment. The parallel line segment will be on the right side of the source line segment if this parameter is greater than zero.

pls

The parallel line segment

Return Value

S_OK if Successful

Remarks

None

Visual Basic

Syntax

CreateParallelLineSegment(ls As LineSegment2d,dist As Double) As LineSegment2d

Parameters

ls

The source line segment

dist

Distance to offset the parallel line segment from the source line segment. The parallel line segment will be on the right side of the source line segment if this parameter is greater than zero.

Return Value

Returns the parallel line segment

Remarks
