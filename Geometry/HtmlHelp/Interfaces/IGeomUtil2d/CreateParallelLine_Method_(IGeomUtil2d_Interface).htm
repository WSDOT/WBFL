CreateParallelLine Method (IGeomUtil2d Interface)

Creates a line that is parallel to the specified line.

C++

Syntax

HRESULT CreateParallelLine(ILine2d* l,double dist,ILine2d** pl);

Parameters

l

The source line

dist

Distance to offset the parallel line from the source line. The parallel line will be on the right side of the source line if this parameter is greater than zero.

pl

The parallel line

Return Value

S_OK if Successful

Remarks

None

Visual Basic

Syntax

CreateParallelLine(l As Line2d,dist As Double) As Line2d

Parameters

l

The source line

dist

Distance to offset the parallel line from the source line. The parallel line will be on the right side of the source line if this parameter is greater than zero.

Return Value

Returns the parallel line

Remarks

None
