PointOnLineNearest Method (IGeomUtil2d Interface)

Finds the point on a line nearest a specified point.

C++

Syntax

HRESULT PointOnLineNearest(ILine2d* l,IPoint2d* p,IPoint2d** poln);

Parameters

l

The line

p

The point

poln

The point on the line nearest the specified point

Return Value

S_OK if Successful

Remarks

None

Visual Basic

Syntax

PointOnLineNearest(l As Line2d,p As Point2d) As Point2d

Parameters

l

The line

p

The point

Return Value

Returns the point on the line nearest the specified point.

Remarks

None
