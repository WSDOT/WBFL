CreateNormalLineThroughPoint Method (IGeomUtil2d Interface)

Creates a line normal to the specified line, and passing through a specified point. The normal line is directed to the right of the specified line.

C++

Syntax

HRESULT CreateNormalLineThroughPoint(ILine2d* l,IPoint2d* p,ILine2d** n);

Parameters

l

The source line

p

The point that the normal line passes through

n

The normal line

Return Value

S_OK if Successful

Remarks

None

Visual Basic

Syntax

CreateNormalLineThroughPoint(l As Line2d,p As Point2d) As Line2d

Parameters

l

The source line

p

The point that the normal line passes through

Return Value

Returns the normal line

Remarks

None
