Angle Method (IGeom2d Interface)

Computes the angle formed by three points.

C++

Syntax

HRESULT Angle(IPoint2d* start,IPoint2d* center,IPoint2d* end,double* angle);

Parameters

start

Starting point

center

Vertex point

end

Ending point

angle

Angle, measured in radians

Return Value

S_OK if Successful

Remarks

None

Visual Basic

Syntax

Angle(start As Point2d,center As Point2d,end As Point2d) As Double

Parameters

start

Starting point

center

Vertex point

end

Ending point

Return Value

Returns the angle in radians

Remarks

None
