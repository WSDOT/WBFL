CircleCircleIntersect Method (IGeomUtil2d Interface)

Finds the intersection of two circles.

C++

Syntax

HRESULT CircleCirlceIntersect(ICircle* c1,ICircle* c2,VARIANT* p1,VARIANT* p2,short* nIntersect);

Parameters

c1

The first circle

c2

The second circle

p1

The first intersection point

p2

The section intersection point

nIntersect

Number of intersections found

Return Value

S_OK if Successful

Remarks

nIntersect contains the number of intersection points found. If no intersections were found, p1 and p2 are Nothing. If one intersection is found, p2 is Nothing.

If c1 and c2 have the same center and radius, p1 and p2 are Nothing and nIntersect is 3.

Visual Basic

Syntax

CircleCircleIntersect(c1 As Circle,c2 As Circle,p1,p2) As Short

Parameters

c1

The first circle

c2

The second circle

p1

The first intersection point

p2

The section intersection point

Return Value

Returns the number of intersections found

Remarks

If the number of intersections found is zero, p1 and p2 are Nothing. If only one intersection was found, p2 is Nothing.

If c1 and c2 have the same center and radius, p1 and p2 are Nothing and the return value is 3.
