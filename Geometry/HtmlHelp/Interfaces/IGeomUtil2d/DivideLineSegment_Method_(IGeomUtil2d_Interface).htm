DivideLineSegment Method (IGeomUtil2d Interface)

Divides a line segment into equal length pieces

The coordinates of N-1 points are computed, where N is the number of spaces.

C++

Syntax

HRESULT DivideLineSegment(ILineSegment2d* ls,long nSpaces,IPoint2dCollection** points);

Parameters

ls

The line segment

nSpaces

Number of spaces

points

Collection of points

Return Value

S_OK if Successful

Remarks

The points collection contains all the computing points as well as the start and end point of the line segment.

Visual Basic

Syntax

DivideLineSegment(ls As LineSegment2d,nSpaces As Long) As Point2dCollection

Parameters

ls

The line segment

nSpaces

Number of spaces

Return Value

Returns a collection of points

Remarks

The points collection contains all the computing points as well as the start and end point of the line segment.
