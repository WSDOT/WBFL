GenerateCircle Method (IGeomUtil2d Interface)

Generates a series of points that form a circle.

C++

Syntax

HRESULT GenerateCircle(long nPoints,IPoint2d* center,double radius,double initAngle,IPoint2dCollection** points);

Parameters

nPoints

Number of points to generate

center

Center of the circle

radius

Radius of the circle

initAngle

Counter-clockwise angle from the positive X axis, measured in radians, that determines where the first point is placed

points

A collection of points

Return Value

S_OK if Successful

Remarks

None

Visual Basic

Syntax

GenerateCircle(nPoints As Long,center As Point2d,radius As Double,initAngle As Double) As Point2dCollection

Parameters

nPoints

Number of points to generate

center

Center of the circle

radius

Radius of the circle

initAngle

Counter-clockwise angle from the positive X axis, measured in radians, that determines where the first point is placed

Return Value

Returns a collection of points

Remarks

None
