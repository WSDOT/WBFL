PolyPoints Method (IShape Interface)

Creates a collection of points that is a polygon representation of the shape.

C++

Syntax

HRESULT PolyPoints(IPoint2dCollection** points);

Parameters

points

Pointer to a collection of points

Return Value

S_OK if Successful

Remarks

None

Visual Basic

Syntax

PolyPoints() As Point2dCollection

Parameters

None

Return Value

Returns a collection of points

Remarks

None
