Revisions

Version 1.1.004

  • Added shape for Nebraska University (NU) Girders

Version 1.1.003

  • Made the geometry objects persistable

Version 1.1.002

  • The points generated by IGeomUtil2d::DivideArc were incorrectly based on a clockwise arc. The points should be based on a counter-clockwise arc. This problem has been corrected.

  • The CreateParallelLine method of the GeomUtil object created parallel lines on the wrong side of the source line. A parallel line to the right of the source line is supposed to be created for a positive offset distance. A parallel line on the left of the source line was actually created. This problem has been corrected.

  • The Angle method on the IGeomUtil2d interface computed the angle formed by three points incorrectly. The document indicates that the computed angle is counter-clockwise. The method was computing a clockwise angle. The method has been updated to compute a counter-clockwise angle.

Version 1.1

  • Added CircularSegment object. This object represents that shape of a circular segment.

  • Implemented the IShapePropertiesEx interface on the ShapeProperties object. This interface extends IShapeProperties and has one method, AddProperties. AddProperties adds together two ShapeProperties objects, using the parallel axis theorem.

  • Added the ShapeCollection object. This object is a standard collection that contains objects implementing the IShape interface. The collection can be viewed as a composite shape in that you can obtain the combined shape properties for all of the shapes in the collection. However, this collection does not implemented the IShape interface.

  • Fixed Vector2d and Vector3d so that the default vector had the value (1,0) and (1,0,0), respecitvely
