MidOrdinate Property (CircularSegment Object)

Depth of the segment

C++

Syntax

HRESULT put_MidOrdinate(double mo);

HRESULT get_MidOrdinane(double* mo);

Parameters

mo

Mid-ordinate. Valid inputs are between 0 and 2*Radius

Return Value

Returns GEOMETRY_E_MIDORDINATE if the input is out of range, otherwise returns S_OK.

Remarks

None

Visual Basic

Syntax

MidOrdinate(mo As Double)

MidOrdinate() As Double

Parameters

mo

Mid-ordinate. Valid inputs are between 0 and 2*Radius

Return Value

The get property returns the mid-ordinate

Remarks

None
