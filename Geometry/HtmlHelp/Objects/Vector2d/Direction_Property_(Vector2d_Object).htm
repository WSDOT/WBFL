Direction Property (Vector2d Object)

Direction of the vector

C++

Syntax

HRESULT put_Direction(double dir);

HRESULT get_Direction(double* dir);

Parameters

dir

Direction of the vector. This is an angle, in radians, measured from the positive X axis in a counter-clockwise direction.

Return Value

S_OK if Successful

Remarks

None

Visual Basic

Syntax

Direction(dir As Double)

Direction() As Double

Parameters

dir

Direction of the vector. This is an angle, in radians, measured from the positive X axis in a counter-clockwise direction.

Return Value

Returns the current direction of the vector

Remarks

None
