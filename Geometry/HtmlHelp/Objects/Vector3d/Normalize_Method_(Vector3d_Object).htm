Normalize Method (Vector3d Object)

Normalizes the vector. A normalized vector has a magnitude of 1.0. A vector is normalized by dividing its X, Y, and Z components by the magnitude of the vector. An error will occur if the vector has a magnitude of zero.

C++

Syntax

HRESULT Normalize();

Parameters

None

Return Value

S_OK if Successful

GEOMETRY_E_ZEROMAGNITUDE if the magnitude of the vector is zero

Remarks

None

Parameters

None
