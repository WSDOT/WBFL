SetImplicit Method (Line2d Object)

Redefines the line using its implicit form.

C++

Syntax

HRESULT SetImplicit(double c,IVector2d* n);

Parameters

c

Shortest distance from the origin to the line

n

A vector that is normal to the line. This vector is directed towards the left hand side of the line

Return Value

S_OK if Successful

Remarks

None

Visual Basic

Syntax

SetImplicit(c As Double,n As Vector2d)

Parameters

c

Shortest distance from the origin to the line

n

A vector that is normal to the line. This vector is directed towards the left hand side of the line

Return Value

None

Remarks
