PrimitiveFactory

Utility object used to create instances of geometric primitives. Instances of the geometric primitves implemented in the WBFL Geometry library are created with this implementation. That is, IPoint2dFactory creates instances of the Point2d object from this library.

Implements

IPoint2dFactory

IPoint3dFactory

ILine2dFactory

ILineSegment2dFactory
