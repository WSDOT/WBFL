Center Property (Circle Object)

Center of the circle. This is also the hook point.

C++

Syntax

HRESULT putref_Center(IPoint2d* center);

HRESULT get_Center(IPoint2d** center);

Parameters

center

Center of the circle.

Return Value

S_OK if Successful

Remarks

None

Visual Basic

Syntax

Center(center As Point2d)

Center() As Point2d

Parameters

center

Center of the circle

Return Value

The get property returns the center of the circle

Remarks

None
