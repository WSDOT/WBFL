XformEx Method (CoordinateXform2d Object)

Transforms the coordinates of a point. Coordinate transformation can be from the original system to the new system or from the new system to the original system.

C++

Syntax

HRESULT XformEx(IPoint2d* pnt,XformType type,IPoint2d** result);

Parameters

pnt

Coordinate to be transformed

type

Designates type of transformation

result

The transformed point

Return Value

S_OK if Successful

Remarks

If type is xfrmOldToNew, pnt is a point in the original coordinate system and it is transformed into the new coordinate system. If type is xfrmNewToOld, pnt is a point in the new coordinate system and it is transformed into the original system.

Visual Basic

Syntax

XformEx(pnt As Point2d,type As XformType) As Point2d

Parameters

pnt

Coordinate to be transformed

type

Designates type of transformation

Return Value

Returns the transformed point

Remarks

If type is xfrmOldToNew, pnt is a point in the original coordinate system and it is transformed into the new coordinate system. If type is xfrmNewToOld, pnt is a point in the new coordinate system and it is transformed into the original system.
