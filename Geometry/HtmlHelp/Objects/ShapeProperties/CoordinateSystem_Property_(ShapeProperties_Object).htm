CoordinateSystem Property (ShapeProperties Object)

Coordinate system in which section properties are measured.

The Put version causes the section properties to be transformed to the specified coordinate system. For example, if section properties are currently in the centroidal coordinate system and the coordinate system is changed to principle, the origin, orientation, and section properties will all be transformed to the principle direction coordinate system.

C++

Syntax

HRESULT put_CoordinateSystem(CoordinateSystemType cs);

HRESULT get_CoordinateSystem(CoordinateSystemType* cs);

Parameters

cs

Coordinate system

Return Value

S_OK if Successful

Remarks

None

Visual Basic

Syntax

CoordinateSystem(cs As CoordinateSystemType)

CoordinateSystem() As CoordinateSystemType

Parameters

cs

Coordinate system

Return Value

The get property returns the current coordinate system type.
