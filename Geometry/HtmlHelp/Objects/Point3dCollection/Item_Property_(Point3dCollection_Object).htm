Item Property (Point3dCollection Object)

Returns a point from the collection

C++

Syntax

HRESULT get_Item(long Index,IPoint3d** item);

Parameters

Index

0-based index

item

Pointer to an object that supports the IPoint3d interface

Return Value

S_OK if Successful

Remarks

None

Visual Basic

Syntax

Item(Long Index) As Point3d

Parameters

Index

0-based index

Return Value

Returns a Point3d object

Remarks

None
