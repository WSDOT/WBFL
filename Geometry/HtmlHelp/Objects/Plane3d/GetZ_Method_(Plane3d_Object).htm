GetZ Method (Plane3d Object)

Returns the GetZ ordinate given X and Y

C++

Syntax

HRESULT GetZ(double x,double y,double* z);

Parameters

x

X ordinate

y

Y ordinate

z

Z ordinate

Return Value

S_OK if Successful

GEOMETRY_E_INFINITESOLUTIONS if there are an infinite number of points at the specified coordinate

GEOMETRY_E_NOSOLUTIONS if the specified coordinate is not on the plane.

Remarks

None

Visual Basic

Syntax

GetZ(x As Double, y As Double) As Double

Parameters

x

X ordinate

y

Y ordinate

z

Z ordinate

Return Value

The get property returns the current value of Z

Remarks

None
