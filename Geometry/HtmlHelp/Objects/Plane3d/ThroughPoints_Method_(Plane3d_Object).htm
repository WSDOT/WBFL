ThroughPoints Method (Plane3d Object)

Makes the plane that contains specified three points.

C++

Syntax

HRESULT ThroughLineEx(IPoint3d* p1,IPoint3d* p2,IPoint3d* p3);

Parameters

p1

A point on the plane

p2

A point on the plane

p3

A point on the plane

Return Value

S_OK if Successful

GEOMETRY_E_SAMEPOINTS if the specified points are not unique.

Remarks

None
