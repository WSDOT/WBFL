ContainsPoint Method (Rect2d Object)

Tests if a point is contained within this rectangle. Points that touch the top or right edges of the rectangle are not considered contained.

C++

Syntax

HRESULT ContainsPoint(IPoint2d* pnt,VARIANT_BOOL* bContains);

Parameters

pnt

Pointer to the point that will be tested for containment

bContains

Pointer to the test outcome. VARIANT_TRUE if the point is contained in this rectangle, otherwise VARIANT_FALSE.

Return Value

S_OK if Successful

Remarks

None

Visual Basic

Syntax

ContainsPoint(pnt As Point2d) As Boolean

Parameters

pnt

Point that will be tested for containment

Return Value

Returns True if pnt is contained in this rectangle.

Remarks

None
