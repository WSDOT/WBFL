Bottom Property (Rect2d Object)

Location of the bottom edge of the rectangle.

C++

Syntax

HRESULT put_Bottom(double bottom);

HRESULT get_Bottom(double* bottom);

Parameters

bottom

Bottom edge of the rectangle

Return Value

S_OK if Successful

Remarks

None

Visual Basic

Syntax

Bottom(bottom As Double)

Bottom() As Double

Parameters

bottom

Bottom edge of the rectangle

Return Value

The get property returns the current value of Bottom

Remarks

None
