Rotate Method (Point2d Object)

Repositions the point by rotating it about another point.

C++

Syntax

HRESULT Rotate(double cx,double cy,double angle)

Parameters

cx

Horizontal coordinate of the center point

cy

Vertical coordinate of the center point

angle

Rotation angle, measured counter-clockwise from the positive X axis.

Return Value

S_OK if Successful

Remarks

None

Visual Basic

Syntax

Rotate(cx As Double, cy As Double, angle As Double)

Parameters

cx

Horizontal coordinate of the center point

cy

Vertical coordinate of the center point

angle

Rotation angle, measured counter-clockwise from the positive X axis.

Return Value

None
